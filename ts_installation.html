TypeScript Installation

Set Up TypeScript on Your System

Step 1: Install Node.js

TypeScript requires Node.js to run. Download and install Node.js from the official website or use your system package manager.

// Check Node.js version node -v // Check npm version npm -v

Step 2: Install TypeScript

Install TypeScript globally using npm.

npm install -g typescript

Step 3: Verify Installation

After installation, check the TypeScript version.

tsc -v

Step 4: Create a TypeScript File

Create a file with the .ts extension and write TypeScript code.

let message: string = "Hello TypeScript"; console.log(message);

Step 5: Compile TypeScript to JavaScript

Use the TypeScript compiler (tsc) to convert TypeScript into JavaScript.

tsc filename.ts

This will generate a filename.js file that can run in the browser or Node.js.

Step 6: Run JavaScript

node filename.js

Step 7: Using tsconfig.json

For larger projects, use tsconfig.json to configure TypeScript compilation options.

tsc --init

Conclusion

TypeScript is now ready on your system. You can start writing strongly-typed code, compile it to JavaScript, and run it in browsers or Node.js.
