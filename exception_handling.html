Python Exception Handling

Complete Explanation with Examples and Output

What is Exception Handling?

Exception handling allows a program to deal with errors gracefully instead of crashing. Python uses try and except blocks to handle exceptions.

Basic try-except Block

try: num = int(input("Enter a number: ")) print("Number is:", num) except ValueError: print("Invalid input! Please enter a number.")
Output (if input is 'abc'):
Invalid input! Please enter a number.

Handling Multiple Exceptions

try: x = int(input("Enter first number: ")) y = int(input("Enter second number: ")) result = x / y except ValueError: print("Invalid input!") except ZeroDivisionError: print("Cannot divide by zero!") else: print("Result is:", result)
Output (if y=0):
Cannot divide by zero!

Using finally Block

The finally block executes whether or not an exception occurs.

try: file = open("example.txt", "r") content = file.read() except FileNotFoundError: print("File not found!") finally: print("Execution completed.")
Output (if file does not exist):
File not found!
Execution completed.

Raising Exceptions

def check_age(age): if age < 18: raise ValueError("Age must be at least 18.") else: print("Access granted.") check_age(15)
Output:
ValueError: Age must be at least 18.

Common Python Exceptions

  • ValueError – Invalid value
  • TypeError – Wrong data type
  • ZeroDivisionError – Division by zero
  • FileNotFoundError – File does not exist
  • IndexError – Index out of range
  • KeyError – Key not found in dictionary

Conclusion

Exception handling is essential for building robust programs. Using try, except, else, and finally ensures your Python program handles errors gracefully.
