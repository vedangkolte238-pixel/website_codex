SQL with Python

Connecting Python applications to SQL databases

1. Installing Required Libraries

Use Python libraries like sqlite3, mysql-connector-python, or psycopg2 to interact with databases.

# Install MySQL connector pip install mysql-connector-python

2. Connecting to a Database

Establish a connection to your database from Python.

import mysql.connector conn = mysql.connector.connect( host="localhost", user="root", password="password", database="testdb" ) cursor = conn.cursor()

3. Creating Tables

Execute SQL commands using Python to create tables.

cursor.execute(""" CREATE TABLE employees ( emp_id INT AUTO_INCREMENT PRIMARY KEY, name VARCHAR(50), department VARCHAR(50) ) """)

4. Inserting Data

Insert records using Python with parameterized queries to prevent SQL injection.

sql = "INSERT INTO employees (name, department) VALUES (%s, %s)" values = ("Alice", "HR") cursor.execute(sql, values) conn.commit()

5. Reading Data

Fetch and process data from the database.

cursor.execute("SELECT * FROM employees") rows = cursor.fetchall() for row in rows: print(row)

6. Updating Data

Modify existing records from Python.

sql = "UPDATE employees SET department=%s WHERE name=%s" values = ("Finance", "Alice") cursor.execute(sql, values) conn.commit()

7. Deleting Data

Remove records programmatically.

sql = "DELETE FROM employees WHERE name=%s" values = ("Alice",) cursor.execute(sql, values) conn.commit()

8. Closing the Connection

Always close the cursor and connection when done.

cursor.close() conn.close()

Conclusion

Integrating SQL with Python allows you to build database-driven applications, automate tasks, and perform data analysis efficiently.
