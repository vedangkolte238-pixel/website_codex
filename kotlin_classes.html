Kotlin Classes

Understand Classes and Objects in Kotlin

What is a Class?

A class is a blueprint for creating objects. It defines properties (variables) and functions (methods) that describe the behavior of an object.

Class Syntax in Kotlin

class ClassName { // properties // functions }

Example of a Class

class Person { var name: String = "John" var age: Int = 25 fun displayInfo() { println("Name: $name, Age: $age") } }

Creating an Object

An object is an instance of a class.

val person = Person() person.displayInfo()

Primary Constructor

Kotlin allows defining a constructor directly in the class header.

class Student(val name: String, var marks: Int)

Class with Constructor and Method

class Car(val brand: String, val year: Int) { fun details() { println("Brand: $brand, Year: $year") } }

Init Block

The init block runs when the object is created.

class Laptop(val model: String) { init { println("Laptop model: $model") } }

Access Modifiers

  • public – default, accessible everywhere
  • private – accessible inside the class
  • protected – accessible in subclasses
  • internal – accessible within the module

Conclusion

Classes are the foundation of object-oriented programming in Kotlin. They help organize code, represent real-world entities, and build scalable applications.
