HTML Images

Learn how to display images in a web page

Introduction

Images make web pages more attractive and informative. In HTML, images are displayed using the <img> tag.

Basic Image Syntax

<img src="image.jpg" alt="Sample Image">

Explanation

  • src specifies the image file path
  • alt provides alternative text if the image does not load
  • The <img> tag is self-closing

Setting Image Width and Height

You can control the size of an image using width and height attributes.

<img src="photo.png" alt="Photo" width="200" height="150">

Using CSS to Resize Images

Images can also be styled using CSS.

<img src="logo.png" class="example" alt="Logo">

Image Folder Structure

Images are usually stored in a separate folder.

project-folder/ │ ├── images/ │ └── picture.jpg ├── index.html

Common Image Formats

  • JPG – photos
  • PNG – transparent images
  • GIF – simple animations
  • SVG – scalable vector graphics

Important Tips

  • Always use the alt attribute
  • Keep image size optimized
  • Use meaningful image names

Conclusion

HTML images help improve the visual appearance of a website. Using images correctly makes your pages more engaging and user-friendly.
