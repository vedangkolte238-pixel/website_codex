JavaScript Variables

Store and Manage Data in JavaScript

What are Variables?

Variables are containers used to store data values. In JavaScript, variables allow you to store numbers, text, and other data types.

Declaring Variables

JavaScript provides three keywords to declare variables.

1. var

The var keyword is function-scoped and can be redeclared.

var name = "Alex"; console.log(name);

2. let

The let keyword is block-scoped and recommended for most cases.

let age = 25; console.log(age);

3. const

The const keyword is used for values that should not change.

const country = "India"; console.log(country);

Variable Naming Rules

  • Names must start with a letter, underscore, or dollar sign
  • Cannot start with a number
  • Cannot use JavaScript keywords
  • Case-sensitive

Reassigning Variables

Variables declared with let can be reassigned.

let score = 10; score = 20;

Constant Variables

Values declared with const cannot be reassigned.

const pi = 3.14; // pi = 3.15; ❌ Error

Multiple Variables

You can declare multiple variables in one statement.

let x = 10, y = 20, z = 30;

Conclusion

Variables are the foundation of JavaScript programming. Understanding how to use var, let, and const helps you write clean and reliable code.
