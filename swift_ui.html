SwiftUI

Building Modern UIs for Apple Platforms

What Is SwiftUI?

SwiftUI is Apple's declarative framework for building user interfaces across iOS, macOS, watchOS, and tvOS. It allows developers to create UI using simple Swift code and live previews.

Getting Started

To use SwiftUI, create a new SwiftUI project in Xcode. SwiftUI works seamlessly with Combine and other Swift features.

Basic SwiftUI View

import SwiftUI struct ContentView: View { var body: some View { Text("Hello, SwiftUI!") .font(.title) .foregroundColor(.blue) .padding() } } struct ContentView_Previews: PreviewProvider { static var previews: some View { ContentView() } }

Layout Containers

SwiftUI uses stacks and other containers to layout UI elements.

VStack { Text("Hello") Text("World") } HStack { Text("Left") Text("Right") } ZStack { Color.gray Text("On Top") }

Buttons

Button(action: { print("Button tapped") }) { Text("Tap Me") .padding() .background(Color.orange) .foregroundColor(.white) .cornerRadius(10) }

State Management

Use @State to manage mutable data in SwiftUI.

struct CounterView: View { @State private var count = 0 var body: some View { VStack { Text("Count: \(count)") Button("Increment") { count += 1 } } } }

Binding

Use @Binding to share state between views.

struct ParentView: View { @State private var isOn = false var body: some View { ToggleView(isOn: $isOn) } } struct ToggleView: View { @Binding var isOn: Bool var body: some View { Toggle("Switch", isOn: $isOn) } }

Modifiers

SwiftUI uses modifiers to change the appearance or behavior of views.

Text("Hello") .font(.headline) .foregroundColor(.red) .padding() .background(Color.yellow) .cornerRadius(8)

Conclusion

SwiftUI makes UI development faster, easier, and more interactive. Its declarative syntax, live previews, and seamless integration with Swift allow developers to build modern apps efficiently.
