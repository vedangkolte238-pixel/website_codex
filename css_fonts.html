CSS Fonts

Styling Text Using CSS Fonts

What are CSS Fonts?

CSS fonts are used to control the appearance of text on a web page. They define font style, size, weight, and family.

Font Family

Specifies the typeface for text.

p { font-family: Arial, Helvetica, sans-serif; }

Font Size

Controls the size of the text.

h1 { font-size: 32px; }

Font Weight

Defines how bold or light the text appears.

p { font-weight: bold; }

Font Style

Sets italic or normal text style.

p { font-style: italic; }

Font Variant

Used to display text in small caps.

p { font-variant: small-caps; }

Line Height

Controls the space between lines of text.

p { line-height: 1.6; }

Font Shorthand

All font properties can be written in one line.

p { font: italic small-caps bold 16px Arial; }

Web Safe Fonts

Web safe fonts work across all browsers and devices.

font-family: Arial, Times New Roman, Verdana;

Best Practices

  • Use readable font sizes
  • Limit the number of fonts
  • Use fallback fonts

Conclusion

CSS fonts play a major role in readability and design. Choosing the right fonts improves user experience.
