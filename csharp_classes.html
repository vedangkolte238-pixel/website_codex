C# Classes

Understanding Classes in C#

What is a Class?

A class in C# is a blueprint for creating objects. It defines properties (attributes) and methods (functions) that an object can have.

Defining a Class

Syntax to define a class:

class ClassName { // Fields (attributes) // Methods (functions) }

Example of a Class

Creating a simple class called Car:

using System; class Car { // Fields public string brand; public string model; public int year; // Method public void DisplayInfo() { Console.WriteLine("Brand: " + brand); Console.WriteLine("Model: " + model); Console.WriteLine("Year: " + year); } } class Program { static void Main() { // Creating an object of Car Car myCar = new Car(); myCar.brand = "Toyota"; myCar.model = "Corolla"; myCar.year = 2021; myCar.DisplayInfo(); } }

Key Points

  • Classes are templates; objects are instances of classes.
  • Fields store data; methods perform actions.
  • Access modifiers like public, private control visibility.

Benefits of Using Classes

  • Encapsulation – keeps data and methods together.
  • Reusability – classes can be reused in multiple programs.
  • Organized code – makes programs easier to maintain.

Conclusion

Classes are the foundation of object-oriented programming in C#. They allow you to model real-world entities and organize your code efficiently.
