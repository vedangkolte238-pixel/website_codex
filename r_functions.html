R Functions

Reusable blocks of code in R programming

Introduction

A function in R is a reusable block of code that performs a specific task. Functions help reduce repetition and make programs easier to manage.

Built-in Functions

R provides many built-in functions for common operations.

sqrt(25) sum(10, 20, 30) mean(c(5, 10, 15))

User-Defined Functions

You can create your own functions using the function keyword.

add_numbers <- function(a, b) { result <- a + b return(result) } add_numbers(5, 3)

Function Without Return

If no return statement is used, R returns the last evaluated expression.

square <- function(x) { x * x } square(4)

Function Arguments

Functions can accept one or more arguments.

multiply <- function(a, b, c) { a * b * c } multiply(2, 3, 4)

Default Arguments

Default values can be assigned to function parameters.

greet <- function(name = "User") { paste("Hello", name) } greet() greet("R Programmer")

Anonymous Functions

Functions without names are called anonymous functions.

result <- (function(x, y) { x + y })(4, 6) result

Passing Functions as Arguments

Functions can be passed as arguments to other functions.

apply_function <- function(x, func) { func(x) } apply_function(9, sqrt)

Conclusion

Functions are a core concept in R programming. They improve code readability, reusability, and structure.
