JavaScript Objects

Store Data in Key and Value Pairs

What is an Object?

An object is a collection of properties. Each property has a name (key) and a value. Objects are used to represent real world entities.

Creating an Object

let person = { name: "Rahul", age: 22, isStudent: true };

Accessing Object Properties

Using Dot Notation

console.log(person.name); console.log(person.age);

Using Bracket Notation

console.log(person["name"]); console.log(person["age"]);

Updating Object Properties

person.age = 23; person.city = "Pune";

Object Methods

Objects can also contain functions called methods.

let person = { name: "Rahul", greet: function() { console.log("Hello " + this.name); } }; person.greet();

Accessing Methods

person.greet();

Looping Through an Object

for (let key in person) { console.log(key + ": " + person[key]); }

Nested Objects

let student = { name: "Amit", marks: { math: 80, science: 85 } }; console.log(student.marks.math);

Array of Objects

let users = [ { name: "Asha", age: 20 }, { name: "Neha", age: 22 } ]; console.log(users[0].name);

Why Use Objects?

  • Organize related data
  • Represent real world entities
  • Make code easier to manage
  • Used in APIs and applications

Conclusion

Objects are one of the most important parts of JavaScript. Understanding objects helps you build real applications.
