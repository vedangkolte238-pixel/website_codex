TypeScript Data Types

Learn About Variables and Their Types

What are Data Types?

TypeScript provides types to describe the kind of values a variable can hold. It helps catch errors during development.

Basic Data Types

1. Number

Represents numeric values (integers and floating points).

let age: number = 25; let price: number = 99.99;

2. String

Represents text values.

let name: string = "Alice"; let greeting: string = `Hello, ${name}`;

3. Boolean

Represents true or false.

let isStudent: boolean = true; let hasLicense: boolean = false;

4. Array

Stores multiple values of the same type.

let numbers: number[] = [1, 2, 3, 4]; let fruits: string[] = ["Apple", "Banana", "Orange"];

5. Tuple

Stores a fixed number of elements of different types.

let person: [string, number] = ["Alice", 25];

6. Enum

Defines a set of named constants.

enum Color {Red, Green, Blue} let favorite: Color = Color.Green;

7. Any

Can hold any type of value. Use sparingly.

let data: any = 10; data = "Hello"; data = true;

8. Void

Used for functions that do not return a value.

function sayHello(): void { console.log("Hello!"); }

9. Null and Undefined

let nothing: null = null; let notAssigned: undefined = undefined;

10. Object

Represents non-primitive types like objects or arrays.

let user: object = {name: "Alice", age: 25};

11. Never

Represents values that never occur (e.g., functions that throw errors).

function error(message: string): never { throw new Error(message); }

Conclusion

Using TypeScript data types makes your code safer and easier to maintain. You can catch mistakes early and write predictable applications.
