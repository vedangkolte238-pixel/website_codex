Pandas Indexing & Selection

Accessing Rows and Columns in Pandas DataFrames

Selecting Columns

import pandas as pd data = { 'Name': ['Alice', 'Bob', 'Charlie'], 'Age': [25, 30, 35], 'City': ['New York', 'Los Angeles', 'Chicago'] } df = pd.DataFrame(data) # Single column print(df['Name']) # Multiple columns print(df[['Name', 'Age']])

Selecting Rows

# Using iloc (integer-location based) print(df.iloc[0]) # First row print(df.iloc[0:2]) # First two rows # Using loc (label-based) print(df.loc[0]) # First row print(df.loc[0:1]) # First two rows

Conditional Selection

# Select rows where Age > 28 print(df[df['Age'] > 28]) # Select rows where City is 'Chicago' print(df[df['City'] == 'Chicago'])

Setting and Accessing Index

df_indexed = df.set_index('Name') print(df_indexed) # Access row by index print(df_indexed.loc['Bob'])

Boolean Indexing

# Create a boolean mask mask = df['Age'] > 28 print(df[mask]) # Only rows with Age > 28

Using .at and .iat

# Fast access to single value print(df.at[1, 'City']) # 'Los Angeles' print(df.iat[1, 2]) # 'Los Angeles' (integer-based)

Applications

  • Filtering data based on conditions
  • Accessing specific rows or columns efficiently
  • Preparing subsets of data for analysis or visualization
  • Integrating with other libraries like NumPy and Matplotlib

Conclusion

Indexing and selection in Pandas is essential for efficient data manipulation. Using loc, iloc, conditional and boolean indexing allows precise control over your dataset.
