HTML Basic Structure

Understanding the skeleton of an HTML page

Introduction

Every HTML page follows a standard structure that allows browsers to understand and render the content correctly. Using proper structure ensures the page is readable, maintainable, and works across all browsers.

Basic Structure

The simplest HTML document looks like this:

My First Page

Welcome to HTML

This is a paragraph of text.

Explanation of Each Section

  • <!DOCTYPE html>: Declares that this document is HTML5.
  • <html lang="en">: The root element that wraps the entire page.
  • <head>: Contains metadata, including the page title, character encoding, links to CSS, and scripts.
  • <meta charset="UTF-8">: Sets the character encoding to UTF-8 for proper text rendering.
  • <title>: Specifies the page title shown in the browser tab.
  • <body>: Contains all visible content such as headings, paragraphs, images, links, and other elements.

Indented Visual Structure

Using indentation helps show the hierarchy of elements:

Conclusion

The basic structure of HTML forms the foundation of every web page. Always start with a proper DOCTYPE, include the <head> for metadata, and place all visible content inside the <body>. This structure is essential before adding styles with CSS or interactivity with JavaScript.
