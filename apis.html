Python API Programming

Working with Web APIs using Python

Introduction

API (Application Programming Interface) allows applications to communicate with each other. Python's requests library makes it easy to interact with web APIs.

1. Installing Requests Library

# Install requests if not installed pip install requests

2. Making a GET Request

import requests response = requests.get("https://api.github.com") print(response.status_code) print(response.json())
Output:
200
{'current_user_url': 'https://api.github.com/user', 'current_user_authorizations_html_url': 'https://github.com/settings/connections/applications{/client_id}', ...}

3. Making a POST Request

import requests url = "https://jsonplaceholder.typicode.com/posts" data = { "title": "foo", "body": "bar", "userId": 1 } response = requests.post(url, json=data) print(response.status_code) print(response.json())
Output:
201
{'title': 'foo', 'body': 'bar', 'userId': 1, 'id': 101}

4. Using Headers and Parameters

import requests url = "https://api.github.com/search/repositories" params = {"q": "python"} headers = {"Accept": "application/vnd.github.v3+json"} response = requests.get(url, params=params, headers=headers) data = response.json() print(data["total_count"]) print(data["items"][0]["full_name"])
Output:
Total repositories matching 'python': 500000+
First repository: The actual repo name from API response

5. Error Handling

import requests try: response = requests.get("https://api.github.com/invalidendpoint") response.raise_for_status() except requests.exceptions.HTTPError as err: print("HTTP error occurred:", err) except Exception as e: print("Other error occurred:", e)
Output:
HTTP error occurred: 404 Client Error: Not Found for url: https://api.github.com/invalidendpoint

Conclusion

Python's requests library makes it simple to work with web APIs. You can perform GET, POST, PUT, DELETE requests, handle errors, and interact with JSON data efficiently. APIs are essential for connecting your Python applications with web services, databases, or other applications.
