Practical Projects Using NumPy

1. Array Operations Calculator

Create a program that takes two arrays as input and performs addition, subtraction, multiplication, and division using NumPy operations. Display results for each operation.

import numpy as np a = np.array([1,2,3]) b = np.array([4,5,6]) print("Addition:", a + b) print("Subtraction:", a - b) print("Multiplication:", a * b) print("Division:", a / b)

2. Random Data Generator

Generate random datasets for testing machine learning models. Include random integers, floats, and normal distributions.

import numpy as np # Random integers rand_int = np.random.randint(0, 100, size=10) # Random floats rand_float = np.random.rand(10) # Normal distribution rand_normal = np.random.normal(0, 1, size=10) print(rand_int, rand_float, rand_normal)

3. Statistical Analysis of Data

Use NumPy to calculate mean, median, standard deviation, and cumulative sum of a dataset.

data = np.array([12, 15, 20, 21, 19, 18]) print("Mean:", np.mean(data)) print("Median:", np.median(data)) print("Std Dev:", np.std(data)) print("Cumulative Sum:", np.cumsum(data))

4. Image Brightness Adjuster

Using NumPy arrays, read an image as a matrix and adjust its brightness by adding a scalar to all pixel values.

from PIL import Image import numpy as np img = Image.open("image.jpg") arr = np.array(img) # Increase brightness bright_img = np.clip(arr + 50, 0, 255) Image.fromarray(bright_img.astype('uint8')).save("bright_image.jpg")

5. Matrix Operations

Perform matrix addition, multiplication, transpose, and determinant calculation using NumPy.

A = np.array([[1,2],[3,4]]) B = np.array([[5,6],[7,8]]) print("Addition:\n", A + B) print("Multiplication:\n", np.dot(A, B)) print("Transpose of A:\n", A.T) print("Determinant of A:", np.linalg.det(A))

Conclusion

These projects help reinforce NumPy concepts like arrays, operations, random number generation, and linear algebra. They are perfect for beginners to intermediate learners to gain practical experience in Python numerical computing.
