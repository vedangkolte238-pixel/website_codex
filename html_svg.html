Scalable Vector Graphics for Web

Introduction

SVG (Scalable Vector Graphics) is an XML-based markup language for describing 2D graphics. It allows graphics to scale without losing quality.

Basic SVG Syntax

<svg width="100" height="100"> <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" /> </svg>

Example: Circle

Example: Rectangle

<svg width="200" height="100"> <rect width="150" height="80" x="25" y="10" stroke="black" fill="#F06529" stroke-width="3" /> </svg>

Example: Line

<svg width="200" height="100"> <line x1="0" y1="50" x2="200" y2="50" stroke="#F06529" stroke-width="4" /> </svg>

Example: Text

<svg width="200" height="100"> <text x="10" y="50" font-family="Arial" font-size="20" fill="#F06529">Hello SVG</text> </svg>

Key Points

  • SVG is scalable and resolution-independent
  • Can include shapes, text, lines, and paths
  • Works well with CSS and JavaScript for animations

Conclusion

SVG is perfect for logos, icons, charts, and interactive graphics because it scales without losing quality.
