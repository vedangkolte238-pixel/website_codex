Go Functions

Understanding Functions in Go

What is a Function?

A function is a block of code that performs a specific task. Functions help make code reusable, readable, and easier to maintain.

Defining a Function

Basic syntax:

func functionName(parameters) returnType { // function body }

Example Function

package main import "fmt" func greet() { fmt.Println("Welcome to Go Functions") } func main() { greet() }

Function with Parameters

func add(a int, b int) int { return a + b }

Function with Multiple Return Values

Go supports returning multiple values from a function.

func calculate(a int, b int) (int, int) { return a + b, a * b }

Named Return Values

func rectangle(length int, width int) (area int, perimeter int) { area = length * width perimeter = 2 * (length + width) return }

Anonymous Functions

Functions without a name are called anonymous functions.

func() { fmt.Println("Anonymous function in Go") }()

Passing Functions as Arguments

func operate(a int, b int, op func(int, int) int) int { return op(a, b) }

Key Points

  • Functions are defined using the func keyword.
  • Go supports multiple return values.
  • Functions can be passed as arguments.
  • Anonymous functions are allowed.

Conclusion

Functions are a core part of Go programming. They help you write clean, modular, and efficient code.
