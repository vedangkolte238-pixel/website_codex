R Hypothesis Testing

Performing statistical tests using R

Introduction

Hypothesis testing is a statistical method to make decisions or inferences about population parameters based on sample data. R provides built-in functions to perform common tests such as t-tests, chi-square tests, and proportion tests.

One-Sample t-Test

# Sample data data <- c(5, 7, 8, 6, 9) # One-sample t-test (testing mean = 6) t.test(data, mu = 6)

Two-Sample t-Test

# Two groups group1 <- c(5, 6, 7, 8) group2 <- c(6, 7, 8, 9) # Two-sample t-test t.test(group1, group2, alternative = "two.sided")

Paired t-Test

# Before and after measurements before <- c(90, 85, 88, 92) after <- c(95, 87, 90, 94) t.test(before, after, paired = TRUE)

Chi-Square Test

# Contingency table data <- matrix(c(20, 15, 30, 35), nrow = 2) # Chi-square test chisq.test(data)

Proportion Test

# Number of successes and trials success <- c(45, 30) trials <- c(50, 40) prop.test(success, trials)

ANOVA (Analysis of Variance)

# Sample data group1 <- c(5, 6, 7) group2 <- c(8, 9, 10) group3 <- c(6, 7, 8) data <- data.frame( value = c(group1, group2, group3), group = factor(rep(c("A","B","C"), each = 3)) ) # Perform ANOVA anova_result <- aov(value ~ group, data = data) summary(anova_result)

Conclusion

R provides a wide variety of hypothesis testing functions to analyze and infer population characteristics from sample data. Mastering these tests is essential for statistical analysis and research.
