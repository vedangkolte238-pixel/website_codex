Python Tuple

Complete Explanation with Examples and Output

What is a Tuple?

A tuple is a collection of items stored in a single variable. Tuples are ordered, immutable (cannot be changed), and allow duplicate values.

Creating a Tuple

numbers = (10, 20, 30, 40) print(numbers)
Output:
(10, 20, 30, 40)

Tuple with Different Data Types

data = (1, "Python", 3.5, True) print(data)
Output:
(1, 'Python', 3.5, True)

Accessing Tuple Elements

print(numbers[0]) print(numbers[2]) print(numbers[-1])
Output:
10
30
40

Tuple Slicing

print(numbers[1:3]) print(numbers[:2]) print(numbers[2:])
Output:
(20, 30)
(10, 20)
(30, 40)

Tuple is Immutable

You cannot change elements of a tuple after it is created.

numbers[0] = 100
Output:
TypeError: 'tuple' object does not support item assignment

Tuple Operations

Concatenation (+)

a = (1, 2) b = (3, 4) print(a + b)
Output:
(1, 2, 3, 4)

Repetition (*)

print(a * 2)
Output:
(1, 2, 1, 2)

Looping Through a Tuple

for item in numbers: print(item)
Output:
10
20
30
40

Tuple Methods

Tuples have only two built-in methods: count() and index()

count()

nums = (1, 2, 2, 3, 2) print(nums.count(2))
Output:
3

index()

print(nums.index(3))
Output:
3

Tuple Packing and Unpacking

person = ("Alice", 25) name, age = person print(name) print(age)
Output:
Alice
25

Conclusion

Tuples are ideal when you want to store fixed, unchangeable data. They are fast, reliable, and widely used in Python.
