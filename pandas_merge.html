Pandas Merge & Join

Combining DataFrames in Pandas

What is Merge?

The merge() function in Pandas combines two DataFrames based on a key column(s) similar to SQL joins. This allows integrating data from multiple sources into a single DataFrame.

Basic Merge

import pandas as pd df1 = pd.DataFrame({ 'EmployeeID': [1, 2, 3], 'Name': ['Alice', 'Bob', 'Charlie'] }) df2 = pd.DataFrame({ 'EmployeeID': [1, 2, 4], 'Salary': [50000, 60000, 70000] }) # Inner join (only matching keys) merged_inner = pd.merge(df1, df2, on='EmployeeID', how='inner') print(merged_inner)

Types of Merge / Join

# Left join (all rows from left, matched from right) merged_left = pd.merge(df1, df2, on='EmployeeID', how='left') # Right join (all rows from right, matched from left) merged_right = pd.merge(df1, df2, on='EmployeeID', how='right') # Outer join (all rows from both, NaN for missing) merged_outer = pd.merge(df1, df2, on='EmployeeID', how='outer')

Merge on Multiple Columns

df1 = pd.DataFrame({ 'EmployeeID': [1, 2, 3], 'Dept': ['HR', 'IT', 'IT'], 'Name': ['Alice', 'Bob', 'Charlie'] }) df2 = pd.DataFrame({ 'EmployeeID': [1, 2, 3], 'Dept': ['HR', 'IT', 'Sales'], 'Salary': [50000, 60000, 70000] }) merged_multi = pd.merge(df1, df2, on=['EmployeeID', 'Dept'], how='inner') print(merged_multi)

Concatenation (Stacking DataFrames)

# Combine vertically df_top = pd.DataFrame({'A':[1,2], 'B':[3,4]}) df_bottom = pd.DataFrame({'A':[5,6], 'B':[7,8]}) df_concat = pd.concat([df_top, df_bottom], axis=0) # Rows print(df_concat) # Combine horizontally df_concat_h = pd.concat([df_top, df_bottom], axis=1) # Columns print(df_concat_h)

Applications

  • Integrating data from multiple CSV, Excel, or database tables
  • Performing SQL-style joins in Python
  • Combining different datasets for analysis or visualization
  • Stacking or appending datasets vertically or horizontally

Conclusion

Pandas provides versatile tools to merge, join, and concatenate DataFrames. This is essential for combining datasets, integrating data sources, and preparing data for analysis.
