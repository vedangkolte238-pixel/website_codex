Python File Handling

Complete Explanation with Examples and Output

What is File Handling?

File handling allows you to read from and write to files using Python. Python provides built-in functions to work with files efficiently.

Opening a File

Use the open() function. Modes include:

  • 'r' – Read (default)
  • 'w' – Write (overwrite if exists)
  • 'a' – Append
  • 'x' – Create (fails if file exists)
  • 'b' – Binary mode
file = open("example.txt", "w") file.write("Hello, Python File Handling!") file.close()

Reading from a File

file = open("example.txt", "r") content = file.read() print(content) file.close()
Output:
Hello, Python File Handling!

Reading Line by Line

file = open("example.txt", "r") for line in file: print(line) file.close()
Output:
Hello, Python File Handling!

Appending to a File

file = open("example.txt", "a") file.write("\nThis is a new line.") file.close()

Using with Statement (Recommended)

Automatically closes the file after usage.

with open("example.txt", "r") as file: content = file.read() print(content)
Output:
Hello, Python File Handling!
This is a new line.

File Methods

  • read() – Read entire file
  • readline() – Read a single line
  • readlines() – Read all lines into a list
  • write() – Write data to file
  • writelines() – Write a list of lines
  • close() – Close the file

Conclusion

Python file handling is essential for data storage, reading configuration files, logging, and managing text or binary data efficiently. Always use with for safe and clean file handling.
