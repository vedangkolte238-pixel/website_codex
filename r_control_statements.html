R Control Statements

Decision making and looping in R programming

Introduction

Control statements are used to control the flow of execution of a program. They help in decision making and repeating tasks.

1. if Statement

Executes a block of code if a condition is TRUE.

x <- 10 if (x > 5) { print("x is greater than 5") }

2. if...else Statement

Provides an alternative block when the condition is FALSE.

x <- 3 if (x > 5) { print("x is greater than 5") } else { print("x is less than or equal to 5") }

3. if...else if...else

Used to check multiple conditions.

marks <- 75 if (marks >= 90) { print("Grade A") } else if (marks >= 60) { print("Grade B") } else { print("Grade C") }

4. switch Statement

Selects one option from multiple choices.

choice <- 2 result <- switch(choice, "Apple", "Banana", "Mango") print(result)

5. for Loop

Used to repeat a block of code for a fixed number of times.

for (i in 1:5) { print(i) }

6. while Loop

Executes code as long as the condition is TRUE.

i <- 1 while (i <= 5) { print(i) i <- i + 1 }

7. repeat Loop

Repeats execution until a break condition is met.

i <- 1 repeat { print(i) i <- i + 1 if (i > 5) { break } }

8. break and next

break exits the loop, and next skips the current iteration.

for (i in 1:5) { if (i == 3) { next } print(i) }

Summary

  • if, if-else for decision making
  • switch for multiple conditions
  • for, while, repeat loops
  • break and next for loop control

Conclusion

Control statements are the backbone of logical programming in R. They allow programs to respond dynamically to different conditions.
