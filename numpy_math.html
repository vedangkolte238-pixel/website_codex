NumPy Mathematical Functions

Perform Mathematical Operations on NumPy Arrays

Universal Functions (ufuncs)

NumPy provides built-in functions (ufuncs) for element-wise mathematical operations on arrays. These are faster and more efficient than Python loops.

Basic Arithmetic Functions

import numpy as np arr = np.array([1, 4, 9, 16]) print(np.sqrt(arr)) # Square root → [1. 2. 3. 4.] print(np.square(arr)) # Square → [1 16 81 256] print(np.abs(np.array([-5, 3, -1]))) # Absolute → [5 3 1]

Exponential and Logarithmic Functions

print(np.exp(np.array([1, 2, 3]))) # Exponential e^x print(np.log(np.array([1, np.e, np.e**2]))) # Natural log → [0 1 2] print(np.log10(np.array([1, 10, 100]))) # Base-10 logarithm → [0 1 2]

Trigonometric Functions

angles = np.array([0, np.pi/2, np.pi]) print(np.sin(angles)) # [0. 1. 0.] print(np.cos(angles)) # [1. 0. -1.] print(np.tan(angles)) # [0. inf 0.]

Aggregate Functions

Compute statistics on arrays:

arr = np.array([1, 2, 3, 4, 5]) print(np.sum(arr)) # 15 print(np.mean(arr)) # 3.0 print(np.min(arr)) # 1 print(np.max(arr)) # 5 print(np.std(arr)) # Standard deviation print(np.cumsum(arr)) # Cumulative sum → [1 3 6 10 15]

Other Useful Functions

  • np.floor(arr): Round down
  • np.ceil(arr): Round up
  • np.round(arr, n): Round to n decimals
  • np.mod(arr1, arr2): Element-wise modulo
  • np.power(arr, n): Raise to power n

Applications

  • Scientific computations and simulations
  • Data analysis and feature scaling
  • Machine learning preprocessing
  • Engineering and financial calculations

Conclusion

NumPy mathematical functions make array computations fast, efficient, and easy to read. Mastering these functions is essential for data science, ML, and scientific programming.
