R Arrays

Multi-dimensional data structures in R programming

Introduction

An array in R is a data structure that can store data in more than two dimensions. All elements in an array must be of the same data type.

Creating an Array

Arrays are created using the array() function.

arr <- array(1:12, dim = c(3, 2, 2)) arr

Understanding Array Dimensions

The dim parameter defines rows, columns, and layers.

dim(arr)

Accessing Array Elements

Elements are accessed using indices for each dimension.

arr[1, 2, 1] arr[, , 2]

Modifying Array Elements

You can change array values by assigning new data.

arr[1, 1, 1] <- 100 arr

Array Operations

R supports element-wise operations on arrays.

arr2 <- array(1:12, dim = c(3, 2, 2)) arr + arr2 arr * arr2

Array Functions

Common functions used with arrays.

length(arr) apply(arr, c(1, 2), sum)

Difference Between Matrix and Array

  • Matrix: Two-dimensional
  • Array: Multi-dimensional
  • Both store same data type elements

Conclusion

Arrays are useful for handling complex multi-dimensional data. They are commonly used in scientific and statistical computing.
