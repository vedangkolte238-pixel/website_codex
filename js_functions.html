JavaScript Functions

Reusable Blocks of Code in JavaScript

What is a Function?

A function is a block of reusable code designed to perform a specific task. Functions help reduce repetition and make code easier to manage.

Function Syntax

function functionName() { // code to execute }

Simple Function Example

function greet() { console.log("Hello, JavaScript"); } greet();

Function with Parameters

Parameters allow you to pass values to a function.

function add(a, b) { return a + b; } add(5, 3);

Function with Return Value

The return statement sends a value back to the caller.

function square(num) { return num * num; } square(4);

Function Expression

A function can be stored inside a variable.

const multiply = function(x, y) { return x * y; }; multiply(4, 5);

Arrow Function

Arrow functions provide a shorter syntax.

const subtract = (a, b) => a - b; subtract(10, 3);

Default Parameters

Default values are used if no argument is passed.

function greetUser(name = "Guest") { return "Hello " + name; } greetUser();

Why Use Functions?

  • Makes code reusable
  • Improves readability
  • Reduces errors
  • Easy to debug

Conclusion

Functions are a core concept in JavaScript. Mastering functions allows you to build clean, efficient, and scalable programs.
