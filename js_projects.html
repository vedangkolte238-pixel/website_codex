JavaScript Projects

Ideas and Examples to Practice JavaScript

Introduction

These projects help beginners and intermediate learners practice JavaScript. You can showcase these on your portfolio or website. Projects are divided into categories: DOM Manipulation, Games, Forms, API, and Mini Apps.

1. DOM Manipulation Projects

  • To-Do List

    Add, delete, and mark tasks as completed using DOM methods.

    // Example: Add task to list let taskInput = document.getElementById("taskInput"); let taskList = document.getElementById("taskList"); function addTask() { let li = document.createElement("li"); li.textContent = taskInput.value; taskList.appendChild(li); }
  • Color Changer

    Change background or text color dynamically on button clicks.

    document.getElementById("btn").onclick = function() { document.body.style.backgroundColor = "lightblue"; };

2. Game Projects

  • Number Guessing Game

    Guess a number between 1 and 100. Show hints for higher/lower.

    let secret = Math.floor(Math.random() * 100) + 1; function guessNumber(userGuess) { if(userGuess === secret) alert("Correct!"); else if(userGuess < secret) alert("Too low!"); else alert("Too high!"); }
  • Simple Quiz

    Display multiple-choice questions and show score at the end.

3. Form Projects

  • Form Validation

    Check if all fields are filled and show error messages.

    let form = document.getElementById("myForm"); form.addEventListener("submit", function(e) { e.preventDefault(); if(document.getElementById("name").value === "") { alert("Name is required"); } });
  • Signup Form

    Validate email, password, and confirm password fields.

4. API & Fetch Projects

  • Weather App

    Fetch weather data from an API and display temperature and condition.

    fetch("https://api.example.com/weather") .then(response => response.json()) .then(data => console.log(data));
  • Random User Generator

    Display random user info using fetch from a public API.

5. Mini Apps

  • Calculator

    Simple arithmetic calculator using JavaScript functions.

  • Stopwatch

    Start, stop, and reset a timer using setInterval.

Conclusion

These projects help you practice key JavaScript concepts like DOM, events, arrays, objects, and asynchronous code. Start small, and gradually move to API-based or interactive apps.
