TypeScript Classes

Learn Object-Oriented Programming in TypeScript

What is a Class?

A class is a blueprint for creating objects with properties and methods. TypeScript supports classes with features like access modifiers and inheritance.

Basic Class Example

class Person { name: string; age: number; constructor(name: string, age: number) { this.name = name; this.age = age; } greet(): string { return `Hello, my name is ${this.name} and I am ${this.age} years old.`; } } let alice = new Person("Alice", 25); console.log(alice.greet());

Access Modifiers

TypeScript supports public, private, and protected.

class Person { private ssn: string; public name: string; constructor(name: string, ssn: string) { this.name = name; this.ssn = ssn; } showSSN() { return this.ssn; } } let bob = new Person("Bob", "123-45-6789"); console.log(bob.name); // Accessible // console.log(bob.ssn); // Error: private

Inheritance

Classes can extend other classes to inherit properties and methods.

class Employee extends Person { employeeId: number; constructor(name: string, age: number, employeeId: number) { super(name, age); this.employeeId = employeeId; } showEmployee() { return `${this.name}, ID: ${this.employeeId}`; } } let emp = new Employee("Charlie", 30, 101); console.log(emp.showEmployee());

Readonly Properties

Use readonly to prevent modification after initialization.

class Circle { readonly pi: number = 3.14; radius: number; constructor(radius: number) { this.radius = radius; } area() { return this.pi * this.radius * this.radius; } } let c = new Circle(5); console.log(c.area()); // c.pi = 3.1415; // Error: readonly

Conclusion

Classes in TypeScript allow you to structure code using object-oriented principles. They improve readability, reusability, and maintainability in large applications.
