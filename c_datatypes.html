C Data Types

Understanding Different Types of Data in C

What are Data Types?

Data types specify the type of data that a variable can store. They help the compiler understand how much memory to allocate and what operations can be performed.

Types of Data Types in C

  • Basic Data Types
  • Derived Data Types
  • User-defined Data Types

Basic Data Types

1. int

Used to store integer values.

int age = 20;

2. float

Used to store decimal values with single precision.

float price = 99.50;

3. double

Used to store decimal values with double precision.

double distance = 12345.6789;

4. char

Used to store single characters.

char grade = 'A';

Derived Data Types

  • Arrays
  • Pointers
  • Structures
  • Unions
int numbers[5]; int *ptr;

User-defined Data Types

  • struct
  • union
  • enum
  • typedef
struct Student { int id; char name[20]; };

Size of Data Types

The size of data types depends on the system and compiler.

printf("%lu", sizeof(int));

Conclusion

Data types are the foundation of C programming. Understanding them helps in efficient memory usage and writing error-free programs.
