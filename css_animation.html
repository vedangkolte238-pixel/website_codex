CSS Animation

Creating Motion Using CSS Animations

What is CSS Animation?

CSS animation allows elements to change style gradually over time. Animations can be created without JavaScript.

Keyframes

Animations are defined using keyframes.

@keyframes moveBox { from { left: 0; } to { left: 200px; } }

Animation Property

Apply animation to an element.

.box { position: relative; animation-name: moveBox; animation-duration: 2s; }

Animation Duration

Controls how long the animation runs.

.box { animation-duration: 3s; }

Animation Delay

Delays the start of the animation.

.box { animation-delay: 1s; }

Animation Iteration Count

Defines how many times the animation runs.

.box { animation-iteration-count: infinite; }

Animation Direction

Controls the direction of the animation.

.box { animation-direction: alternate; }

Animation Timing Function

Controls animation speed curve.

.box { animation-timing-function: ease-in-out; }

Animation Fill Mode

Defines element styles before and after animation.

.box { animation-fill-mode: forwards; }

Animation Shorthand

All animation properties in one line.

.box { animation: moveBox 2s ease-in-out infinite alternate; }

Best Practices

  • Keep animations smooth and subtle
  • Avoid excessive animations
  • Use animations to guide attention

Conclusion

CSS animations add life to web pages. They improve user interaction and visual appeal.
