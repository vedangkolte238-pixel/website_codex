JavaScript Operators

Perform Operations on Variables and Values

What are Operators?

Operators are symbols used to perform operations on variables and values. JavaScript provides different types of operators.

Arithmetic Operators

Used to perform mathematical calculations.

let a = 10; let b = 5; a + b; // Addition a - b; // Subtraction a * b; // Multiplication a / b; // Division a % b; // Modulus

Assignment Operators

Used to assign values to variables.

let x = 10; x += 5; // 15 x -= 2; // 8 x *= 3; // 30 x /= 2; // 5

Comparison Operators

Used to compare two values.

10 == "10"; // true 10 === "10"; // false 10 != 5; // true 10 > 5; // true 10 <= 10; // true

Logical Operators

Used to combine conditions.

true && false; // AND true || false; // OR !true; // NOT

String Operators

Used to combine strings.

let first = "Hello"; let second = "World"; first + " " + second;

Unary Operators

Operate on a single operand.

let num = 5; num++; num--;

Ternary Operator

A short form of if-else.

let age = 18; let result = (age >= 18) ? "Adult" : "Minor";

Type Operators

Used to check data types.

typeof "Hello"; // string typeof 25; // number

Conclusion

JavaScript operators are essential for performing calculations, making decisions, and controlling program logic.
