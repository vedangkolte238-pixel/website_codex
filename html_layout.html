HTML Layout

Structure of a Web Page

Introduction

HTML layout defines how different parts of a webpage are arranged. A typical layout includes header, navigation, main content, sidebar, and footer.

Common Layout Elements

  • <header> – Top section of the page
  • <nav> – Navigation menu
  • <main> – Main content area
  • <section> – Grouped content
  • <aside> – Sidebar content
  • <footer> – Bottom section

Basic HTML Layout Structure

<header>Header</header> <nav>Navigation</nav> <main> <section>Content</section> <aside>Sidebar</aside> </main> <footer>Footer</footer>

Why HTML Layout is Important

  • Makes content organized
  • Improves readability
  • Helps search engines understand the page
  • Makes styling easier with CSS

Conclusion

HTML layout provides a clear structure to web pages. Using proper layout elements makes your website clean, readable, and professional.
