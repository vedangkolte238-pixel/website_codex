Python Dictionary

Complete Explanation with Examples and Output

What is a Dictionary?

A dictionary is a collection of key-value pairs in Python. Dictionaries are unordered, changeable, and keys must be unique.

Creating a Dictionary

person = {"name": "Alice", "age": 25, "city": "New York"} print(person)
Output:
{'name': 'Alice', 'age': 25, 'city': 'New York'}

Accessing Dictionary Values

print(person["name"]) print(person.get("age"))
Output:
Alice
25

Adding or Updating Elements

person["email"] = "alice@example.com" person["age"] = 26 print(person)
Output:
{'name': 'Alice', 'age': 26, 'city': 'New York', 'email': 'alice@example.com'}

Removing Elements

pop()

person.pop("city") print(person)
Output:
{'name': 'Alice', 'age': 26, 'email': 'alice@example.com'}

del

del person["email"] print(person)
Output:
{'name': 'Alice', 'age': 26}

clear()

person.clear() print(person)
Output:
{}

Dictionary Operations

Keys, Values, Items

person = {"name": "Alice", "age": 26} print(person.keys()) print(person.values()) print(person.items())
Output:
dict_keys(['name', 'age'])
dict_values(['Alice', 26])
dict_items([('name', 'Alice'), ('age', 26)])

Looping Through a Dictionary

for key, value in person.items(): print(key, ":", value)
Output:
name : Alice
age : 26

Conclusion

Dictionaries are ideal for storing data with a key-value relationship. They are highly flexible, efficient, and widely used in Python programming.
