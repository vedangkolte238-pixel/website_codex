Rust Data Types

Understanding Basic and Compound Data Types

Scalar Types

Rust has primitive scalar types that represent a single value:

  • Integers: i8, i16, i32, i64, i128, u8, u16, u32, u64, u128
  • Floating-point: f32, f64
  • Boolean: bool (true/false)
  • Character: char (e.g., 'a', 'ðŸš€')

Examples

fn main() { let age: u32 = 30; let pi: f64 = 3.1415; let is_active: bool = true; let letter: char = 'R'; println!("Age: {}, Pi: {}, Active: {}, Letter: {}", age, pi, is_active, letter); }

Compound Types

Compound types can group multiple values into one:

  • Tuples: Fixed-size collection of values of different types.
  • Arrays: Fixed-size collection of values of the same type.

Tuple Example

fn main() { let person: (&str, u32) = ("Alice", 25); let (name, age) = person; println!("Name: {}, Age: {}", name, age); }

Array Example

fn main() { let numbers: [i32; 5] = [1, 2, 3, 4, 5]; println!("First number: {}", numbers[0]); }

Type Inference

Rust can automatically infer types, so explicit type annotation is optional in many cases.

fn main() { let x = 10; // inferred as i32 let y = 3.14; // inferred as f64 println!("x: {}, y: {}", x, y); }

Conclusion

Understanding Rust data types is essential for writing safe and efficient programs. Scalar and compound types allow you to model most data structures in your application.
