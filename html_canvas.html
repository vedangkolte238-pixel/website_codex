HTML Canvas

Drawing Graphics on a Web Page

Introduction

The HTML <canvas> element allows you to draw graphics such as shapes, text, images, and animations directly in the browser using JavaScript.

Basic Canvas Syntax

<canvas id="myCanvas" width="300" height="150"></canvas>

Drawing a Rectangle

<canvas id="rectCanvas" width="300" height="150"></canvas> <script> var canvas = document.getElementById('rectCanvas'); var ctx = canvas.getContext('2d'); ctx.fillStyle = '#F06529'; ctx.fillRect(50, 20, 150, 100); </script>

Drawing a Circle

<canvas id="circleCanvas" width="300" height="150"></canvas> <script> var canvas = document.getElementById('circleCanvas'); var ctx = canvas.getContext('2d'); ctx.beginPath(); ctx.arc(150, 75, 50, 0, 2 * Math.PI); ctx.fillStyle = '#F06529'; ctx.fill(); </script>

Adding Text on Canvas

ctx.font = "20px Arial"; ctx.fillStyle = "blue"; ctx.fillText("Hello Canvas", 50, 50);

Key Points

  • <canvas> defines a drawable area
  • JavaScript is used for drawing
  • Supports shapes, text, images, and animations
  • Requires specifying width and height

Conclusion

HTML Canvas is a powerful feature to create dynamic graphics and interactive web applications without needing external plugins.
