Python Basics

Complete Explanation with Examples and Output

1. Python Variables

Variables are used to store data values. Python does not require variable type declaration.

x = 10 name = "Python" print(x) print(name)
Output:
10
Python

2. Python Operators

Arithmetic Operators

a = 10 b = 3 print(a + b) print(a - b) print(a * b) print(a / b) print(a % b)
Output:
13
7
30
3.3333333333333335
1

Assignment Operators

x = 5 x += 3 print(x)
Output:
8

Relational (Comparison) Operators

a = 10 b = 5 print(a > b) print(a == b) print(a != b)
Output:
True
False
True

Logical Operators

a = True b = False print(a and b) print(a or b) print(not a)
Output:
False
True
False

Bitwise Operators

a = 5 # 101 b = 3 # 011 print(a & b) print(a | b) print(a ^ b)
Output:
1
7
6

Membership Operators

nums = [1, 2, 3, 4] print(3 in nums) print(10 not in nums)
Output:
True
True

Identity Operators

a = 10 b = 10 print(a is b) print(a is not b)
Output:
True
False

3. Conditional Statements

If Statement

age = 20 if age >= 18: print("Eligible to vote")
Output:
Eligible to vote

If-Else Statement

age = 16 if age >= 18: print("Adult") else: print("Minor")
Output:
Minor

If-Elif-Else Statement

marks = 72 if marks >= 90: print("Grade A") elif marks >= 60: print("Grade B") else: print("Grade C")
Output:
Grade B

4. Loops

For Loop

for i in range(1, 6): print(i)
Output:
1 2 3 4 5

While Loop

i = 1 while i <= 5: print(i) i += 1
Output:
1 2 3 4 5

5. Functions

Functions are reusable blocks of code.

def add(a, b): return a + b print(add(3, 5))
Output:
8
