Go Structs

Working with Structs in Go

What is a Struct?

A struct in Go is a user-defined data type that groups variables of different types into a single unit. Structs are commonly used to represent real-world entities.

Defining a Struct

type Person struct { Name string Age int City string }

Creating a Struct Variable

var p1 Person p1.Name = "Alice" p1.Age = 25 p1.City = "New York"

Struct Initialization

p2 := Person{ Name: "Bob", Age: 30, City: "London", }

Accessing Struct Fields

fmt.Println(p2.Name) fmt.Println(p2.Age)

Structs with Functions

Structs can be passed to functions just like other data types.

func printPerson(p Person) { fmt.Println(p.Name, p.Age, p.City) }

Methods on Structs

A method is a function with a receiver.

func (p Person) greet() { fmt.Println("Hello, my name is", p.Name) }

Pointer to Struct

p := Person{"Charlie", 28, "Berlin"} ptr := &p fmt.Println(ptr.Name)

Nested Structs

type Address struct { City string State string } type Employee struct { Name string Address Address }

Key Points

  • Structs group related data.
  • Fields are accessed using the dot operator.
  • Methods can be defined on structs.
  • Structs support composition.

Conclusion

Structs are one of the most important features in Go. They help create clean, structured, and maintainable code.
