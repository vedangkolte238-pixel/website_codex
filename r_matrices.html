R Matrices

Two-dimensional data structure in R programming

Introduction

A matrix in R is a two-dimensional data structure that stores elements of the same data type arranged in rows and columns.

Creating a Matrix

Matrices are created using the matrix() function.

m <- matrix(c(1, 2, 3, 4, 5, 6), nrow = 2, ncol = 3) m

Matrix by Rows

You can fill a matrix row-wise using the byrow parameter.

m_row <- matrix(c(1, 2, 3, 4, 5, 6), nrow = 2, byrow = TRUE) m_row

Accessing Matrix Elements

Elements are accessed using row and column indices.

m[1, 2] m[, 1] m[2, ]

Modifying Matrix Elements

You can modify matrix values by assigning new values.

m[1, 1] <- 100 m

Matrix Operations

R supports element-wise matrix operations.

a <- matrix(c(1, 2, 3, 4), nrow = 2) b <- matrix(c(5, 6, 7, 8), nrow = 2) a + b a * b

Matrix Multiplication

Matrix multiplication uses the %*% operator.

x <- matrix(c(1, 2, 3, 4), nrow = 2) y <- matrix(c(5, 6, 7, 8), nrow = 2) x %*% y

Matrix Functions

Common functions used with matrices.

dim(m) nrow(m) ncol(m) t(m)

Conclusion

Matrices are useful for mathematical and statistical operations in R. They form the basis for advanced data analysis techniques.
