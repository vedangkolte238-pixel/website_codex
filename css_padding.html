CSS Padding

Controlling Space Inside Elements

What is CSS Padding?

CSS padding is used to create space inside an element, between the content and its border. It improves readability and layout appearance.

Padding Properties

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Padding Example

Adding padding inside an element.

div { padding: 20px; }

Individual Padding Sides

Apply padding to specific sides.

div { padding-top: 10px; padding-right: 20px; padding-bottom: 10px; padding-left: 20px; }

Padding Shorthand

Use shorthand to set all sides at once.

div { padding: 10px 20px 10px 20px; }

Padding with Percentage

Padding can be defined using percentage values.

div { padding: 5%; }

Padding and Element Size

Padding increases the total size of an element unless box-sizing is changed.

div { width: 300px; padding: 20px; }

Box Sizing Example

Using box-sizing to control element size.

div { box-sizing: border-box; padding: 20px; }

Best Practices

  • Use padding for inner spacing
  • Combine padding with margin wisely
  • Use box-sizing for predictable layouts

Conclusion

CSS padding is essential for clean and readable designs. It helps control the spacing inside elements effectively.
