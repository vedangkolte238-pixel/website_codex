R Machine Learning

Implementing basic machine learning models in R

Introduction

Machine Learning in R allows building predictive models using supervised and unsupervised techniques. Popular packages include caret, randomForest, and e1071.

Linear Regression (Supervised Learning)

# Sample data data <- data.frame( x = c(1,2,3,4,5), y = c(2,4,5,4,5) ) # Linear regression model model <- lm(y ~ x, data = data) summary(model) # Predict new values predict(model, data.frame(x = c(6,7)))

Logistic Regression

# Sample data data <- data.frame( x = c(1,2,3,4,5,6), y = c(0,0,0,1,1,1) ) # Logistic regression model <- glm(y ~ x, data = data, family = binomial) summary(model) # Predict probabilities predict(model, data.frame(x = c(2,5)), type = "response")

Decision Tree

library(rpart) # Sample data data <- data.frame( x = c(1,2,3,4,5), y = factor(c("A","A","B","B","A")) ) # Build decision tree tree_model <- rpart(y ~ x, data = data) tree_model

Random Forest

library(randomForest) # Sample data data <- data.frame( x1 = c(1,2,3,4,5), x2 = c(5,4,3,2,1), y = factor(c("Yes","No","Yes","No","Yes")) ) # Random forest model rf_model <- randomForest(y ~ x1 + x2, data = data, ntree = 100) rf_model

k-Nearest Neighbors (kNN)

library(class) train_data <- data.frame( x = c(1,2,3,4,5), y = c(0,0,1,1,0) ) test_data <- data.frame(x = c(2,4)) knn_result <- knn(train = train_data[1], test = test_data, cl = train_data$y, k = 3) knn_result

Conclusion

R provides a variety of machine learning techniques for classification, regression, and predictive modeling. Mastering these models allows data scientists to solve real-world problems efficiently.
