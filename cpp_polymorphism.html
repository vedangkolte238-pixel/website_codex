C++ Polymorphism

Understanding Polymorphism in C++

What is Polymorphism?

Polymorphism is an OOP concept that allows a function, operator, or object to behave differently in different contexts. It comes from Greek meaning "many forms".

Types of Polymorphism in C++

  • Compile-time (Static) Polymorphism – Achieved using function overloading and operator overloading.
  • Run-time (Dynamic) Polymorphism – Achieved using inheritance and virtual functions.

1. Function Overloading (Compile-time)

You can have multiple functions with the same name but different parameters.

#include <iostream> using namespace std; class Calculator { public: int add(int a, int b) { return a + b; } double add(double a, double b) { return a + b; } }; int main() { Calculator calc; cout << calc.add(5, 10) << endl; cout << calc.add(3.5, 2.5) << endl; return 0; }

2. Operator Overloading (Compile-time)

You can redefine how operators work with user-defined types.

#include <iostream> using namespace std; class Number { public: int value; Number(int v) { value = v; } Number operator+(Number n) { return Number(value + n.value); } }; int main() { Number n1(5), n2(10); Number n3 = n1 + n2; // Uses overloaded + cout << n3.value; return 0; }

3. Run-time Polymorphism (Dynamic)

Achieved using inheritance and virtual functions. The function that is called is determined at runtime.

#include <iostream> using namespace std; class Animal { public: virtual void sound() { cout << "Animal sound\n"; } }; class Dog : public Animal { public: void sound() { cout << "Bark\n"; } }; class Cat : public Animal { public: void sound() { cout << "Meow\n"; } }; int main() { Animal* a; Dog d; Cat c; a = &d; a->sound(); // Bark a = &c; a->sound(); // Meow return 0; }

Key Points

  • Polymorphism allows flexibility and reusability in code.
  • Compile-time polymorphism is resolved during compilation.
  • Run-time polymorphism is resolved during program execution.
  • Virtual functions are essential for dynamic polymorphism.

Advantages of Polymorphism

  • Improves code readability and maintenance
  • Supports method overriding
  • Enables dynamic behavior
  • Promotes code reusability

Conclusion

Polymorphism is a powerful concept in C++ that allows objects and functions to take many forms. Understanding it is essential for advanced OOP programming.
