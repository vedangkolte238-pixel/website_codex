Python Operators

Complete Explanation with Examples and Output

What are Operators?

Operators are symbols used to perform operations on values and variables. Python provides many types of operators to handle different kinds of operations.

1. Arithmetic Operators

Used to perform mathematical calculations.

  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
  • % (Modulus)
  • // (Floor Division)
  • ** (Exponentiation)
a = 10 b = 3 print(a + b) print(a - b) print(a * b) print(a / b) print(a % b) print(a // b) print(a ** b)
Output:
13
7
30
3.3333333333333335
1
3
1000

2. Assignment Operators

Used to assign values to variables.

  • =
  • +=
  • -=
  • *=
  • /=
  • %=
x = 5 x += 3 x *= 2 print(x)
Output:
16

3. Relational (Comparison) Operators

Used to compare two values. Result is True or False.

  • ==
  • !=
  • >
  • <
  • >=
  • <=
a = 10 b = 5 print(a > b) print(a < b) print(a == b) print(a != b)
Output:
True
False
False
True

4. Logical Operators

Used to combine conditional expressions.

  • and
  • or
  • not
a = True b = False print(a and b) print(a or b) print(not a)
Output:
False
True
False

5. Bitwise Operators

Operate on binary numbers.

  • & (AND)
  • | (OR)
  • ^ (XOR)
  • ~ (NOT)
  • << (Left Shift)
  • >> (Right Shift)
a = 5 # 101 b = 3 # 011 print(a & b) print(a | b) print(a ^ b) print(~a) print(a << 1) print(a >> 1)
Output:
1
7
6
-6
10
2

6. Membership Operators

Used to check whether a value exists in a sequence.

  • in
  • not in
data = [1, 2, 3, 4] print(3 in data) print(10 not in data)
Output:
True
True

7. Identity Operators

Used to compare memory locations of objects.

  • is
  • is not
a = 10 b = 10 print(a is b) print(a is not b)
Output:
True
False

Conclusion

Python operators are essential for performing calculations, comparisons, and logical decisions. Mastering operators helps in writing efficient and meaningful Python programs.
