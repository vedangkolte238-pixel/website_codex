Swift Structs

Value Types for Storing Data in Swift

What Is a Struct?

A struct (structure) is a value type in Swift used to store related data and functionality. Unlike classes, structs are copied when assigned or passed to functions.

Defining a Struct

Structs are defined using the struct keyword.

struct Person { var name: String var age: Int }

Creating a Struct Instance

let person1 = Person(name: "Alice", age: 30) print(person1.name) // Alice print(person1.age) // 30

Mutability

To modify properties of a struct, the instance must be declared as var.

var person2 = Person(name: "Bob", age: 25) person2.age = 26

Struct Methods

Structs can have functions (methods) inside them.

struct Rectangle { var width: Double var height: Double func area() -> Double { return width * height } } let rect = Rectangle(width: 5, height: 10) print(rect.area()) // 50

Value Type Behavior

Structs are copied when assigned to a new variable.

var original = Person(name: "Charlie", age: 40) var copy = original copy.age = 41 print(original.age) // 40 print(copy.age) // 41

Structs vs Classes

  • Structs are value types, classes are reference types
  • Structs don’t support inheritance
  • Structs are simpler and safer for lightweight data

Conclusion

Structs are ideal for storing small, immutable, or value-based data in Swift. They are lightweight, safe, and widely used in Swift programming.
