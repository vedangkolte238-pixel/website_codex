C++ Templates

Learn How to Use Templates in C++

What is a Template?

A template is a feature in C++ that allows you to write generic functions or classes. It helps you create reusable code for different data types without rewriting the same logic.

Types of Templates

  • Function Templates
  • Class Templates

1. Function Templates

Function templates allow the same function to work with different data types.

#include <iostream> using namespace std; template <typename T> T add(T a, T b) { return a + b; } int main() { cout << add(5, 10) << endl; // int cout << add(3.5, 2.5) << endl; // double return 0; }

2. Class Templates

Class templates allow you to create a class that can handle different data types.

#include <iostream> using namespace std; template <typename T> class Box { private: T content; public: void setContent(T c) { content = c; } T getContent() { return content; } }; int main() { Box<int> intBox; intBox.setContent(123); cout << intBox.getContent() << endl; Box<string> strBox; strBox.setContent("Hello Templates"); cout << strBox.getContent() << endl; return 0; }

Advantages of Templates

  • Code Reusability – write once, use for any data type
  • Type Safety – compiler checks types at compile-time
  • Maintainability – fewer lines of code to maintain
  • Flexibility – works with user-defined types

Conclusion

Templates are a powerful feature in C++ that allow you to write generic and reusable code. They are widely used in C++ Standard Template Library (STL) and advanced programming.
