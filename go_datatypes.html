Go Data Types

Understand Built-in Data Types in Go

What are Data Types?

Data types specify the kind of data a variable can store. Go is a statically typed language, which means the type of a variable is known at compile time.

Basic Data Types

1. Integer Types

Used to store whole numbers.

  • int
  • int8, int16, int32, int64
  • uint, uint8, uint16, uint32, uint64
var a int = 10 var b int64 = 100

2. Floating Point Types

Used to store decimal numbers.

  • float32
  • float64
var pi float64 = 3.14

3. Boolean Type

Stores true or false values.

var isActive bool = true

4. String Type

Used to store text.

var name string = "Go Language"

Derived Data Types

1. Array

Stores multiple values of the same type.

var numbers [3]int = [3]int{1, 2, 3}

2. Slice

Dynamic version of arrays.

nums := []int{10, 20, 30}

3. Struct

Custom data type that groups variables.

type Person struct { Name string Age int }

4. Map

Stores key-value pairs.

ages := map[string]int{ "Alice": 25, "Bob": 30, }

Special Data Types

1. Pointer

Stores the memory address of a variable.

var x int = 10 var ptr *int = &x

2. Interface

Defines behavior using methods.

type Shape interface { Area() float64 }

Type Inference

Go can automatically infer data types using the := operator.

count := 5 message := "Hello Go"

Conclusion

Go provides simple yet powerful data types that help in building efficient and scalable applications. Understanding data types is the foundation of Go programming.
