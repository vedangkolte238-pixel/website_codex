C++ Installation

Set Up C++ Programming Environment

What Do You Need?

To run C++ programs, you need a compiler. A compiler converts C++ source code into machine code.

  • C++ Compiler
  • Text editor or IDE

Installing C++ on Windows

Step 1: Install a Compiler

You can use a compiler package that includes GCC.

  • Download a GCC-based compiler
  • Install it using default settings

Step 2: Verify Installation

Open Command Prompt and type:

g++ --version

If the compiler version appears, installation is successful.

Installing C++ on Linux

Step 1: Install GCC

sudo apt install g++

Step 2: Check Compiler

g++ --version

Installing C++ on macOS

Step 1: Install Compiler Tools

macOS includes compiler tools that can be installed via terminal.

xcode-select --install

Step 2: Verify

g++ --version

Writing Your First C++ Program

Create a file named hello.cpp and write:

#include <iostream> using namespace std; int main() { cout << "Hello, C++!"; return 0; }

Compile and Run

Use the following commands:

g++ hello.cpp ./a.out

Popular C++ IDEs

  • Visual Studio Code
  • Code::Blocks
  • Dev C++
  • CLion

Conclusion

After installing a compiler and IDE, you are ready to start coding in C++. Proper setup helps you write, compile, and run programs efficiently.
