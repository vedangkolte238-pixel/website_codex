CSS Grid

Two-Dimensional Layout System in CSS

What is CSS Grid?

CSS Grid is a powerful layout system that allows you to create complex web layouts using rows and columns. It is ideal for two-dimensional layouts.

Grid Container

To use Grid, an element must be a grid container.

.container { display: grid; }

Grid Columns

Defines the number and size of columns.

.container { grid-template-columns: 1fr 1fr 1fr; }

Grid Rows

Defines the height of rows.

.container { grid-template-rows: 100px auto; }

Grid Gap

Creates space between rows and columns.

.container { gap: 20px; }

Grid Areas

Names sections of the layout.

.container { grid-template-areas: "header header" "sidebar content" "footer footer"; }

Placing Grid Items

Control item placement using line numbers.

.item { grid-column: 1 / 3; grid-row: 1 / 2; }

Auto-Fill and Auto-Fit

Creates responsive grid layouts.

.container { grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); }

Aligning Grid Items

Justify Items

.container { justify-items: center; }

Align Items

.container { align-items: center; }

Best Practices

  • Use Grid for complex layouts
  • Combine Grid with Flexbox
  • Use fractional units for responsiveness

Conclusion

CSS Grid makes building complex layouts easier and cleaner. It is an essential tool for modern responsive design.
