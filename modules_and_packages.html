Python Modules and Packages

Complete Explanation with Examples and Output

What is a Module?

A module is a Python file containing functions, variables, or classes that can be imported and used in other Python programs.

Creating and Using a Module

# file: mymodule.py def greet(name): return "Hello, " + name # file: main.py import mymodule print(mymodule.greet("Alice"))
Output:
Hello, Alice

Importing Specific Functions

from mymodule import greet print(greet("Bob"))
Output:
Hello, Bob

Using Aliases

import mymodule as mm print(mm.greet("Charlie"))
Output:
Hello, Charlie

What is a Package?

A package is a collection of Python modules organized in directories. It usually contains an __init__.py file to mark the directory as a package.

Using a Package

# Directory structure: # mypackage/ # __init__.py # module1.py # module2.py from mypackage import module1, module2 module1.func1() module2.func2()

Python Built-in Modules

Python provides many built-in modules for common tasks:

  • math – mathematical functions
  • random – generate random numbers
  • os – interact with the operating system
  • sys – system-specific parameters and functions
  • datetime – date and time operations

Example: Using a Built-in Module

import math print(math.sqrt(25)) print(math.factorial(5))
Output:
5.0
120

Conclusion

Modules and packages help organize Python code, promote reusability, and simplify project management. Using them effectively makes your programs modular and efficient.
