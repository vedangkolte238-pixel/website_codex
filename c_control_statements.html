C Control Statements

Decision Making and Looping in C

What are Control Statements?

Control statements are used to control the flow of execution in a C program. They help make decisions, repeat actions, and jump to specific parts of code.

Types of Control Statements

  • Decision Making Statements
  • Looping Statements
  • Jump Statements

Decision Making Statements

1. if Statement

Executes code when the condition is true.

if (condition) { statement; } if (age >= 18) { printf("Eligible to vote"); }

2. if-else Statement

Executes one block if condition is true, otherwise another block.

if (condition) { statement1; } else { statement2; }

3. else-if Ladder

Used to check multiple conditions.

if (marks >= 90) { printf("Grade A"); } else if (marks >= 75) { printf("Grade B"); } else { printf("Grade C"); }

4. switch Statement

Used when there are multiple fixed values.

switch (choice) { case 1: printf("Option 1"); break; case 2: printf("Option 2"); break; default: printf("Invalid choice"); }

Looping Statements

1. for Loop

Used when the number of iterations is known.

for (int i = 1; i <= 5; i++) { printf("%d ", i); }

2. while Loop

Executes code while the condition is true.

int i = 1; while (i <= 5) { printf("%d ", i); i++; }

3. do-while Loop

Executes the loop at least once.

int i = 1; do { printf("%d ", i); i++; } while (i <= 5);

Jump Statements

1. break

Terminates the loop or switch statement.

for (int i = 1; i <= 10; i++) { if (i == 5) { break; } printf("%d ", i); }

2. continue

Skips the current iteration.

for (int i = 1; i <= 5; i++) { if (i == 3) { continue; } printf("%d ", i); }

3. goto

Transfers control to a labeled statement.

goto label; label: printf("Jumped here");

Conclusion

Control statements are essential for building logical and dynamic C programs. They allow decision making, repetition, and better program flow.
