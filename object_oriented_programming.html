Python Object-Oriented Programming (OOP)

Complete Explanation with Examples and Output

What is Object-Oriented Programming?

OOP is a programming paradigm based on objects and classes. It allows organizing code around real-world entities.

Class and Object

class Person: def __init__(self, name, age): self.name = name self.age = age def display(self): print("Name:", self.name) print("Age:", self.age) # Creating object p1 = Person("Alice", 25) p1.display()
Output:
Name: Alice
Age: 25

Constructor (__init__)

The __init__ method initializes object attributes when an object is created.

Methods

Functions inside a class are called methods. They define behaviors of objects.

Inheritance

Inheritance allows a class to inherit properties and methods of another class.

class Student(Person): def __init__(self, name, age, student_id): super().__init__(name, age) self.student_id = student_id def display_student(self): print("Student ID:", self.student_id) s1 = Student("Bob", 20, "S101") s1.display() s1.display_student()
Output:
Name: Bob
Age: 20
Student ID: S101

Encapsulation

Encapsulation hides internal details of an object. Use _ or __ for private attributes.

class Account: def __init__(self, balance): self.__balance = balance def deposit(self, amount): self.__balance += amount def get_balance(self): return self.__balance acc = Account(1000) acc.deposit(500) print(acc.get_balance())
Output:
1500

Polymorphism

Polymorphism allows objects of different classes to be treated similarly.

class Cat: def sound(self): print("Meow") class Dog: def sound(self): print("Woof") def make_sound(animal): animal.sound() make_sound(Cat()) make_sound(Dog())
Output:
Meow
Woof

Conclusion

OOP in Python helps structure programs using real-world entities, making code reusable, modular, and easier to maintain. Understanding classes, objects, inheritance, encapsulation, and polymorphism is essential for advanced Python programming.
