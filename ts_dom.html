TypeScript DOM

Access and Manipulate HTML Elements Using TypeScript

What is DOM?

The Document Object Model (DOM) represents the structure of a web page. TypeScript can interact with the DOM while providing type safety for elements.

Accessing Elements

Use document.getElementById or querySelector.

const myButton = document.getElementById("btn") as HTMLButtonElement; const myInput = document.querySelector("input") as HTMLInputElement; console.log(myButton); console.log(myInput.value);

Adding Event Listeners

Attach events to elements safely.

myButton.addEventListener("click", () => { alert("Button clicked! Input value: " + myInput.value); });

Modifying Elements

Change text, style, or attributes of elements.

const myParagraph = document.getElementById("para") as HTMLParagraphElement; myParagraph.textContent = "This text is updated by TypeScript!"; myParagraph.style.color = "blue";

Creating Elements

Create and append new elements dynamically.

const newDiv = document.createElement("div"); newDiv.textContent = "Hello from TypeScript!"; document.body.appendChild(newDiv);

Removing Elements

const removeDiv = document.getElementById("removeMe"); if(removeDiv) { removeDiv.remove(); }

Conclusion

TypeScript makes DOM manipulation safer by providing type checking for HTML elements. It helps prevent runtime errors when working with web pages.
