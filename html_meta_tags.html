HTML Meta Tags

Providing Information About the Web Page

Introduction

Meta tags are used in the <head> section of an HTML document to provide metadata about the page. Metadata is information about the page such as description, keywords, author, and viewport settings.

Why Meta Tags Are Important

  • Improve search engine optimization (SEO)
  • Provide information to browsers and social media
  • Control page behavior like refresh or charset

Basic Syntax

<meta name="name" content="value">

Common Meta Tags

1. Charset

Specifies the character encoding for the HTML document.

<meta charset="UTF-8">

2. Viewport

Controls layout on mobile devices for responsive design.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

3. Description

Provides a summary of the page for search engines.

<meta name="description" content="This is a sample HTML page">

4. Keywords

Helps search engines understand important topics.

<meta name="keywords" content="HTML, CSS, JavaScript">

5. Author

Specifies the author of the page.

<meta name="author" content="Your Name">

6. Refresh

Automatically refresh the page after a certain time.

<meta http-equiv="refresh" content="30"> <!-- Refresh every 30 seconds -->

Key Points to Remember

  • Meta tags go inside the <head> section
  • They do not display on the page
  • Useful for SEO, responsive design, and page info

Conclusion

HTML meta tags provide essential information about a web page to browsers, search engines, and social media platforms. Using them properly improves page accessibility, SEO, and responsiveness.
