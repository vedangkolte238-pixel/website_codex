Python Advanced Topics

Overview of Key Advanced Concepts with Examples

1. Decorators

Decorators allow you to modify the behavior of a function or class without changing its code.

def decorator(func): def wrapper(): print("Before function call") func() print("After function call") return wrapper @decorator def greet(): print("Hello World!") greet()
Output:
Before function call
Hello World!
After function call

2. Generators

Generators allow you to iterate over data lazily, producing items one at a time using yield.

def numbers(): for i in range(1, 4): yield i for num in numbers(): print(num)
Output:
1
2
3

3. List Comprehensions

Concise way to create lists using expressions.

squares = [x**2 for x in range(5)] print(squares)
Output:
[0, 1, 4, 9, 16]

4. Lambda Functions

Anonymous functions created using lambda.

add = lambda a, b: a + b print(add(5, 3))
Output:
8

5. Context Managers

Use with to handle resources like files efficiently.

with open("example.txt", "w") as f: f.write("Hello Context Manager!")

6. Exception Handling Advanced

Catching multiple exceptions and raising custom exceptions.

try: x = int("abc") except (ValueError, TypeError) as e: print("Error:", e)
Output:
Error: invalid literal for int() with base 10: 'abc'

7. Modules and Packages

Advanced usage includes creating reusable packages and importing modules efficiently.

import os print(os.getcwd())
Output:
C:\Your\Current\Directory

8. OOP Advanced

Advanced OOP concepts: inheritance, polymorphism, encapsulation, abstract classes, and multiple inheritance.

from abc import ABC, abstractmethod class Shape(ABC): @abstractmethod def area(self): pass class Square(Shape): def __init__(self, side): self.side = side def area(self): return self.side * self.side s = Square(4) print(s.area())
Output:
16

9. File Handling Advanced

Reading and writing large files efficiently or in binary mode.

with open("example.txt", "rb") as f: content = f.read() print(content)

Conclusion

Advanced Python topics like decorators, generators, context managers, and OOP allow writing efficient, clean, and reusable code. Mastering these topics is essential for professional Python programming.
