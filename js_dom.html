JavaScript DOM

Document Object Model Explained Simply

What is DOM?

DOM stands for Document Object Model. It represents an HTML document as a tree of objects. JavaScript uses the DOM to access and change HTML content, structure, and styles.

DOM Tree Structure

The browser converts HTML into a tree-like structure. Each element becomes an object called a node.

Document └── html ├── head └── body

Accessing DOM Elements

JavaScript can find HTML elements using different methods.

getElementById()

document.getElementById("title");

getElementsByClassName()

document.getElementsByClassName("box");

getElementsByTagName()

document.getElementsByTagName("p");

querySelector()

document.querySelector(".container");

querySelectorAll()

document.querySelectorAll("li");

Changing HTML Content

You can change text and HTML using JavaScript.

document.getElementById("msg").innerText = "Hello DOM";

Changing Styles

CSS styles can be changed dynamically.

document.getElementById("box").style.color = "red";

Adding and Removing Elements

Create Element

let p = document.createElement("p"); p.innerText = "New Paragraph";

Append Element

document.body.appendChild(p);

Remove Element

p.remove();

DOM Events

Events allow JavaScript to respond to user actions.

button.addEventListener("click", function() { alert("Button clicked"); });

Why DOM is Important

  • Makes web pages interactive
  • Allows dynamic content updates
  • Controls HTML and CSS using JavaScript

Conclusion

The DOM is the bridge between HTML and JavaScript. Understanding DOM manipulation is essential for building dynamic web applications.
