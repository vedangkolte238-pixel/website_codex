HTML Audio and Video

Embed audio and video files in web pages

Introduction

HTML allows you to add multimedia content such as audio and video directly to web pages using the <audio> and <video> tags.

HTML Audio

The <audio> tag is used to play sound files.

<audio controls> <source src="music.mp3" type="audio/mpeg"> </audio>

Audio Attributes

  • controls – shows play and pause buttons
  • autoplay – plays audio automatically
  • loop – repeats the audio

HTML Video

The <video> tag is used to play video files.

<video width="400" controls> <source src="movie.mp4" type="video/mp4"> </video>

Video Attributes

  • controls – shows video controls
  • width – sets video width
  • height – sets video height
  • poster – image shown before playing

Multiple Source Files

You can provide multiple formats for better browser support.

<video controls> <source src="video.mp4" type="video/mp4"> <source src="video.webm" type="video/webm"> </video>

Important Points

  • Always use the controls attribute
  • Keep file size optimized
  • Use supported formats like MP3 and MP4

Conclusion

HTML audio and video elements make websites more interactive. They are widely used for tutorials, presentations, and entertainment.
