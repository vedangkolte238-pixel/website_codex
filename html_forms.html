HTML Forms

Collect user input using forms

Introduction

HTML forms are used to collect data from users. Forms contain input fields like text boxes, radio buttons, checkboxes, and buttons.

Basic Form Structure

<form> <input type="text"> </form>

Text Input Field

<label>Name:</label> <input type="text" placeholder="Enter name">

Password Input

<label>Password:</label> <input type="password">

Radio Buttons

<input type="radio" name="gender"> Male <input type="radio" name="gender"> Female

Checkboxes

<input type="checkbox"> HTML <input type="checkbox"> CSS <input type="checkbox"> JavaScript

Dropdown List

<select> <option>BCA</option> <option>MCA</option> </select>

Textarea

<textarea rows="4"></textarea>

Submit Button

<input type="submit" value="Submit">

Important Form Elements

  • <form> – form container
  • <input> – user input
  • <textarea> – multi-line text
  • <select> – dropdown
  • <option> – dropdown item

Conclusion

HTML forms allow interaction between users and websites. They are widely used for login pages, registration forms, and feedback forms.
