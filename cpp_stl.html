C++ Standard Template Library (STL)

Introduction to C++ STL

What is STL?

STL stands for Standard Template Library. It is a powerful library in C++ that provides predefined classes and functions for common data structures and algorithms.

Main Components of STL

  • Containers: Store data (vector, list, deque, stack, queue, map, set)
  • Algorithms: Perform operations on containers (sort, find, reverse, etc.)
  • Iterators: Used to traverse elements of containers

1. Containers Example

Vector is a dynamic array in STL:

#include <iostream> #include <vector> using namespace std; int main() { vector<int> numbers; numbers.push_back(10); numbers.push_back(20); numbers.push_back(30); for(int i : numbers) { cout << i << " "; } return 0; }

2. Iterators Example

Iterators are used to navigate through containers:

#include <iostream> #include <vector> using namespace std; int main() { vector<int> numbers = {1, 2, 3, 4, 5}; vector<int>::iterator it; for(it = numbers.begin(); it != numbers.end(); ++it) { cout << *it << " "; } return 0; }

3. Algorithms Example

STL provides many useful algorithms:

#include <iostream> #include <vector> #include <algorithm> using namespace std; int main() { vector<int> numbers = {5, 2, 9, 1, 7}; sort(numbers.begin(), numbers.end()); for(int i : numbers) { cout << i << " "; } return 0; }

Advantages of STL

  • Code Reusability – no need to implement common structures
  • Efficiency – highly optimized and tested
  • Faster Development – reduces coding effort
  • Consistency – standardized across C++ programs

Conclusion

STL is an essential part of modern C++ programming. It simplifies working with data structures and algorithms, and is widely used in competitive programming and real-world applications.
