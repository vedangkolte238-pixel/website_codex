R Vectors

Basic data structure for storing multiple values in R

Introduction

A vector is the most basic data structure in R. It stores multiple values of the same data type.

Creating Vectors

Vectors are created using the c() function.

numbers <- c(1, 2, 3, 4, 5) names <- c("A", "B", "C") logical_values <- c(TRUE, FALSE, TRUE)

Vector Data Types

All elements in a vector must be of the same data type.

mixed <- c(1, "R", TRUE) mixed

Accessing Vector Elements

Elements can be accessed using index positions.

numbers <- c(10, 20, 30, 40) numbers[1] numbers[2:3]

Modifying Vectors

You can modify vector elements by assigning new values.

numbers[1] <- 100 numbers

Vector Operations

R performs element-wise operations on vectors.

a <- c(1, 2, 3) b <- c(4, 5, 6) a + b a * b

Vector Functions

R provides many built-in functions for vectors.

sum(numbers) mean(numbers) length(numbers) max(numbers) min(numbers)

Sequences and Repetition

Vectors can be created using sequences and repetition.

seq(1, 10, by = 2) rep(5, times = 4)

Conclusion

Vectors are the foundation of R programming. Understanding vectors is essential before working with lists, matrices, and data frames.
