HTML Entities

Displaying Reserved and Special Characters

Introduction

HTML entities are used to display special characters that have a reserved meaning in HTML or characters that are not easily typed on a keyboard.

Why HTML Entities Are Needed

  • To display reserved characters like < and >
  • To show special symbols
  • To avoid breaking HTML code

Basic Syntax

An HTML entity starts with an ampersand (&) and ends with a semicolon (;).

&entity_name; &#entity_number;

Common HTML Entities

Character Entity Name Entity Number
< &lt; &#60;
> &gt; &#62;
& &amp; &#38;
" &quot; &#34;
' &apos; &#39;

Non-Breaking Space

The non-breaking space prevents the browser from breaking lines at that position.

Hello&nbsp;World

Symbols and Currency

&copy; → © &reg; → ® &euro; → € &trade; → ™

Using Entities in Text

<p>5 &lt; 10 and 10 &gt; 5</p>

Key Points

  • Entities start with & and end with ;
  • They prevent HTML syntax errors
  • They display special characters safely

Conclusion

HTML entities help display special and reserved characters correctly in web pages. They are essential for writing safe and readable HTML content.
