HTML Tags

Learn about different HTML tags and their usage

Introduction

HTML tags are the basic building blocks of a web page. Tags are used to create elements, and most come in pairs with an opening and closing tag. Some tags are self-closing and do not need an end tag.

Basic Syntax

<tagname>Content</tagname>

Example:

<p>This is a paragraph.</p>

Common HTML Tags

  • <h1> to <h6> – Headings, h1 is the largest
  • <p> – Paragraph
  • <a> – Hyperlink
  • <img> – Image (self-closing)
  • <ul>, <ol>, <li> – Lists
  • <div> – Block-level container
  • <span> – Inline container
  • <form> – Form for user input
  • <input> – Input field (self-closing)
  • <button> – Button element
  • <table>, <tr>, <td>, <th> – Table structure
  • <br> – Line break (self-closing)
  • <hr> – Horizontal rule (self-closing)
  • <header>, <footer>, <main>, <section>, <article> – Semantic HTML tags

Self-closing Tags

Some tags do not need a closing tag. They are self-closing:

<br> <hr> <img src="image.jpg" alt="Image"> <input type="text">

Nesting Tags

HTML elements can be nested inside other elements:

<div> <h1>Heading inside div</h1> <p>Paragraph inside div</p> </div>

Conclusion

HTML tags are the foundation of web pages. Learning the most commonly used tags and their correct usage is essential before moving to styling with CSS or interactivity with JavaScript.
