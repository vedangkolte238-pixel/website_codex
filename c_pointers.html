C Pointers

Understanding Memory Addressing in C

What is a Pointer?

A pointer is a variable that stores the memory address of another variable. Pointers allow direct access to memory locations.

Why Use Pointers?

  • Efficient memory usage
  • Allows call by reference
  • Used in dynamic memory allocation
  • Essential for arrays and strings

Pointer Declaration

Syntax to declare a pointer:

data_type *pointer_name;

Example of Pointer Declaration

int *ptr;

Pointer Initialization

Assigning the address of a variable to a pointer.

int num = 10; int *ptr = #

Accessing Value Using Pointer

The dereference operator (*) is used to access the value stored at the address.

printf("%d", *ptr); // Output: 10

Complete Example Program

#include <stdio.h> int main() { int num = 20; int *ptr = # printf("Value of num: %d\n", num); printf("Address of num: %p\n", &num); printf("Value using pointer: %d\n", *ptr); return 0; }

Pointer to Pointer

A pointer that stores the address of another pointer.

int num = 5; int *p = # int **pp = &p;

Access Using Pointer to Pointer

printf("%d", **pp); // Output: 5

Pointers and Arrays

Array name acts as a pointer to the first element.

int arr[3] = {10, 20, 30}; int *p = arr; printf("%d", *p); // 10 printf("%d", *(p+1)); // 20

Null Pointer

A pointer that does not point to any valid memory location.

int *ptr = NULL;

Advantages of Pointers

  • Direct memory access
  • Improves program efficiency
  • Used in advanced data structures
  • Supports dynamic memory allocation

Disadvantages of Pointers

  • Complex to understand
  • Can cause memory errors
  • Risk of segmentation faults

Conclusion

Pointers are a powerful feature of C. They provide flexibility and efficiency but must be used carefully.
