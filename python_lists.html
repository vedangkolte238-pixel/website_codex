Python List

Complete Explanation with Examples and Output

What is a List?

A list is a collection of items stored in a single variable. Lists are ordered, changeable, and allow duplicate values.

Creating a List

numbers = [10, 20, 30, 40] print(numbers)
Output:
[10, 20, 30, 40]

Accessing List Elements

print(numbers[0]) print(numbers[2]) print(numbers[-1])
Output:
10
30
40

List Slicing

print(numbers[1:3]) print(numbers[:2]) print(numbers[2:])
Output:
[20, 30]
[10, 20]
[30, 40]

Adding Elements

append()

numbers.append(50) print(numbers)
Output:
[10, 20, 30, 40, 50]

insert()

numbers.insert(1, 15) print(numbers)
Output:
[10, 15, 20, 30, 40, 50]

extend()

numbers.extend([60, 70]) print(numbers)
Output:
[10, 15, 20, 30, 40, 50, 60, 70]

Removing Elements

remove()

numbers.remove(15) print(numbers)
Output:
[10, 20, 30, 40, 50, 60, 70]

pop()

numbers.pop() print(numbers)
Output:
[10, 20, 30, 40, 50, 60]

del

del numbers[0] print(numbers)
Output:
[20, 30, 40, 50, 60]

List Operations

Concatenation (+)

a = [1, 2] b = [3, 4] print(a + b)
Output:
[1, 2, 3, 4]

Repetition (*)

print(a * 2)
Output:
[1, 2, 1, 2]

Looping Through a List

for item in numbers: print(item)
Output:
20
30
40
50
60

Common List Methods

nums = [4, 1, 3, 2] nums.sort() print(nums) print(len(nums))
Output:
[1, 2, 3, 4]
4

Conclusion

Lists are flexible and widely used data structures in Python. They allow storing, modifying, and accessing data efficiently.
