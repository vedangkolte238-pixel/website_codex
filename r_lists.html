R Lists

Flexible data structure to store different types of elements

Introduction

A list in R is a collection of elements that can be of different data types. It is one of the most flexible data structures in R.

Creating a List

Lists are created using the list() function.

my_list <- list( number = 10, text = "R Programming", logical = TRUE, vector = c(1, 2, 3) ) my_list

Accessing List Elements

Elements in a list can be accessed in different ways.

my_list$number my_list[[2]] my_list["text"]

Modifying List Elements

You can modify existing elements or add new ones.

my_list$number <- 50 my_list$new_value <- "Data Science" my_list

Removing List Elements

Elements can be removed by assigning NULL.

my_list$text <- NULL my_list

Nested Lists

A list can contain another list inside it.

nested_list <- list( a = 1, b = list(x = 10, y = 20) ) nested_list

List Functions

R provides built-in functions to work with lists.

length(my_list) names(my_list) str(my_list)

Converting Lists

Lists can be converted to other data structures.

unlist(my_list)

Conclusion

Lists are powerful and flexible. They are widely used in R to store complex data and results.
