Swift Functions

Reusable Blocks of Code in Swift

What Is a Function?

A function is a block of reusable code that performs a specific task. Functions help make programs modular, readable, and easy to maintain.

Defining a Function

Functions in Swift are defined using the func keyword.

func greet() { print("Hello, Swift!") }

Calling a Function

Call a function by using its name followed by parentheses.

greet()

Function with Parameters

Functions can accept input values called parameters.

func greetUser(name: String) { print("Hello, \(name)!") } greetUser(name: "Alex")

Function with Return Value

Functions can return a value using the return keyword.

func add(a: Int, b: Int) -> Int { return a + b } let result = add(a: 5, b: 3)

Multiple Parameters

func calculateArea(length: Double, width: Double) -> Double { return length * width }

Default Parameters

Swift allows default values for parameters.

func greetAgain(name: String = "Guest") { print("Welcome, \(name)") } greetAgain() greetAgain(name: "Sam")

External and Internal Parameter Names

func multiply(_ a: Int, by b: Int) -> Int { return a * b } multiply(4, by: 5)

Function Without Return

Functions that do not return a value have a return type of Void.

func showMessage() { print("No return value") }

Nested Functions

Functions can be defined inside other functions.

func outerFunction() { func innerFunction() { print("Inside inner function") } innerFunction() }

Conclusion

Functions are essential in Swift for organizing code, reducing repetition, and improving clarity.
