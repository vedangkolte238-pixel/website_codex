JavaScript Arrays

Store and Manage Multiple Values

What is an Array?

An array is a variable that can store multiple values in a single place. Each value in an array has an index number starting from 0.

Creating an Array

let fruits = ["Apple", "Banana", "Mango"];

Accessing Array Elements

let fruits = ["Apple", "Banana", "Mango"]; console.log(fruits[0]); // Apple console.log(fruits[2]); // Mango

Changing Array Values

let fruits = ["Apple", "Banana", "Mango"]; fruits[1] = "Orange"; console.log(fruits);

Array Length

The length property returns the number of elements.

let fruits = ["Apple", "Banana", "Mango"]; console.log(fruits.length);

Common Array Methods

push() – Add at End

let numbers = [1, 2, 3]; numbers.push(4);

pop() – Remove from End

let numbers = [1, 2, 3]; numbers.pop();

shift() – Remove from Start

let numbers = [1, 2, 3]; numbers.shift();

unshift() – Add at Start

let numbers = [1, 2, 3]; numbers.unshift(0);

Looping Through an Array

let fruits = ["Apple", "Banana", "Mango"]; for (let i = 0; i < fruits.length; i++) { console.log(fruits[i]); }

forEach() Method

let fruits = ["Apple", "Banana", "Mango"]; fruits.forEach(function(item) { console.log(item); });

Array with Mixed Data

let data = ["John", 25, true];

Why Use Arrays?

  • Store multiple values easily
  • Organize related data
  • Work efficiently with loops
  • Essential for real applications

Conclusion

Arrays are a core part of JavaScript. They help manage collections of data and are used in almost every program.
