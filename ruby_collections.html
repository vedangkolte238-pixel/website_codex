Ruby Collections

Working with Arrays, Hashes, and Ranges

Introduction

Ruby provides several collection types for storing and organizing multiple values, including Arrays, Hashes, Ranges, and Sets.

Arrays

Arrays are ordered collections of objects.

# Creating arrays fruits = ["Apple", "Banana", "Cherry"] # Accessing elements puts fruits[0] # Apple # Adding elements fruits << "Orange" # Iterating fruits.each do |fruit| puts fruit end

Hashes

Hashes store key-value pairs.

# Creating a hash person = { "name" => "Alice", "age" => 25, "city" => "New York" } # Accessing values puts person["name"] # Alice # Adding a new key person["country"] = "USA" # Iterating person.each do |key, value| puts "#{key}: #{value}" end

Ranges

Ranges represent a sequence of values.

# Numeric range numbers = (1..5).to_a # [1,2,3,4,5] # Alphabetic range letters = ("a".."e").to_a # ["a","b","c","d","e"] # Iterating numbers.each { |n| puts n }

Set

Sets are collections of unique elements (require 'set').

require 'set' unique_numbers = Set.new([1,2,2,3,4]) unique_numbers.add(5) unique_numbers.each { |n| puts n }

Applications

  • Storing lists of values with arrays
  • Mapping keys to values using hashes
  • Creating ranges for iteration and sequences
  • Ensuring uniqueness with sets

Conclusion

Ruby collections provide flexible and powerful ways to store and manipulate multiple values efficiently. Choosing the right collection type depends on the problem and required operations.
