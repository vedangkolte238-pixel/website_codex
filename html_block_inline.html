HTML Block vs Inline Elements

Understand how elements behave in a web page

Introduction

HTML elements are mainly divided into two types: Block-level elements and Inline elements.

Block-Level Elements

Block elements always start on a new line and take full width.

<div>This is a block element</div> <p>This is another block element</p>
Block element example

Common Block Elements

  • <div>
  • <p>
  • <h1> to <h6>
  • <section>
  • <article>

Inline Elements

Inline elements do not start on a new line and only take required width.

<span>Inline text</span> <strong>Bold text</strong>
Inline element one Inline element two

Common Inline Elements

  • <span>
  • <a>
  • <strong>
  • <em>
  • <img>

Main Differences

Block elements: - Start on a new line - Take full width Inline elements: - Do not start on new line - Take only content width

Conclusion

Understanding block and inline elements helps in designing page layouts properly. They control how content flows on a web page.
