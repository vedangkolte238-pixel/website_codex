JavaScript Syntax

Understanding the Basic Structure of JavaScript

What is JavaScript Syntax?

JavaScript syntax is the set of rules that defines how JavaScript programs are written and executed. It controls how statements, variables, functions, and expressions are structured.

JavaScript Statements

A JavaScript program is made up of statements. Each statement tells the browser what action to perform.

console.log("Hello World");

Semicolons

Semicolons are used to separate statements. They are optional but recommended for clean and error-free code.

let x = 10; let y = 20; console.log(x + y);

JavaScript Comments

Comments are ignored by JavaScript and used to explain code.

Single-line Comment

// This is a single-line comment

Multi-line Comment

/* This is a multi-line comment */

Case Sensitivity

JavaScript is case-sensitive. Variable names with different cases are treated as different variables.

let name = "Alex"; let Name = "John";

Whitespace

JavaScript ignores extra spaces, tabs, and line breaks. They are added to make code more readable.

let total = 10 + 20;

Code Blocks

Curly braces are used to define blocks of code.

if (x > 5) { console.log("x is greater than 5"); }

JavaScript Keywords

Keywords are reserved words with special meaning in JavaScript.

let, var, const, if, else, function, return

Conclusion

Understanding JavaScript syntax is the first step toward writing effective programs. Once you know the rules, you can build logic, functions, and interactive applications with confidence.
