R Installation and Setup

Step-by-step guide to install and configure R

What is R?

R is a programming language and software environment used for statistical computing, data analysis, and data visualization.

System Requirements

  • Windows, macOS, or Linux operating system
  • Minimum 2 GB RAM (4 GB recommended)
  • Internet connection for downloading packages

Step 1: Download R

Download R from the official CRAN (Comprehensive R Archive Network) website.

https://cran.r-project.org

Step 2: Install R on Windows

  • Open the CRAN website
  • Click on Download R for Windows
  • Download the base installer
  • Run the installer and follow the setup instructions

Step 3: Install R on macOS

  • Open the CRAN website
  • Click on Download R for macOS
  • Download the .pkg file
  • Open the file and complete the installation

Step 4: Install R on Linux

Example for Ubuntu:

sudo apt update sudo apt install r-base

Step 5: Install RStudio (Recommended)

RStudio is an IDE that makes working with R easier and more productive.

https://posit.co/download/rstudio-desktop/

Step 6: Verify Installation

Open R or RStudio and run the following command:

print("R is installed successfully")

Step 7: Install R Packages

Packages extend R functionality.

install.packages("ggplot2") library(ggplot2)

Conclusion

After installing R and RStudio, you are ready to start data analysis and visualization. A proper setup ensures smooth learning and development with R.
