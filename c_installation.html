C Language Installation

Set Up C Programming Environment

Requirements

  • A computer with Windows, Linux, or macOS
  • C compiler (GCC recommended)
  • Text editor or IDE

Install C on Windows

To run C programs on Windows, you need a compiler like GCC. MinGW is commonly used for this purpose.

Steps:

  • Install MinGW on your system
  • Select GCC compiler during installation
  • Add MinGW bin folder to system PATH
  • Verify installation using command prompt
gcc --version

Install C on Linux

Most Linux distributions come with GCC or allow easy installation.

sudo apt update sudo apt install gcc

Install C on macOS

On macOS, C compiler can be installed using Xcode Command Line Tools.

xcode-select --install

Choose an Editor or IDE

  • VS Code
  • Code::Blocks
  • Dev-C++
  • Sublime Text

Run Your First C Program

After installation, create a file with .c extension and run:

gcc hello.c ./a.out

Conclusion

Once the compiler is installed, you are ready to start writing and running C programs. Proper setup ensures smooth development.
