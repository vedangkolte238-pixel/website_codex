HTML Introduction

Learn HTML in detail for web development

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It structures content into elements such as headings, paragraphs, lists, images, links, tables, and more.

HTML Elements and Tags

HTML consists of elements defined by tags. Most tags have an opening and closing tag:

This is a paragraph.

This is a heading
  • <h1> to <h6> – Headings, h1 is the largest
  • <p> – Paragraphs
  • <a> – Hyperlinks
  • <img> – Images
  • <ul>, <ol>, <li> – Unordered and ordered lists
  • <div> – Section or container
  • <span> – Inline container

HTML Attributes

Attributes provide additional information about an element:

  • href – URL for links
  • src – Image source
  • alt – Alternative text for images
  • id – Unique identifier
  • class – Class name for styling
  • title – Tooltip text

Semantic HTML

Semantic elements provide meaning:

  • <header> – Header section
  • <footer> – Footer section
  • <main> – Main content
  • <article> – Article content
  • <section> – Section of content
  • <nav> – Navigation links

Conclusion

HTML is the backbone of web development. Learning HTML thoroughly allows you to create structured, meaningful, and accessible web pages. Next steps include learning CSS for styling and JavaScript for interactivity.
