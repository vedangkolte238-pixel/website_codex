Java Classes

Building blocks of Java programs

What is a Class?

A class is a blueprint or template used to create objects. It defines properties (variables) and behaviors (methods).

Why Use Classes?

  • To organize code
  • To model real-world entities
  • To support object-oriented programming
  • To reuse code easily

Basic Class Structure

class ClassName { // variables // methods }

Example: Simple Class

class Student { int id; String name; }

Creating an Object

An object is an instance of a class.

Student s1 = new Student();

Accessing Class Members

s1.id = 101; s1.name = "Rahul";

Class with Method

class Student { int id; String name; void display() { System.out.println(id); System.out.println(name); } }

Calling a Method

Student s1 = new Student(); s1.id = 101; s1.name = "Rahul"; s1.display();

Multiple Objects

Student s1 = new Student(); Student s2 = new Student(); s1.name = "Amit"; s2.name = "Neha";

Class with Constructor

A constructor initializes objects. It has the same name as the class.

class Student { int id; String name; Student(int i, String n) { id = i; name = n; } }

Constructor Example

Student s1 = new Student(101, "Rahul"); Student s2 = new Student(102, "Anita");

Main Method Inside Class

class Test { public static void main(String[] args) { Student s1 = new Student(101, "Rahul"); s1.display(); } }

Key Points

  • Class is a blueprint
  • Object is a real entity
  • Variables store data
  • Methods define behavior

Conclusion

Classes are the foundation of Java programming. Understanding classes helps you learn objects, inheritance, polymorphism, and other OOP concepts.
