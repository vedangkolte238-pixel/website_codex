Pandas Visualization

Creating Simple Plots with Pandas

Introduction

Pandas integrates with Matplotlib to provide simple plotting capabilities directly from Series and DataFrames. You can quickly visualize data for analysis and insights.

Basic Plotting

import pandas as pd import matplotlib.pyplot as plt data = {'Month': ['Jan', 'Feb', 'Mar', 'Apr'], 'Sales': [2500, 3000, 2800, 3500]} df = pd.DataFrame(data) # Line plot df.plot(x='Month', y='Sales', kind='line', title='Monthly Sales') plt.show()

Other Plot Types

# Bar plot df.plot(x='Month', y='Sales', kind='bar', title='Monthly Sales') plt.show() # Horizontal bar df.plot(x='Month', y='Sales', kind='barh', title='Monthly Sales') plt.show() # Pie chart (using Series) df['Sales'].plot(kind='pie', labels=df['Month'], autopct='%1.1f%%', title='Sales Distribution') plt.show() # Histogram df['Sales'].plot(kind='hist', bins=4, title='Sales Histogram') plt.show()

Customizing Plots

# Line plot with style df.plot(x='Month', y='Sales', kind='line', color='green', linestyle='--', marker='o', title='Monthly Sales') plt.xlabel('Month') plt.ylabel('Sales Amount') plt.grid(True) plt.show()

Applications

  • Quick data visualization for analysis and insights
  • Checking trends, distributions, and comparisons
  • Preparing charts for reports and presentations
  • Exploratory Data Analysis (EDA) before modeling

Conclusion

Pandas’ plotting features make it easy to visualize Series and DataFrame data. For more advanced visualization, Pandas works seamlessly with Matplotlib, Seaborn, and Plotly.
