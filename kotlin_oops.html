Object-Oriented Programming in Kotlin

What is OOP?

Object-Oriented Programming (OOP) is a programming approach based on objects rather than functions and logic. It helps in organizing complex programs efficiently.

Core OOPs Concepts

  • Class
  • Object
  • Encapsulation
  • Inheritance
  • Polymorphism
  • Abstraction

1. Class and Object

A class is a blueprint, and an object is its instance.

class Animal { fun sound() { println("Animal makes sound") } } val dog = Animal() dog.sound()

2. Encapsulation

Encapsulation hides internal data and allows access only through methods.

class Account { private var balance = 1000 fun getBalance(): Int { return balance } }

3. Inheritance

Inheritance allows one class to acquire properties of another class.

open class Vehicle { fun move() { println("Vehicle is moving") } } class Bike : Vehicle()

4. Polymorphism

Polymorphism allows methods to behave differently based on the object.

open class Shape { open fun draw() { println("Drawing shape") } } class Circle : Shape() { override fun draw() { println("Drawing circle") } }

5. Abstraction

Abstraction hides implementation details and shows only essential features.

abstract class Animal { abstract fun sound() } class Cat : Animal() { override fun sound() { println("Meow") } }

Conclusion

OOP concepts in Kotlin make programs modular, reusable, and easier to maintain. They are essential for building real-world applications.
