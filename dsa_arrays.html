Learn About Arrays in Data Structures & Algorithms

What Is an Array?

An array is a collection of elements of the same type stored in contiguous memory locations. It allows fast access to elements using an index.

Features of Arrays

  • Fixed size (in most languages)
  • Elements are of the same type
  • Access elements using an index (0-based)
  • Efficient for searching and iteration

Declaring an Array

// Example in C int numbers[5]; // Example in C++ int numbers[5]; // Example in Java int[] numbers = new int[5]; // Example in Python numbers = [1, 2, 3, 4, 5]

Accessing Array Elements

numbers[0] = 10; // First element int x = numbers[2]; // Third element

Common Operations on Arrays

  • Traversal: Visiting each element sequentially
  • Insertion: Add an element at a specific position
  • Deletion: Remove an element from a specific position
  • Search: Find an element (linear or binary search)
  • Update: Modify an element at a specific index

Example: Traversal in C

#include <stdio.h> int main() { int numbers[5] = {1, 2, 3, 4, 5}; for(int i = 0; i < 5; i++) { printf("%d ", numbers[i]); } return 0; }

Advantages of Arrays

  • Fast access using index
  • Easy to iterate and manipulate
  • Simple to implement

Disadvantages of Arrays

  • Fixed size (cannot easily grow or shrink)
  • Insertion and deletion at middle is costly

Conclusion

Arrays are one of the most fundamental data structures. They are efficient for accessing elements by index and are widely used in algorithms and programming.
