HTML Symbols

Displaying Mathematical, Currency, and Special Symbols

Introduction

HTML symbols are special characters such as currency signs, mathematical operators, and arrows that are displayed using HTML entities.

Why HTML Symbols Are Used

  • Some symbols are not available on the keyboard
  • Ensures symbols display correctly in all browsers
  • Prevents HTML syntax issues

Basic Syntax

HTML symbols are written using entity names or entity numbers.

&symbol_name; &#symbol_number;

Currency Symbols

Arrow Symbols

&larr; → ← &rarr; → → &uarr; → ↑ &darr; → ↓

Copyright and Legal Symbols

&copy; → © &reg; → ® &trade; → ™

Using Symbols in Text

<p>Price: ₹500 &copy; 2025</p>

Key Points

  • Symbols are written using HTML entities
  • They start with & and end with ;
  • They work in all browsers

Conclusion

HTML symbols help display special characters correctly and professionally. They are useful for currency, math, arrows, and legal symbols in web pages.
