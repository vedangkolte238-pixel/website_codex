JavaScript Events

Responding to User Actions in JavaScript

What is an Event?

An event is an action performed by the user or the browser. JavaScript can detect these actions and execute code in response.

Common JavaScript Events

  • click
  • mouseover
  • mouseout
  • keydown
  • keyup
  • submit
  • load

Event Using HTML Attribute

An event can be handled directly in HTML.

<button onclick="alert('Button Clicked')">Click Me</button>

Event Using JavaScript

Events can also be handled using JavaScript functions.

let btn = document.getElementById("myBtn"); btn.onclick = function() { alert("Button clicked"); };

addEventListener()

The recommended way to handle events.

let button = document.getElementById("btn"); button.addEventListener("click", function() { console.log("Button clicked"); });

Mouse Events

element.addEventListener("mouseover", function() { console.log("Mouse over element"); });

Keyboard Events

document.addEventListener("keydown", function() { console.log("Key pressed"); });

Form Events

form.addEventListener("submit", function() { console.log("Form submitted"); });

Event Object

The event object contains details about the event.

button.addEventListener("click", function(event) { console.log(event.type); });

Why Use Events?

  • Makes web pages interactive
  • Responds to user actions
  • Controls dynamic behavior

Conclusion

JavaScript events allow your website to react to user input. They are essential for creating interactive and dynamic web applications.
