R Basic Syntax

Fundamental rules and structure of the R programming language

Introduction

R syntax is simple and easy to read. It is case-sensitive and mainly used for data analysis, statistics, and visualization.

Comments in R

Comments are used to explain code and are ignored by R during execution.

# This is a single-line comment in R

Printing Output

You can display output using the print() function or by directly typing the variable name.

print("Hello, R") x <- 10 x

Variables in R

Variables store data values. R uses assignment operators to assign values.

  • <- (recommended)
  • =
a <- 5 b = 10

Data Types (Basic)

R supports different types of data.

x <- 10 # numeric y <- "R" # character z <- TRUE # logical

Basic Arithmetic Operations

R supports standard mathematical operations.

a <- 10 b <- 3 a + b a - b a * b a / b a %% b

Functions in R

Functions perform specific tasks. R has many built-in functions.

sqrt(16) sum(1, 2, 3, 4) mean(c(10, 20, 30))

Vectors (Basic)

A vector is a collection of values of the same data type.

numbers <- c(1, 2, 3, 4, 5) names <- c("A", "B", "C")

Using the Help System

R provides a built-in help system to understand functions.

?mean help(sum)

Conclusion

Understanding R basic syntax is essential before moving to data structures, control statements, and advanced data analysis concepts.
