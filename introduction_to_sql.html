Introduction to SQL

Learn the Basics of Structured Query Language

What is SQL?

SQL stands for Structured Query Language. It is used to store, retrieve, update, and manage data in a database. SQL works with relational databases like MySQL, Oracle, PostgreSQL, and SQL Server.

Why SQL is Important?

  • SQL is used in almost every application that stores data
  • It is easy to learn and very powerful
  • Required skill for developers, data analysts, and testers
  • Works with large amounts of data efficiently

Types of SQL Commands

1. DDL (Data Definition Language)

Used to define and modify database structure.

CREATE ALTER DROP TRUNCATE

2. DML (Data Manipulation Language)

Used to manage data inside tables.

INSERT UPDATE DELETE

3. DQL (Data Query Language)

Used to retrieve data from the database.

SELECT

4. DCL (Data Control Language)

Used to control access to data.

GRANT REVOKE

5. TCL (Transaction Control Language)

Used to manage transactions.

COMMIT ROLLBACK SAVEPOINT

Basic SQL Example

Creating a table and inserting data:

CREATE TABLE students ( id INT, name VARCHAR(50), age INT ); INSERT INTO students VALUES (1, 'Rahul', 20);

Where SQL is Used?

  • Web applications
  • Banking systems
  • Student management systems
  • Data analysis and reporting

Conclusion

SQL is a must-know language for anyone working with data. Once you understand the basics, you can easily move to advanced queries, joins, and database optimization.
