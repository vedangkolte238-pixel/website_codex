Practical Projects to Showcase TypeScript Skills

Introduction

This section lists practical TypeScript projects suitable for beginners to intermediate learners. Projects cover DOM manipulation, APIs, classes, and type-safe applications.

1. To-Do List Application

Create a type-safe To-Do list with classes and DOM manipulation.

class Task { constructor(public title: string, public done: boolean = false) {} } let tasks: Task[] = []; function addTask(title: string) { tasks.push(new Task(title)); console.log(tasks); } addTask("Learn TypeScript"); addTask("Build a Project");

2. Simple Calculator

Use TypeScript classes and functions to build a calculator.

class Calculator { add(a: number, b: number): number { return a + b; } subtract(a: number, b: number): number { return a - b; } } let calc = new Calculator(); console.log(calc.add(5, 3)); // 8 console.log(calc.subtract(10, 4)); // 6

3. Weather App (API)

Fetch weather data from a public API and display it.

interface Weather { temp: number; description: string; } async function getWeather(city: string): Promise { const response = await fetch(`https://api.example.com/weather/${city}`); const data: Weather = await response.json(); console.log(`Temperature: ${data.temp}, Description: ${data.description}`); }

4. Quiz App

Create a type-safe quiz application using interfaces and classes.

interface Question { text: string; answer: string; } class Quiz { questions: Question[] = []; addQuestion(q: Question) { this.questions.push(q); } } let quiz = new Quiz(); quiz.addQuestion({text: "What is TypeScript?", answer: "A superset of JavaScript"}); console.log(quiz.questions);

5. Shopping Cart

Use classes, interfaces, and generics to build a shopping cart system.

interface Product { id: number; name: string; price: number; } class Cart { items: T[] = []; addItem(item: T) { this.items.push(item); } total(): number { return this.items.reduce((sum, item) => sum + item.price, 0); } } let cart = new Cart(); cart.addItem({id:1, name:"Laptop", price:500}); cart.addItem({id:2, name:"Mouse", price:50}); console.log(cart.total()); // 550

Conclusion

These projects help reinforce TypeScript concepts such as types, interfaces, classes, and async programming. You can showcase these projects with screenshots, code snippets, and live demos to demonstrate your TypeScript skills.
