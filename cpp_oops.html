Object-Oriented Programming in C++

What is OOP?

Object-Oriented Programming (OOP) is a programming approach that organizes code using objects and classes. It helps make programs more modular, reusable, and easy to maintain.

Main OOP Concepts in C++

  • Class
  • Object
  • Encapsulation
  • Inheritance
  • Polymorphism
  • Abstraction

Class

A class is a blueprint for creating objects. It defines properties (variables) and behaviors (functions).

class Student { public: int id; string name; void display() { cout << id << " " << name; } };

Object

An object is an instance of a class.

Student s1; s1.id = 1; s1.name = "Alex"; s1.display();

Encapsulation

Encapsulation means binding data and functions together and restricting direct access to data.

class Account { private: int balance; public: void setBalance(int b) { balance = b; } int getBalance() { return balance; } };

Inheritance

Inheritance allows a class to acquire properties of another class.

class Animal { public: void eat() { cout << "Eating"; } }; class Dog : public Animal { public: void bark() { cout << "Barking"; } };

Polymorphism

Polymorphism allows the same function name to behave differently based on the object.

class Shape { public: virtual void draw() { cout << "Drawing shape"; } }; class Circle : public Shape { public: void draw() { cout << "Drawing circle"; } };

Abstraction

Abstraction hides implementation details and shows only essential features.

class Vehicle { public: virtual void start() = 0; };

Advantages of OOP

  • Code reusability
  • Better security
  • Easy maintenance
  • Scalable applications

Conclusion

OOP concepts form the backbone of C++ programming. Understanding these principles helps you build efficient, structured, and professional software.
