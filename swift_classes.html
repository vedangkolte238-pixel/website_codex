Swift Classes

Blueprints for Creating Objects in Swift

What Is a Class?

A class is a blueprint used to create objects. It defines properties and methods that describe the behavior of an object.

Defining a Class

Classes in Swift are defined using the class keyword.

class Person { var name: String var age: Int init(name: String, age: Int) { self.name = name self.age = age } }

Creating an Object

An object is created using the class name followed by parentheses.

let person1 = Person(name: "John", age: 25)

Accessing Properties

print(person1.name) print(person1.age)

Class Methods

Methods define actions that a class can perform.

class Car { var brand: String init(brand: String) { self.brand = brand } func start() { print("\(brand) car is starting") } } let car1 = Car(brand: "Tesla") car1.start()

Reference Type

Classes are reference types. Multiple variables can refer to the same object.

let car2 = car1 car2.brand = "BMW" print(car1.brand) // BMW

Inheritance

A class can inherit properties and methods from another class.

class Animal { func sound() { print("Animal makes a sound") } } class Dog: Animal { override func sound() { print("Dog barks") } } let dog = Dog() dog.sound()

Deinitialization

The deinit method is called when an object is removed from memory.

class FileHandler { deinit { print("Object destroyed") } }

Conclusion

Classes are a core part of Swift's object-oriented programming. They support inheritance, reference behavior, and complex data modeling.
