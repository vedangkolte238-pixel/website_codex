R Data Visualization

Creating charts, graphs, and plots using R

Introduction

Data visualization in R helps in understanding data patterns and trends. R provides built-in plotting functions and powerful packages like ggplot2 for advanced visualization.

Basic Plot

Use the plot() function for simple graphs.

x <- 1:10 y <- c(2,5,7,3,8,6,9,4,10,7) plot(x, y, type = "b", col = "blue", xlab = "X-Axis", ylab = "Y-Axis", main = "Basic Plot")

Bar Plot

counts <- c(10, 20, 15, 30) names <- c("A", "B", "C", "D") barplot(counts, names.arg = names, col = "purple", main = "Bar Plot Example")

Histogram

data <- c(5, 10, 15, 10, 20, 15, 25, 10) hist(data, col = "orange", main = "Histogram Example", xlab = "Values", ylab = "Frequency")

Boxplot

scores <- c(85, 90, 78, 92, 88) boxplot(scores, col = "green", main = "Boxplot Example", ylab = "Scores")

Pie Chart

values <- c(30, 20, 25, 25) labels <- c("A", "B", "C", "D") pie(values, labels = labels, col = rainbow(4), main = "Pie Chart Example")

Advanced Plots with ggplot2

The ggplot2 package provides flexible and powerful plotting.

library(ggplot2) data <- data.frame( category = c("A", "B", "C"), values = c(10, 20, 15) ) ggplot(data, aes(x = category, y = values, fill = category)) + geom_bar(stat = "identity") + ggtitle("Bar Plot with ggplot2")

Conclusion

Data visualization is crucial for interpreting data and presenting results. R provides both simple and advanced tools to create meaningful visualizations for analysis.
