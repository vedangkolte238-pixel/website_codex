Java Data Types

Understanding how data is stored in Java

What are Data Types?

Data types specify the type of data a variable can store. Java is a strongly typed language, which means every variable must have a declared data type.

Types of Data Types in Java

  • Primitive Data Types
  • Non-Primitive (Reference) Data Types

Primitive Data Types

Primitive data types store simple values directly in memory. Java has 8 primitive data types.

Data Type Size Example
byte 1 byte byte a = 10;
short 2 bytes short b = 200;
int 4 bytes int c = 5000;
long 8 bytes long d = 100000L;
float 4 bytes float e = 10.5f;
double 8 bytes double f = 99.99;
char 2 bytes char g = 'A';
boolean 1 bit boolean h = true;

Primitive Example

int age = 20; double price = 499.99; char grade = 'A'; boolean isPassed = true;

Non-Primitive Data Types

Non-primitive data types store references to objects. They are created using classes.

  • String
  • Arrays
  • Classes
  • Interfaces

String Example

String name = "Java Programming";

Array Example

int[] numbers = {10, 20, 30, 40};

Class Example

class Student { int id; String name; }

Difference: Primitive vs Non-Primitive

  • Primitive stores value directly
  • Non-primitive stores memory address
  • Primitive has fixed size
  • Non-primitive size varies

Conclusion

Understanding Java data types is essential for writing efficient and error-free programs. They form the foundation of Java programming.
