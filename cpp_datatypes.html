C++ Data Types

Understanding Different Types of Data in C++

What are Data Types?

Data types specify the type of data a variable can store. They help the compiler understand how much memory to allocate and what kind of operations can be performed on the data.

Types of Data Types in C++

  • Primitive (Basic) Data Types
  • Derived Data Types
  • User-defined Data Types

1. Primitive Data Types

These are the basic data types provided by C++.

int

Used to store whole numbers.

int age = 20;

float

Used to store decimal numbers (single precision).

float price = 99.5;

double

Used to store decimal numbers with higher precision.

double salary = 45678.75;

char

Used to store a single character.

char grade = 'A';

bool

Used to store true or false values.

bool isPassed = true;

2. Derived Data Types

Derived data types are created from basic data types.

Array

int numbers[5] = {1, 2, 3, 4, 5};

Pointer

int x = 10; int* ptr = &x;

Reference

int a = 5; int& ref = a;

3. User-defined Data Types

These data types allow users to define their own types.

struct

struct Student { int id; char grade; };

enum

enum Day {Monday, Tuesday, Wednesday};

typedef

typedef int marks; marks score = 85;

Size of Data Types (Typical)

  • int → 4 bytes
  • float → 4 bytes
  • double → 8 bytes
  • char → 1 byte
  • bool → 1 byte

Conclusion

Data types are fundamental in C++ programming. Choosing the correct data type improves memory usage and makes programs more efficient and readable.
