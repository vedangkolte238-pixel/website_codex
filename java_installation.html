Java Installation

Step by Step Guide to Install Java

Requirements

  • A computer (Windows, macOS, or Linux)
  • Minimum 4 GB RAM recommended
  • Basic knowledge of using a computer

What is JDK?

JDK stands for Java Development Kit. It is required to develop and run Java programs. JDK includes the compiler, runtime environment, and tools.

Installation Steps

Step 1: Download JDK

Download the latest Java Development Kit (JDK) for your operating system. Choose the correct version based on your system (32-bit or 64-bit).

Step 2: Install JDK

Run the installer and follow the on-screen instructions. By default, Java is installed in the Program Files folder.

Step 3: Set Environment Variables

Set the JAVA_HOME variable to the JDK installation path and add the bin folder to the system PATH.

JAVA_HOME = C:\Program Files\Java\jdk PATH = %JAVA_HOME%\bin

Verify Java Installation

Open the command prompt or terminal and type the following commands:

java -version javac -version

If Java is installed correctly, the installed version will be displayed.

First Java Program After Installation

Create a file named Hello.java and write the following code:

class Hello { public static void main(String[] args) { System.out.println("Java Installed Successfully!"); } }

Compile and Run

javac Hello.java java Hello

Common Installation Errors

  • Java command not recognized
  • Incorrect PATH configuration
  • Wrong JDK version installed

Conclusion

After installing Java correctly, you are ready to start learning Java programming and building real-world applications.
