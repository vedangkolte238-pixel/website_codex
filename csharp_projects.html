C# Projects

Example Projects to Practice C# Concepts

1. Calculator

A simple console-based calculator that performs basic arithmetic operations.

using System; class Program { static void Main() { Console.WriteLine("Enter first number:"); double num1 = Convert.ToDouble(Console.ReadLine()); Console.WriteLine("Enter second number:"); double num2 = Convert.ToDouble(Console.ReadLine()); Console.WriteLine("Choose operation (+, -, *, /):"); string op = Console.ReadLine(); double result = 0; switch(op) { case "+": result = num1 + num2; break; case "-": result = num1 - num2; break; case "*": result = num1 * num2; break; case "/": result = num2 != 0 ? num1 / num2 : 0; break; default: Console.WriteLine("Invalid operation"); break; } Console.WriteLine("Result: " + result); } }

2. Student Management System

Manage student details using classes, objects, and lists.

using System; using System.Collections.Generic; class Student { public string Name { get; set; } public int Age { get; set; } } class Program { static void Main() { List<Student> students = new List<Student>(); students.Add(new Student { Name = "Alice", Age = 20 }); students.Add(new Student { Name = "Bob", Age = 22 }); foreach (var s in students) { Console.WriteLine("Name: " + s.Name + ", Age: " + s.Age); } } }

3. Bank Account System

Simulate basic banking operations using OOP concepts.

using System; class BankAccount { public string Owner { get; set; } public double Balance { get; private set; } public void Deposit(double amount) { Balance += amount; Console.WriteLine("Deposited: " + amount); } public void Withdraw(double amount) { if (amount <= Balance) { Balance -= amount; Console.WriteLine("Withdrawn: " + amount); } else { Console.WriteLine("Insufficient balance"); } } } class Program { static void Main() { BankAccount account = new BankAccount(); account.Owner = "John"; account.Deposit(500); account.Withdraw(200); Console.WriteLine("Current Balance: " + account.Balance); } }

4. To-Do List Application

Manage tasks using a list and simple console interface.

using System; using System.Collections.Generic; class Program { static void Main() { List<string> tasks = new List<string>(); tasks.Add("Learn C#"); tasks.Add("Build project"); Console.WriteLine("Tasks:"); foreach(var task in tasks) { Console.WriteLine("- " + task); } } }

5. LINQ Example

Filter data from a list using LINQ.

using System; using System.Collections.Generic; using System.Linq; class Program { static void Main() { List<int> numbers = new List<int> { 1, 2, 3, 4, 5, 6 }; var evenNumbers = numbers.Where(n => n % 2 == 0); Console.WriteLine("Even Numbers:"); foreach(var n in evenNumbers) { Console.WriteLine(n); } } }

Conclusion

These projects help you practice and apply your knowledge of C# concepts, including OOP, collections, exception handling, and LINQ. Start small, and gradually build more complex projects as you gain confidence.
