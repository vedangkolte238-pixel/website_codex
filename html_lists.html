HTML Lists

Organize content using lists in HTML

Introduction

HTML lists are used to display information in an organized way. There are three main types of lists in HTML.

1. Unordered List

An unordered list displays items with bullet points.

<ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul>

2. Ordered List

An ordered list displays items in a numbered format.

<ol> <li>Install Editor</li> <li>Write Code</li> <li>Run Program</li> </ol>

3. Description List

A description list is used to define terms and descriptions.

<dl> <dt>HTML</dt> <dd>Markup language for web pages</dd> <dt>CSS</dt> <dd>Used for styling web pages</dd> </dl>

Nested Lists

Lists can be placed inside another list.

<ul> <li>Frontend <ul> <li>HTML</li> <li>CSS</li> </ul> </li> <li>Backend</li> </ul>

Important List Tags

  • <ul> – unordered list
  • <ol> – ordered list
  • <li> – list item
  • <dl> – description list
  • <dt> – term
  • <dd> – description

Conclusion

HTML lists help structure content clearly and neatly. They are commonly used for menus, steps, and grouped information.
