HTML Attributes

Learn how to add additional information to HTML elements

Introduction

HTML attributes provide additional information about an element. They are always written inside the opening tag and usually come in a name="value" format.

Common HTML Attributes

  • id – Unique identifier for an element. Example: <div id="header"></div>
  • class – Assigns one or more class names for styling. Example: <p class="text-primary">Hello</p>
  • href – Specifies the URL of a link. Example: <a href="https://example.com">Link</a>
  • src – Specifies the path of an image, video, or audio. Example: <img src="image.jpg" alt="My Image">
  • alt – Provides alternative text for images. Example: <img src="image.jpg" alt="Description">
  • title – Adds a tooltip when hovering. Example: <button title="Click me">Submit</button>
  • target – Specifies where to open a link. Example: <a href="#" target="_blank">New Tab</a>
  • style – Adds inline CSS styling. Example: <p style="color:red;">Red Text</p>
