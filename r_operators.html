R Operators

Operators used to perform operations in R programming

Introduction

Operators are symbols that perform operations on variables and values. R provides different types of operators for mathematical and logical tasks.

1. Arithmetic Operators

Used to perform mathematical calculations.

a <- 10 b <- 3 a + b # Addition a - b # Subtraction a * b # Multiplication a / b # Division a %% b # Modulus a ^ b # Exponent

2. Relational Operators

Used to compare two values and return TRUE or FALSE.

a <- 10 b <- 5 a == b a != b a > b a < b a >= b a <= b

3. Logical Operators

Used to combine logical expressions.

x <- TRUE y <- FALSE x & y x | y !x

4. Assignment Operators

Used to assign values to variables.

a <- 5 b = 10 c <<- 15

5. Miscellaneous Operators

Used for special purposes in R.

x <- 1:5 y <- c(2, 4, 6, 8, 10) x %in% y x %/% 2

Operator Precedence

Operator precedence defines the order in which operations are performed.

result <- 5 + 2 * 3 result

Summary

  • Arithmetic operators for calculations
  • Relational operators for comparisons
  • Logical operators for conditions
  • Assignment operators for value assignment
  • Miscellaneous operators for special operations

Conclusion

Understanding R operators is essential for writing logical and efficient programs. They are heavily used in data analysis and conditional operations.
