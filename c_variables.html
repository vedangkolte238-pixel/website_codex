C Variables

Storing and Managing Data in C

What is a Variable?

A variable is a named memory location used to store data. The value of a variable can change during program execution.

Declaring Variables

In C, a variable must be declared before it is used.

data_type variable_name; int age; float salary; char grade;

Initializing Variables

Initialization means assigning a value to a variable at the time of declaration.

int count = 10; float price = 99.50; char ch = 'A';

Rules for Naming Variables

  • Must start with a letter or underscore
  • Cannot start with a number
  • No spaces allowed
  • Cannot use C keywords
  • Case-sensitive

Types of Variables in C

1. Local Variables

Declared inside a function and accessible only within that function.

void display() { int x = 5; }

2. Global Variables

Declared outside all functions and accessible throughout the program.

int total; void main() { total = 100; }

3. Static Variables

Retains its value between function calls.

void counter() { static int count = 0; count++; }

4. Register Variables

Stored in CPU registers for faster access.

register int speed;

Scope of Variables

Scope defines where a variable can be accessed in a program.

  • Local scope
  • Global scope
  • Block scope

Example Program

#include int main() { int a = 10; int b = 20; int sum = a + b; printf("Sum = %d", sum); return 0; }

Conclusion

Variables are essential in C programming for storing and manipulating data. Understanding variable types and scope helps in writing efficient programs.
