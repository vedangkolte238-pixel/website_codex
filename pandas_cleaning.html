Pandas Data Cleaning

Handling Missing Data and Preparing Datasets

Handling Missing Values

Pandas provides tools to detect and handle missing data (NaN).

Detecting Missing Values

import pandas as pd import numpy as np data = { 'Name': ['Alice', 'Bob', 'Charlie', None], 'Age': [25, np.nan, 35, 40], 'City': ['New York', 'Los Angeles', None, 'Chicago'] } df = pd.DataFrame(data) # Check for missing values print(df.isnull()) print(df.isnull().sum())

Dropping Missing Values

# Drop rows with any missing values df_drop = df.dropna() print(df_drop) # Drop columns with missing values df_drop_col = df.dropna(axis=1) print(df_drop_col)

Filling Missing Values

# Fill missing values with a constant df_fill = df.fillna({'Name':'Unknown', 'Age':0, 'City':'Unknown'}) print(df_fill) # Fill missing numerical values with mean df['Age'].fillna(df['Age'].mean(), inplace=True) print(df)

Removing Duplicates

data = {'Name': ['Alice', 'Bob', 'Alice', 'Charlie'], 'Age': [25, 30, 25, 35]} df = pd.DataFrame(data) # Drop duplicate rows df_unique = df.drop_duplicates() print(df_unique)

Renaming Columns

df.rename(columns={'Name':'FullName', 'Age':'AgeYears'}, inplace=True) print(df)

Changing Data Types

df['AgeYears'] = df['AgeYears'].astype(float) print(df.dtypes)

Applications

  • Preparing clean datasets for analysis and machine learning
  • Handling missing or inconsistent data efficiently
  • Ensuring data consistency and quality
  • Removing duplicates and renaming columns for readability

Conclusion

Data cleaning is a crucial step in data analysis. Pandas provides powerful functions to detect, fill, and remove missing or duplicate data, making datasets ready for analysis and modeling.
