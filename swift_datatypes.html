Swift Data Types

Understanding Built-in Data Types in Swift

What Are Data Types?

Data types specify the type of data a variable can hold. Swift is a strongly typed language, meaning every variable has a specific data type.

Integer Types

Integers store whole numbers.

let age: Int = 20 let score = 100

Floating-Point Types

Used for decimal values.

  • Float – 32-bit decimal
  • Double – 64-bit decimal (preferred)
let pi: Double = 3.14 let temperature: Float = 36.6

Boolean Type

Stores true or false values.

let isLoggedIn: Bool = true

String Type

Strings store text data.

let name: String = "Swift" let message = "Hello, World!"

Character Type

Stores a single character.

let letter: Character = "A"

Array Type

Arrays store ordered collections of values.

let numbers: [Int] = [1, 2, 3, 4] let fruits = ["Apple", "Banana", "Mango"]

Dictionary Type

Dictionaries store key-value pairs.

let student: [String: Int] = ["Math": 90, "Science": 85]

Tuple Type

Tuples group multiple values into a single variable.

let person = (name: "John", age: 25)

Optional Type

Optionals allow a variable to hold a value or nil.

var email: String? = nil email = "user@example.com"

Type Inference

Swift automatically detects data types.

let city = "New York" let year = 2025

Conclusion

Swift provides powerful and safe data types that help prevent errors and improve code reliability.
