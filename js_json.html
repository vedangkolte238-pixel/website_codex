JavaScript JSON

Store and Exchange Data Easily

What is JSON?

JSON stands for JavaScript Object Notation. It is a lightweight format used to store and transfer data. JSON is easy for humans to read and easy for machines to parse.

JSON Structure

JSON data is written as key and value pairs. Keys must be in double quotes.

{ "name": "John", "age": 25, "isStudent": true }

JSON vs JavaScript Object

They look similar, but JSON is stricter.

// JavaScript Object let user = { name: "John", age: 25 }; // JSON { "name": "John", "age": 25 }

JSON.stringify()

Converts a JavaScript object into a JSON string.

let user = { name: "Alice", age: 30 }; let jsonData = JSON.stringify(user); console.log(jsonData);

JSON.parse()

Converts a JSON string back into a JavaScript object.

let jsonText = '{"name":"Alice","age":30}'; let obj = JSON.parse(jsonText); console.log(obj.name);

JSON Arrays

JSON can store multiple objects using arrays.

[ { "id": 1, "title": "Book" }, { "id": 2, "title": "Pen" } ]

Where JSON is Used

  • Sending data between server and browser
  • APIs and web services
  • Configuration files
  • Storing structured data

Conclusion

JSON is a core part of modern JavaScript development. Understanding JSON is essential for working with APIs and data-driven applications.
