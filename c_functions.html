C Functions

Modular Programming in C

What is a Function?

A function is a block of code that performs a specific task. Functions help break a program into smaller, manageable parts.

Why Use Functions?

  • Improves code reusability
  • Makes programs easier to understand
  • Reduces code repetition
  • Helps in debugging and testing

Function Structure

A function has three main parts: declaration, definition, and call.

return_type function_name(parameters) { // function body }

Example of a Function

#include <stdio.h> void greet() { printf("Hello, Welcome to C!"); } int main() { greet(); return 0; }

Types of Functions

1. Library Functions

Predefined functions provided by C libraries.

printf("Hello"); scanf("%d", &num);

2. User-Defined Functions

Functions created by the programmer.

int add(int a, int b) { return a + b; }

Function Declaration

Tells the compiler about the function name, return type, and parameters.

int add(int, int);

Function Definition

Contains the actual code of the function.

int add(int a, int b) { return a + b; }

Function Call

Used to execute the function.

int result = add(5, 3);

Functions with Arguments and Return Value

#include <stdio.h> int square(int x) { return x * x; } int main() { int result = square(4); printf("%d", result); return 0; }

Advantages of Functions

  • Code modularity
  • Easy maintenance
  • Better readability
  • Time saving

Conclusion

Functions are a core concept in C programming. They make programs structured, reusable, and easier to manage.
