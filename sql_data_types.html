SQL Data Types

Understand how data is stored in databases

What are SQL Data Types?

SQL data types define the type of data that can be stored in a table column. Choosing the correct data type helps in saving storage space and improving performance.

1. Numeric Data Types

Used to store numbers.

  • INT – Stores whole numbers
  • SMALLINT – Stores smaller range integers
  • BIGINT – Stores very large integers
  • DECIMAL(p,s) – Stores exact decimal values
  • FLOAT – Stores approximate decimal values

2. Character / String Data Types

Used to store text or characters.

  • CHAR(n) – Fixed-length string
  • VARCHAR(n) – Variable-length string
  • TEXT – Large text data

3. Date and Time Data Types

Used to store date and time values.

  • DATE – Stores date (YYYY-MM-DD)
  • TIME – Stores time (HH:MM:SS)
  • DATETIME – Stores date and time
  • TIMESTAMP – Stores timestamp value

4. Boolean Data Types

Used to store true or false values.

  • BOOLEAN – Stores TRUE or FALSE

5. Binary Data Types

Used to store binary data such as images or files.

  • BINARY
  • VARBINARY
  • BLOB

Example: Using Data Types

CREATE TABLE employees ( emp_id INT, emp_name VARCHAR(50), salary DECIMAL(10,2), join_date DATE, is_active BOOLEAN );

Conclusion

SQL data types are the foundation of database design. Understanding them helps you create efficient and reliable database tables.
