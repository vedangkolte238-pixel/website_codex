Python Libraries

Popular Libraries with Examples and Output

What are Python Libraries?

Libraries are collections of pre-written code that help perform specific tasks without rewriting code from scratch. Python has a rich ecosystem of libraries for data science, web development, automation, and more.

1. NumPy

Used for numerical computations and arrays.

import numpy as np arr = np.array([1, 2, 3, 4]) print(arr) print("Sum:", np.sum(arr))
Output:
[1 2 3 4]
Sum: 10

2. Pandas

Used for data manipulation and analysis.

import pandas as pd data = {"Name": ["Alice", "Bob"], "Age": [25, 30]} df = pd.DataFrame(data) print(df)
Output:
Name Age
0 Alice 25
1 Bob 30

3. Matplotlib

Used for data visualization.

import matplotlib.pyplot as plt x = [1, 2, 3] y = [2, 4, 6] plt.plot(x, y) plt.title("Simple Plot") plt.xlabel("X-axis") plt.ylabel("Y-axis") plt.show()

4. Seaborn

Used for statistical data visualization.

import seaborn as sns import pandas as pd data = pd.DataFrame({"x":[1,2,3,4], "y":[4,3,2,1]}) sns.lineplot(data=data, x="x", y="y")

5. Requests

Used for sending HTTP requests.

import requests response = requests.get("https://api.github.com") print(response.status_code) print(response.headers["Content-Type"])
Output:
200
application/json; charset=utf-8

6. OS

Used for interacting with the operating system.

import os print("Current Directory:", os.getcwd()) print("List of files:", os.listdir())
Output:
Current Directory: C:\Your\Current\Directory
List of files: ['example.txt', 'script.py', ...]

Conclusion

Python libraries save time, simplify coding, and allow you to perform complex tasks efficiently. Learning popular libraries is essential for Python development and data science.
