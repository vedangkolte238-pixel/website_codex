Understanding Queue Data Structure in DSA

What Is a Queue?

A queue is a linear data structure that follows the FIFO (First In, First Out) principle. The first element added is the first one to be removed. Think of it like a line at a ticket counter.

Queue Operations

  • Enqueue: Add an element to the rear of the queue
  • Dequeue: Remove an element from the front of the queue
  • Front / Peek: Get the front element without removing it
  • isEmpty: Check if the queue is empty

Types of Queues

  • Simple Queue: Linear FIFO structure
  • Circular Queue: Rear connects to the front to utilize space efficiently
  • Priority Queue: Elements are dequeued based on priority, not just order
  • Deque (Double-Ended Queue): Insertions and deletions can be done at both ends

Queue Implementation

// Using array (C++) #include <iostream> using namespace std; #define SIZE 100 class Queue { int arr[SIZE]; int front, rear; public: Queue() { front = -1; rear = -1; } bool isEmpty() { return front == -1; } bool enqueue(int x) { if(rear == SIZE-1) return false; if(front == -1) front = 0; arr[++rear] = x; return true; } int dequeue() { if(isEmpty()) return -1; int x = arr[front]; if(front == rear) front = rear = -1; else front++; return x; } int peek() { if(isEmpty()) return -1; return arr[front]; } };

Queue Using Linked List

// C++ struct Node { int data; Node* next; }; class Queue { Node *front, *rear; public: Queue() { front = rear = NULL; } void enqueue(int x) { Node* newNode = new Node(); newNode->data = x; newNode->next = NULL; if(rear == NULL) front = rear = newNode; else { rear->next = newNode; rear = newNode; } } int dequeue() { if(front == NULL) return -1; int x = front->data; Node* temp = front; front = front->next; if(front == NULL) rear = NULL; delete temp; return x; } int peek() { if(front == NULL) return -1; return front->data; } };

Applications of Queue

  • CPU scheduling
  • Handling requests in web servers
  • Printing tasks in printers
  • Breadth-First Search (BFS) in graphs
  • Order processing systems

Advantages of Queue

  • Maintains the order of elements
  • Efficient for sequential processing

Conclusion

Queues are essential for managing data in order and are widely used in operating systems, networking, and algorithms. Mastering queues and their variations is key for solving many DSA problems.
