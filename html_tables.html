HTML Tables

Display data in rows and columns

Introduction

HTML tables are used to display data in a structured format using rows and columns.

Basic Table Structure

<table> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>Amit</td> <td>21</td> </tr> </table>

Table Example

Name Course Year
Rahul BCA 2
Priya MCA 1

Table Tags Explanation

  • <table> – creates a table
  • <tr> – table row
  • <th> – table header
  • <td> – table data

Adding Border Using HTML

<table border="1"> ... </table>

Important Points

  • Tables are best for structured data
  • Use headings to describe columns
  • Avoid tables for page layout

Conclusion

HTML tables help present data clearly. They are commonly used in reports, dashboards, and result tables.
