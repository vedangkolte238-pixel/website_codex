R Regression

Performing linear and multiple regression analysis in R

Introduction

Regression analysis in R is used to understand relationships between dependent and independent variables. R provides built-in functions for simple linear regression, multiple regression, and model evaluation.

Simple Linear Regression

# Sample data x <- c(1, 2, 3, 4, 5) y <- c(2, 4, 5, 4, 5) # Fit linear model model <- lm(y ~ x) summary(model) # Predict new values predict(model, data.frame(x = c(6,7)))

Multiple Linear Regression

# Sample data data <- data.frame( y = c(5, 7, 9, 11, 13), x1 = c(1, 2, 3, 4, 5), x2 = c(2, 1, 4, 3, 5) ) # Fit multiple regression model <- lm(y ~ x1 + x2, data = data) summary(model)

Plotting Regression Line

plot(x, y, main="Simple Linear Regression") abline(model, col="red")

Residual Analysis

# Residuals res <- residuals(model) res # Plot residuals plot(res, main="Residuals Plot") abline(h=0, col="blue")

Checking Model Assumptions

Use diagnostic plots to check linearity, homoscedasticity, and normality of residuals.

par(mfrow=c(2,2)) plot(model)

Conclusion

Regression analysis in R helps quantify relationships between variables and make predictions. Understanding regression is essential for statistical modeling and data analysis.
