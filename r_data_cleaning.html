R Data Cleaning

Techniques to prepare data for analysis in R

Introduction

Data cleaning is the process of detecting and correcting errors or inconsistencies in datasets. It ensures accurate analysis and reliable results in R.

Handling Missing Values

Use functions like is.na() and na.omit() to handle missing data.

data <- data.frame( id = 1:5, score = c(90, NA, 85, NA, 95) ) # Check missing values is.na(data$score) # Remove rows with missing values clean_data <- na.omit(data) clean_data

Replacing Missing Values

You can replace missing values with mean, median, or custom values.

data$score[is.na(data$score)] <- mean(data$score, na.rm = TRUE) data

Removing Duplicates

Use duplicated() and unique() to handle duplicate rows.

data <- data.frame( id = c(1, 2, 2, 3), score = c(90, 85, 85, 95) ) unique(data)

Trimming Whitespace

Use trimws() to remove extra spaces from character columns.

text <- c(" R Programming ", " Data Science ") trimws(text)

Converting Data Types

Ensure each column has the correct type using as.numeric(), as.character(), etc.

data$score <- as.numeric(data$score) data$id <- as.character(data$id) str(data)

Renaming Columns

Use colnames() to rename column names.

colnames(data) <- c("ID", "Score") data

Conclusion

Data cleaning is a crucial step before analysis. Using R functions efficiently ensures datasets are accurate, consistent, and ready for further processing.
