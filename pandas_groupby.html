Pandas Group By

Grouping and Aggregating Data in Pandas

What is Group By?

The groupby() function in Pandas is used to split data into groups based on some criteria, perform computations on each group, and combine the results. It’s very useful for aggregation and analysis of grouped data.

Basic Group By Example

import pandas as pd data = { 'Department': ['HR', 'IT', 'HR', 'IT', 'Sales'], 'Employee': ['Alice', 'Bob', 'Charlie', 'David', 'Eve'], 'Salary': [50000, 60000, 55000, 65000, 70000] } df = pd.DataFrame(data) # Group by Department grouped = df.groupby('Department') # Sum of salaries in each department print(grouped['Salary'].sum())

Common Aggregation Functions

print(grouped['Salary'].mean()) # Average salary per department print(grouped['Salary'].max()) # Maximum salary per department print(grouped['Salary'].min()) # Minimum salary per department print(grouped['Salary'].count()) # Number of employees per department

Multiple Aggregations

agg_result = grouped['Salary'].agg(['sum', 'mean', 'max', 'min']) print(agg_result)

Grouping by Multiple Columns

data = { 'Department': ['HR', 'IT', 'HR', 'IT', 'Sales'], 'Gender': ['F', 'M', 'M', 'M', 'F'], 'Salary': [50000, 60000, 55000, 65000, 70000] } df = pd.DataFrame(data) grouped_multi = df.groupby(['Department', 'Gender']) print(grouped_multi['Salary'].mean())

Applications

  • Summarizing sales, salaries, or other metrics by category
  • Performing statistical analysis on grouped data
  • Generating reports by department, region, or any categorical variable
  • Data aggregation for visualization

Conclusion

The groupby() function in Pandas is a powerful tool for analyzing and summarizing data. It allows splitting data into logical groups, applying aggregation or transformation functions, and combining the results efficiently.
