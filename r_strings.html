R Strings

Working with text data in R programming

Introduction

Strings in R are sequences of characters. They are commonly used to store names, messages, and textual data.

Creating Strings

Strings are created using single or double quotes.

name <- "R Programming" message <- 'Welcome to R' name message

String Length

Use nchar() to find the length of a string.

text <- "Data Science" nchar(text)

Combining Strings

Use paste() or paste0() to combine strings.

first <- "Hello" second <- "World" paste(first, second) paste0(first, second)

Changing Case

R provides functions to change string case.

text <- "R Language" toupper(text) tolower(text)

Substring

Extract part of a string using substr().

text <- "Programming" substr(text, 1, 4)

Replacing Text

Use gsub() to replace characters or words.

sentence <- "I love Java" gsub("Java", "R", sentence)

Splitting Strings

Use strsplit() to split a string.

text <- "R is powerful" strsplit(text, " ")

Common String Functions

  • nchar() – length of string
  • paste() – combine strings
  • substr() – extract substring
  • gsub() – replace text
  • strsplit() – split strings

Conclusion

String handling is an important part of data cleaning and analysis in R. Mastering string functions helps work with real-world datasets effectively.
