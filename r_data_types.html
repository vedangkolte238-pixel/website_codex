R Data Types

Understanding different types of data in R programming

Introduction

Data types define the kind of value a variable can store. R supports several basic and advanced data types used in data analysis.

1. Numeric

Numeric data types store numbers, including decimals.

x <- 10 y <- 3.14 typeof(x)

2. Integer

Integers are whole numbers. In R, integers are created using L.

a <- 5L b <- 100L typeof(a)

3. Character

Character data types store text or strings.

name <- "R Programming" city <- 'Pune' typeof(name)

4. Logical

Logical data types store boolean values: TRUE or FALSE.

isValid <- TRUE isAdult <- FALSE typeof(isValid)

5. Complex

Complex data types store complex numbers.

z <- 2 + 3i typeof(z)

6. Raw

Raw data type is used to store raw bytes.

r <- charToRaw("R") r

Checking Data Type

You can check the data type of a variable using built-in functions.

class(x) typeof(y) is.numeric(x) is.character(name)

Type Conversion

R allows conversion from one data type to another.

num <- 10 as.character(num) as.numeric("25") as.integer(7.8)

Summary

  • Numeric – decimal values
  • Integer – whole numbers
  • Character – text data
  • Logical – TRUE/FALSE
  • Complex – complex numbers
  • Raw – raw bytes

Conclusion

Understanding R data types helps in writing correct programs and performing accurate data analysis.
