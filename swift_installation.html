Swift Installation

How to Install and Run Swift on Your System

System Requirements

  • macOS, Linux, or Windows (via tools)
  • Minimum 4GB RAM recommended
  • Internet connection

Install Swift on macOS

Swift comes pre-installed on macOS. To get the latest version, install Xcode.

  1. Open the App Store
  2. Search for Xcode
  3. Click Install

Check Swift version:

swift --version

Install Swift on Linux

Swift can be installed using official binaries.

  1. Download Swift from the official website
  2. Extract the archive
  3. Add Swift to PATH
tar -xvzf swift-.tar.gz export PATH=/path/to/swift/usr/bin:$PATH

Install Swift on Windows

Swift is available on Windows using official toolchains.

  • Install Visual Studio
  • Install Swift toolchain for Windows
  • Use command prompt or PowerShell

Run Your First Swift Program

Create a file named hello.swift:

print("Hello, Swift!")

Run the program:

swift hello.swift

Editors for Swift

  • Xcode (Best for macOS)
  • Visual Studio Code
  • IntelliJ IDEA

Conclusion

Swift is easy to install and get started with. Once installed, you can build powerful applications for Apple platforms and beyond.
