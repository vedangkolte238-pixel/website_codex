HTML Input Types

Different types of input fields in HTML forms

Introduction

The <input> element is used to take data from users. HTML provides different input types for different kinds of data.

Text Input

<input type="text" placeholder="Enter name">

Password Input

<input type="password">

Email Input

<input type="email" placeholder="Enter email">

Number Input

<input type="number">

Date Input

<input type="date">

Radio Button

<input type="radio" name="gender"> Male <input type="radio" name="gender"> Female

Checkbox

<input type="checkbox"> HTML <input type="checkbox"> CSS

File Input

<input type="file">

Range Input

<input type="range">

Color Input

<input type="color">

Submit Button

<input type="submit" value="Submit">

Reset Button

<input type="reset" value="Reset">

Common Input Types List

  • text
  • password
  • email
  • number
  • date
  • radio
  • checkbox
  • file
  • range
  • color
  • submit
  • reset

Conclusion

HTML input types help collect correct data from users. Choosing the right input type improves form usability and accuracy.
