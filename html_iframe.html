HTML Iframe

Embed one page inside another page

Introduction

An iframe is used to display another HTML page inside the current web page. It creates a nested browsing context.

Basic Iframe Syntax

<iframe src="page.html"></iframe>

Iframe with Width and Height

<iframe src="content.html" width="400" height="200"></iframe>

Iframe Example

Below is a simple iframe loading a local HTML file.

<iframe src="example.html"></iframe>

Iframe Attributes

  • src – source page
  • width – iframe width
  • height – iframe height
  • title – describes iframe content

Removing Iframe Border

<iframe src="page.html" style="border:none;"></iframe>

Important Points

  • Use iframe only when required
  • Always add a title for accessibility
  • Some pages may block iframe usage

Conclusion

HTML iframes allow embedding content within a page. They are commonly used for layouts, previews, and embedded content.
