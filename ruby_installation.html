Ruby Installation

Setting Up Ruby on Your System

Installing Ruby on Windows

  • Download RubyInstaller from rubyinstaller.org.
  • Run the installer and select "Add Ruby executables to your PATH".
  • Complete the installation and open Command Prompt.
  • Check installation: ruby -v

Installing Ruby on macOS

  • macOS comes with a system Ruby, but you may want a newer version.
  • Install Homebrew if not already installed: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  • Install Ruby: brew install ruby
  • Add Ruby to PATH in ~/.zshrc or ~/.bash_profile:
    • export PATH="/usr/local/opt/ruby/bin:$PATH"
  • Verify installation: ruby -v

Installing Ruby on Linux

  • For Debian/Ubuntu: sudo apt update && sudo apt install ruby-full
  • For Fedora: sudo dnf install ruby
  • For Arch: sudo pacman -S ruby
  • Verify installation: ruby -v

Installing RubyGems

RubyGems is the package manager for Ruby. Most Ruby installations include it by default.

gem -v

Test Ruby Installation

# Create a test file hello.rb puts "Hello, Ruby!" # Run it in terminal ruby hello.rb

Conclusion

Once Ruby is installed, you can start writing Ruby scripts, install gems, and develop applications with ease.
