NumPy Random Module

Generating Random Numbers with NumPy

What Is the Random Module?

NumPy provides the numpy.random module to generate random numbers and perform random sampling. It is widely used in simulations, data analysis, and machine learning.

Generating Random Numbers

Random floats between 0 and 1

import numpy as np arr = np.random.rand(5) # 5 random floats [0,1) print(arr)

Random integers

arr = np.random.randint(0, 10, size=5) # 5 random integers from 0 to 9 print(arr)

Random numbers from a normal distribution

arr = np.random.randn(5) # 5 numbers from standard normal distribution print(arr)

Random numbers from a specific normal distribution

arr = np.random.normal(loc=10, scale=2, size=5) # mean=10, std=2 print(arr)

Random choice from an array

arr = np.array([10, 20, 30, 40]) choice = np.random.choice(arr, size=2) # pick 2 random elements print(choice)

Shuffling arrays

arr = np.array([1, 2, 3, 4, 5]) np.random.shuffle(arr) # shuffle in-place print(arr)

Seeding for reproducibility

np.random.seed(42) # ensures same random numbers every time arr = np.random.rand(3) print(arr)

Applications of NumPy Random

  • Simulations and modeling
  • Random sampling and bootstrapping
  • Machine learning (train/test splitting, initialization)
  • Games and procedural content generation

Conclusion

The NumPy random module is a powerful tool for generating random numbers and sampling data. Mastering it is essential for simulations, statistical analysis, and machine learning applications.
