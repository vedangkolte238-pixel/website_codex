Rust Installation

Setting Up Rust on Your System

Installing Rust Using rustup

The recommended way to install Rust is via rustup, which manages Rust versions and associated tools.

Windows

  • Download and run the rustup installer.
  • Follow the prompts and ensure the option to add Rust to PATH is selected.
  • Open Command Prompt and verify installation: rustc --version

macOS

  • Open Terminal.
  • Run the command: curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • Follow the instructions to add Rust to PATH.
  • Verify installation: rustc --version

Linux

  • Open Terminal.
  • Run: curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • Follow instructions to configure PATH.
  • Verify installation: rustc --version

Updating Rust

To keep Rust up-to-date, use:

rustup update

Uninstalling Rust

If needed, Rust can be uninstalled with:

rustup self uninstall

Testing Rust

fn main() { println!("Hello, Rust!"); }

Save as main.rs and run:

rustc main.rs ./main

Conclusion

Using rustup ensures you always have the latest Rust version and access to tools like Cargo. This setup is essential for developing Rust applications efficiently.
