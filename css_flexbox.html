CSS Flexbox

Flexible Layout System in CSS

What is Flexbox?

CSS Flexbox is a layout model that allows you to design flexible and responsive layouts easily. It arranges elements in rows or columns and adjusts space automatically.

Flex Container

To use Flexbox, an element must be a flex container.

.container { display: flex; }

Flex Direction

Defines the direction of items.

.container { flex-direction: row; }

Justify Content

Aligns items horizontally.

.container { justify-content: space-between; }

Align Items

Aligns items vertically.

.container { align-items: center; }

Flex Wrap

Controls wrapping of flex items.

.container { flex-wrap: wrap; }

Gap

Creates space between flex items.

.container { gap: 20px; }

Flex Items

Properties applied to child elements.

Flex Grow

.item { flex-grow: 1; }

Flex Shrink

.item { flex-shrink: 0; }

Flex Basis

.item { flex-basis: 200px; }

Flex Shorthand

.item { flex: 1 1 200px; }

Centering with Flexbox

Perfect centering example.

.container { display: flex; justify-content: center; align-items: center; }

Best Practices

  • Use Flexbox for one-dimensional layouts
  • Combine with media queries for responsiveness
  • Use gap instead of margins when possible

Conclusion

Flexbox makes layout design simple and responsive. It is essential for modern web development.
