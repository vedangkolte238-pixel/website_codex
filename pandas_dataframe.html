Pandas DataFrame

Two-Dimensional Labeled Data Structure in Pandas

What is a DataFrame?

A Pandas DataFrame is a two-dimensional labeled data structure with columns of potentially different types. It can be thought of as a table or spreadsheet in Python.

Creating a DataFrame

import pandas as pd # From a dictionary data = { 'Name': ['Alice', 'Bob', 'Charlie'], 'Age': [25, 30, 35], 'City': ['New York', 'Los Angeles', 'Chicago'] } df = pd.DataFrame(data) print(df) # From a list of lists data_list = [['Alice', 25], ['Bob', 30], ['Charlie', 35]] df2 = pd.DataFrame(data_list, columns=['Name', 'Age']) print(df2)

Accessing Columns

# Single column as Series print(df['Name']) # Multiple columns as DataFrame print(df[['Name', 'Age']])

Accessing Rows

# Using iloc for index-based selection print(df.iloc[0]) # First row # Using loc for label-based selection print(df.loc[0]) # First row (if default index)

DataFrame Operations

import numpy as np df['AgePlus10'] = df['Age'] + 10 print(df) # Filtering rows print(df[df['Age'] > 28]) # Basic statistics print(df['Age'].mean()) # 30.0 print(df['Age'].max()) # 35

Applications of DataFrames

  • Data cleaning and preprocessing
  • Exploratory Data Analysis (EDA)
  • Joining, merging, and reshaping datasets
  • Time series analysis and financial data
  • Machine learning data pipelines

Conclusion

DataFrames are the core data structure in Pandas for handling tabular data. They provide flexible indexing, selection, filtering, and computational tools, making them essential for data analysis.
