CSS Introduction

Learn the Basics of Cascading Style Sheets

What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and design HTML web pages. With CSS, you can control colors, fonts, spacing, layout, and animations.

Why Use CSS?

  • Makes websites look attractive
  • Separates design from HTML structure
  • Saves time by styling multiple pages at once
  • Helps create responsive designs

How CSS Works

CSS applies styles to HTML elements using rules. Each rule has a selector and a declaration block.

selector { property: value; }

Example of CSS

This example changes text color and font size.

p { color: blue; font-size: 18px; }

Types of CSS

1. Inline CSS

Applied directly inside an HTML element.

This is inline CSS

2. Internal CSS

Written inside the style tag in the head section.

This CSS written for the title - CSS Introduction

3. External CSS

Written in a separate CSS file and linked to HTML.

body { background-color: lightgray; }

CSS Comments

Comments are used to explain the code.

/* This is a CSS comment */

Benefits of CSS

  • Cleaner HTML code
  • Easy maintenance
  • Consistent design
  • Better user experience

Conclusion

CSS is a core technology of web development. Learning CSS helps you turn simple HTML pages into professional-looking websites.
