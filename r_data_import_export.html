R Data Import and Export

Working with external data files in R

Introduction

Importing and exporting data is essential for data analysis in R. You can read and write data in formats like CSV, Excel, and text files.

Importing CSV Files

Use read.csv() to import CSV files.

data <- read.csv("data.csv", header = TRUE, sep = ",") head(data)

Importing Text Files

Use read.table() for general text files.

data <- read.table("data.txt", header = TRUE, sep = "\t") head(data)

Importing Excel Files

Use the readxl package to read Excel files.

library(readxl) data <- read_excel("data.xlsx", sheet = 1) head(data)

Exporting Data to CSV

Use write.csv() to save data to CSV format.

write.csv(data, "output.csv", row.names = FALSE)

Exporting Data to Text Files

Use write.table() to save data as a text file.

write.table(data, "output.txt", sep = "\t", row.names = FALSE)

Exporting Data to Excel

Use the writexl package to export Excel files.

library(writexl) write_xlsx(data, "output.xlsx")

Other Useful Functions

  • file.choose() – Opens a file dialog to select files interactively.
  • readRDS()/saveRDS() – For saving R objects in binary format.
  • read.table() – Flexible function for reading various text files.

Conclusion

R provides multiple functions and packages for importing and exporting data. Understanding these methods is crucial for working with real-world datasets.
