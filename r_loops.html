R Loops

Repeating tasks efficiently using loops in R programming

Introduction

Loops are used to execute a block of code repeatedly until a condition is met. They help reduce code repetition and improve efficiency.

1. for Loop

The for loop is used when the number of iterations is known.

for (i in 1:5) { print(i) }

2. while Loop

The while loop executes code as long as the condition is TRUE.

i <- 1 while (i <= 5) { print(i) i <- i + 1 }

3. repeat Loop

The repeat loop runs indefinitely until a break statement is used.

i <- 1 repeat { print(i) i <- i + 1 if (i > 5) { break } }

4. break Statement

The break statement terminates the loop immediately.

for (i in 1:10) { if (i == 6) { break } print(i) }

5. next Statement

The next statement skips the current iteration and moves to the next one.

for (i in 1:5) { if (i == 3) { next } print(i) }

Nested Loops

A loop inside another loop is called a nested loop.

for (i in 1:3) { for (j in 1:2) { print(paste(i, j)) } }

When to Use Loops

  • Repeat calculations
  • Process vectors and lists
  • Perform iterative tasks
  • Apply conditions repeatedly

Conclusion

Loops are a fundamental part of R programming. Understanding when and how to use loops helps in writing efficient and readable code.
