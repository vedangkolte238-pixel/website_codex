NumPy Broadcasting

Understanding Broadcasting in NumPy

What Is Broadcasting?

Broadcasting allows NumPy to perform operations on arrays of different shapes. Smaller arrays are “broadcast” across larger arrays so that arithmetic operations can be performed without explicit replication.

Rules of Broadcasting

  • If arrays have different dimensions, prepend 1 to the shape of the smaller array.
  • Arrays are compatible in a dimension if they are equal or one of them is 1.
  • If shapes are not compatible, a ValueError is raised.

Example 1: Adding a scalar to an array

import numpy as np arr = np.array([1, 2, 3]) result = arr + 5 # Scalar 5 broadcasted to each element print(result) # [6 7 8]

Example 2: Adding arrays of different shapes

A = np.array([[1, 2, 3], [4, 5, 6]]) B = np.array([10, 20, 30]) C = A + B # B is broadcasted to each row of A print(C) # Output: # [[11 22 33] # [14 25 36]]

Example 3: Broadcasting with column vector

X = np.array([[1,2,3], [4,5,6]]) Y = np.array([[10], [20]]) Z = X + Y # Y is broadcasted across columns print(Z) # Output: # [[11 12 13] # [24 25 26]]

Applications of Broadcasting

  • Vectorized arithmetic operations without loops
  • Efficient matrix operations
  • Feature scaling in machine learning
  • Image processing (adding constants to images)

Advantages of Broadcasting

  • Memory-efficient (no need to replicate arrays)
  • Faster computations due to vectorization
  • Cleaner, simpler code

Conclusion

Broadcasting is a powerful feature in NumPy that allows flexible operations on arrays of different shapes. Mastering broadcasting improves efficiency and simplifies numerical code.
