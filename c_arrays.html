C Arrays

Store Multiple Values Using a Single Variable

What is an Array?

An array is a collection of elements of the same data type stored in contiguous memory locations. Each element is accessed using an index.

Why Use Arrays?

  • Stores multiple values efficiently
  • Reduces code repetition
  • Makes data handling easier
  • Improves program performance

Array Declaration

Syntax to declare an array in C:

data_type array_name[size];

Example of Array Declaration

int numbers[5];

Array Initialization

Assigning values to an array at the time of declaration.

int numbers[5] = {10, 20, 30, 40, 50};

Accessing Array Elements

Array elements are accessed using index numbers. Index starts from 0.

printf("%d", numbers[0]); // Output: 10 printf("%d", numbers[3]); // Output: 40

Example Program Using Arrays

#include <stdio.h> int main() { int marks[3] = {80, 75, 90}; for(int i = 0; i < 3; i++) { printf("%d\n", marks[i]); } return 0; }

Types of Arrays

1. One-Dimensional Array

Stores data in a single row.

int a[5] = {1, 2, 3, 4, 5};

2. Two-Dimensional Array

Stores data in rows and columns.

int matrix[2][2] = { {1, 2}, {3, 4} };

Two-Dimensional Array Example

#include <stdio.h> int main() { int mat[2][2] = {{1, 2}, {3, 4}}; for(int i = 0; i < 2; i++) { for(int j = 0; j < 2; j++) { printf("%d ", mat[i][j]); } printf("\n"); } return 0; }

Advantages of Arrays

  • Efficient data storage
  • Easy access using index
  • Useful for sorting and searching
  • Foundation for advanced data structures

Limitations of Arrays

  • Fixed size
  • Stores only same data type
  • Insertion and deletion is difficult

Conclusion

Arrays are one of the most important concepts in C programming. They help store and manage large amounts of data efficiently.
