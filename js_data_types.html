JavaScript Data Types

Understanding Different Types of Data in JavaScript

What are Data Types?

Data types specify the kind of data a variable can store. JavaScript supports both primitive and non-primitive data types.

Primitive Data Types

1. Number

Represents both integers and floating-point numbers.

let age = 25; let price = 99.99;

2. String

Represents text enclosed in quotes.

let name = "John"; let city = 'Mumbai';

3. Boolean

Represents true or false values.

let isActive = true;

4. Undefined

A variable that is declared but not assigned a value.

let result;

5. Null

Represents an empty or unknown value.

let data = null;

6. Symbol

Represents unique and immutable values.

let id = Symbol("id");

7. BigInt

Used to store very large integers.

let bigNumber = 12345678901234567890n;

Non-Primitive Data Types

1. Object

Used to store collections of data.

let user = { name: "Alex", age: 30 };

2. Array

Stores multiple values in a single variable.

let colors = ["red", "green", "blue"];

3. Function

A block of code designed to perform a task.

function greet() { return "Hello"; }

Typeof Operator

Used to find the data type of a variable.

typeof "Hello"; // string typeof 25; // number typeof true; // boolean

Conclusion

Understanding JavaScript data types helps you store and manipulate data effectively. They are essential for writing correct and efficient programs.
