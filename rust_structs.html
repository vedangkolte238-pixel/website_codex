Rust Structs

Custom Data Types in Rust

What is a Struct?

A struct in Rust is a custom data type that lets you group related values together. It’s useful for modeling complex data with multiple attributes.

Defining a Struct

struct Person { name: String, age: u32, city: String, }

Creating Instances

fn main() { let person1 = Person { name: String::from("Alice"), age: 30, city: String::from("New York"), }; println!("Name: {}, Age: {}, City: {}", person1.name, person1.age, person1.city); }

Structs and Functions

impl Person { // Method to display info fn display(&self) { println!("{} is {} years old and lives in {}", self.name, self.age, self.city); } // Method to increase age fn birthday(&mut self) { self.age += 1; } } fn main() { let mut person1 = Person { name: String::from("Bob"), age: 25, city: String::from("Chicago"), }; person1.display(); person1.birthday(); person1.display(); }

Tuple Structs

Structs can also be defined like tuples for simpler data grouping:

struct Color(u8, u8, u8); fn main() { let black = Color(0, 0, 0); println!("Red: {}, Green: {}, Blue: {}", black.0, black.1, black.2); }

Applications

  • Modeling real-world objects (Person, Car, Product)
  • Grouping multiple related values in a single entity
  • Encapsulation of data with methods using impl
  • Simplifying program structure and readability

Conclusion

Structs are a fundamental feature of Rust for creating custom data types. Combined with methods and ownership rules, structs help you write safe and organized programs.
