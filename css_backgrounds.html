CSS Background

Styling Backgrounds Using CSS

What is CSS Background?

CSS background properties are used to define background effects for elements. You can set background color, images, position, and more.

Background Color

Sets the background color of an element.

body { background-color: lightgray; }

Background Image

Adds an image as the background.

div { background-image: url("image.jpg"); }

Background Repeat

Controls whether the background image repeats.

div { background-repeat: no-repeat; }

Background Position

Sets the position of the background image.

div { background-position: center; }

Background Size

Controls the size of the background image.

div { background-size: cover; }

Background Attachment

Specifies whether the background scrolls or stays fixed.

div { background-attachment: fixed; }

Shorthand Background Property

All background properties can be written in one line.

div { background: #0033FF url("image.jpg") no-repeat center; }

Gradient Background

Creates smooth color transitions.

div { background: linear-gradient(to right, blue, purple); }

Best Practices

  • Use background images carefully for performance
  • Ensure text is readable over backgrounds
  • Use gradients instead of images when possible

Conclusion

CSS background properties help create visually rich designs. They are essential for modern web styling.
