R Data Manipulation

Techniques to modify, summarize, and manage data in R

Introduction

Data manipulation in R involves transforming data to make it suitable for analysis. Common tasks include filtering, selecting, arranging, summarizing, and combining datasets.

Selecting Columns

Use the dplyr package to select specific columns.

library(dplyr) data <- data.frame( id = 1:5, name = c("Amit", "Neha", "Ravi", "Sita", "Anil"), score = c(90, 85, 78, 92, 88) ) select(data, name, score)

Filtering Rows

Filter rows based on conditions using filter().

filter(data, score > 85) filter(data, name == "Neha")

Arranging Data

Sort data by column values using arrange().

arrange(data, score) # ascending arrange(data, desc(score)) # descending

Adding or Modifying Columns

Use mutate() to create or modify columns.

data <- mutate(data, grade = ifelse(score >= 90, "A", "B")) data

Summarizing Data

Use summarise() to get aggregate information.

summarise(data, avg_score = mean(score), max_score = max(score))

Grouping Data

Group data by a factor or column using group_by() and summarize.

grouped <- data %>% group_by(grade) %>% summarise(count = n(), avg_score = mean(score)) grouped

Combining Datasets

Use bind_rows() or merge() to combine datasets.

new_data <- data.frame(id = 6, name = "Rohit", score = 91, grade = "A") combined <- bind_rows(data, new_data) combined

Conclusion

Data manipulation is a crucial step before analysis. The dplyr package makes these operations fast, readable, and efficient.
