Python Strings

Complete Explanation with Examples and Output

What is a String?

A string is a sequence of characters enclosed in single quotes (' '), double quotes (" "), or triple quotes. Strings in Python are immutable, which means they cannot be changed.

Creating Strings

name = "Python" language = 'Programming' print(name) print(language)
Output:
Python
Programming

String Indexing

Each character in a string has an index starting from 0. Negative indexing starts from -1.

text = "Python" print(text[0]) print(text[-1])
Output:
P
n

String Slicing

Slicing is used to get a part of a string. Syntax: string[start:end]

text = "Python" print(text[1:4]) print(text[:3]) print(text[2:])
Output:
yth
Pyt
thon

String Concatenation

Strings can be joined using the + operator.

a = "Hello" b = "World" print(a + " " + b)
Output:
Hello World

String Repetition

The * operator is used to repeat strings.

word = "Hi " print(word * 3)
Output:
Hi Hi Hi

String Methods

upper() and lower()

text = "Python" print(text.upper()) print(text.lower())
Output:
PYTHON
python

strip()

text = " Hello " print(text.strip())
Output:
Hello

replace()

text = "I like Java" print(text.replace("Java", "Python"))
Output:
I like Python

split()

text = "Python is easy" print(text.split())
Output:
['Python', 'is', 'easy']

String Membership Operators

text = "Python Programming" print("Python" in text) print("Java" not in text)
Output:
True
True

String Formatting

Using format()

name = "Rahul" age = 20 print("My name is {} and age is {}".format(name, age))
Output:
My name is Rahul and age is 20

Using f-strings

print(f"My name is {name} and age is {age}")
Output:
My name is Rahul and age is 20

Conclusion

Strings are one of the most important data types in Python. Understanding string operations and methods is essential for text processing and real-world applications.
