R Apply Functions

Efficient ways to apply functions on data structures in R

Introduction

Apply functions in R are used to perform operations on elements of vectors, lists, matrices, and data frames efficiently. They reduce the need for explicit loops.

apply()

Used to apply a function over the rows or columns of a matrix or array.

mat <- matrix(1:9, nrow = 3) apply(mat, 1, sum) # sum of rows apply(mat, 2, sum) # sum of columns

lapply()

Applies a function to each element of a list and returns a list.

my_list <- list(a = 1:5, b = 6:10) lapply(my_list, sum)

sapply()

Similar to lapply() but tries to simplify the result (vector, matrix, etc.).

sapply(my_list, sum)

vapply()

A safer version of sapply() where you define the output type.

vapply(my_list, sum, numeric(1))

tapply()

Applies a function over subsets of a vector defined by a factor or grouping variable.

values <- c(5, 10, 15, 20) groups <- factor(c("A", "A", "B", "B")) tapply(values, groups, sum)

mapply()

Applies a function to multiple arguments in parallel.

a <- 1:5 b <- 6:10 mapply(sum, a, b)

rapply()

Recursively applies a function to elements of a list.

nested_list <- list(a = 1:2, b = list(c = 3:4)) rapply(nested_list, sum)

Conclusion

Apply functions are essential for writing concise and efficient R code. They are faster and more readable than using traditional loops for data operations.
