Pandas Series

One-Dimensional Labeled Array in Pandas

What is a Series?

A Pandas Series is a one-dimensional labeled array that can hold any data type (integers, strings, floats, Python objects, etc.). Each element in a Series has an associated label called an index.

Creating a Series

import pandas as pd # From a list s = pd.Series([10, 20, 30, 40]) print(s) # From a NumPy array import numpy as np arr = np.array([1, 2, 3, 4]) s2 = pd.Series(arr) print(s2) # From a dictionary data = {'a': 100, 'b': 200, 'c': 300} s3 = pd.Series(data) print(s3)

Series Indexing

s = pd.Series([10, 20, 30, 40], index=['a', 'b', 'c', 'd']) print(s['a']) # 10 print(s[['b', 'c']]) # b 20 # c 30 # Slicing print(s[1:3]) # b 20 # c 30

Series Operations

s1 = pd.Series([1, 2, 3, 4]) s2 = pd.Series([10, 20, 30, 40]) print(s1 + s2) # Element-wise addition → 0 11, 1 22, ... print(s1 * 2) # Multiply each element by 2 → 2, 4, 6, 8 print(s2 / s1) # Division → 10, 10, 10, 10

Applications of Series

  • Representing a single column of data
  • Quick computations and aggregations
  • Index-based selection and filtering
  • Used as building blocks for DataFrames

Conclusion

Series are a powerful one-dimensional data structure in Pandas, providing labeled indexing and vectorized operations. They are fundamental for data manipulation and are used extensively in DataFrames.
