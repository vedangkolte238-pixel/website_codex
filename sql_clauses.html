SQL Clauses

Clauses used to filter and organize data

What are SQL Clauses?

SQL clauses are used in SQL queries to filter, sort, group, and limit data. They usually work with the SELECT statement.

1. WHERE Clause

Used to filter records based on conditions.

SELECT * FROM students WHERE age > 18;

2. ORDER BY Clause

Used to sort records in ascending or descending order.

SELECT * FROM students ORDER BY age ASC;

3. GROUP BY Clause

Used to group rows with the same values.

SELECT department, COUNT(*) FROM employees GROUP BY department;

4. HAVING Clause

Used to filter groups created by GROUP BY.

SELECT department, COUNT(*) FROM employees GROUP BY department HAVING COUNT(*) > 5;

5. DISTINCT Clause

Used to remove duplicate records from result.

SELECT DISTINCT department FROM employees;

6. LIMIT Clause

Used to limit the number of records returned.

SELECT * FROM students LIMIT 5;

Conclusion

SQL clauses make queries more powerful and flexible. Understanding clauses is essential for data retrieval and analysis.
