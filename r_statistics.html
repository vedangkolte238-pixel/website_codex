R Basic Statistics

Fundamental statistical functions in R

Introduction

R provides built-in functions to perform basic statistical analysis on datasets. These functions are essential for understanding data distribution and relationships.

Mean, Median, Mode

data <- c(5, 10, 15, 10, 20, 15, 25) # Mean mean(data) # Median median(data) # Mode (using table) mode_val <- names(sort(table(data), decreasing=TRUE))[1] mode_val

Variance and Standard Deviation

# Variance var(data) # Standard Deviation sd(data)

Summary Statistics

summary(data)

Minimum and Maximum

min(data) max(data)

Quantiles

quantile(data, probs = c(0.25, 0.5, 0.75))

Range

range(data) diff(range(data)) # difference between max and min

Correlation

x <- c(1, 2, 3, 4, 5) y <- c(2, 4, 6, 8, 10) cor(x, y)

Covariance

cov(x, y)

Conclusion

These basic statistical functions in R help summarize and understand data efficiently. They are the foundation for more advanced statistical analysis.
