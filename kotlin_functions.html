Kotlin Functions

Learn How Functions Work in Kotlin

What is a Function?

A function is a block of reusable code that performs a specific task. Functions help reduce code duplication and improve readability.

Basic Function Syntax

fun functionName(parameters): ReturnType { // function body }

Example of a Function

fun greet() { println("Hello, Kotlin!") }

Function with Parameters

fun add(a: Int, b: Int): Int { return a + b }

Function with Return Value

fun square(num: Int): Int { return num * num }

Single-Expression Function

If a function has only one expression, it can be written in a shorter form.

fun multiply(a: Int, b: Int) = a * b

Default Parameter Values

fun greetUser(name: String = "Guest") { println("Hello, $name") }

Named Arguments

greetUser(name = "Kotlin")

Unit Return Type

If a function does not return any value, it returns Unit.

fun displayMessage(): Unit { println("No return value") }

Conclusion

Functions are a core part of Kotlin programming. They make code modular, reusable, and easier to maintain. Kotlin also provides concise syntax for writing clean functions.
