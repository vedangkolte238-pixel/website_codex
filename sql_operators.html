SQL Operators

Operators used to perform operations on data

What are SQL Operators?

SQL operators are used to perform operations on database values. They are commonly used in SQL queries with the WHERE clause.

1. Arithmetic Operators

Used to perform mathematical operations.

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus

Example

SELECT salary + 5000 AS total_salary FROM employees;

2. Comparison Operators

Used to compare two values.

  • = Equal to
  • != or <> Not equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to

Example

SELECT * FROM employees WHERE salary >= 30000;

3. Logical Operators

Used to combine multiple conditions.

  • AND
  • OR
  • NOT

Example

SELECT * FROM employees WHERE department = 'IT' AND salary > 40000;

4. Special Operators

Used for specific conditions.

  • IN
  • BETWEEN
  • LIKE
  • IS NULL

Example

SELECT * FROM employees WHERE department IN ('HR', 'IT');

Conclusion

SQL operators help filter, compare, and calculate data efficiently. Mastering operators is essential for writing powerful SQL queries.
