Basic SQL Commands

Learn the most commonly used SQL statements

1. CREATE TABLE

Used to create a new table in the database.

CREATE TABLE students ( id INT, name VARCHAR(50), age INT );

2. INSERT

Used to insert data into a table.

INSERT INTO students VALUES (1, 'Amit', 21);

3. SELECT

Used to fetch data from a table.

SELECT * FROM students;

4. WHERE

Used to filter records based on a condition.

SELECT * FROM students WHERE age > 20;

5. UPDATE

Used to modify existing data in a table.

UPDATE students SET age = 22 WHERE id = 1;

6. DELETE

Used to remove records from a table.

DELETE FROM students WHERE id = 1;

7. DROP TABLE

Used to permanently delete a table from the database.

DROP TABLE students;

Conclusion

These basic SQL commands are enough to start working with databases. Once you are comfortable with them, you can move on to joins, functions, and advanced queries.
