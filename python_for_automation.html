Python for Automation

Automating Tasks Using Python

Introduction

Python is widely used for automation because of its simplicity and powerful libraries. Automation helps save time by performing repetitive tasks automatically.

1. File Automation (os and shutil)

import os import shutil # Create a new folder os.makedirs("backup_folder", exist_ok=True) # Copy a file to the new folder shutil.copy("example.txt", "backup_folder/example.txt") print("File backed up successfully.")
Output:
File backed up successfully.

2. Web Automation (selenium)

# Install selenium: pip install selenium from selenium import webdriver driver = webdriver.Chrome() driver.get("https://www.python.org") print("Page title:", driver.title) driver.quit()
Output:
Page title: Welcome to Python.org

3. Excel Automation (openpyxl)

# Install openpyxl: pip install openpyxl import openpyxl wb = openpyxl.Workbook() sheet = wb.active sheet["A1"] = "Name" sheet["B1"] = "Age" sheet.append(["Alice", 25]) sheet.append(["Bob", 30]) wb.save("example.xlsx") print("Excel file created successfully.")
Output:
Excel file created successfully.

4. Email Automation (smtplib)

import smtplib sender = "youremail@example.com" receiver = "receiver@example.com" password = "yourpassword" message = """\ Subject: Test Email Hello, this is an automated email from Python.""" with smtplib.SMTP("smtp.gmail.com", 587) as server: server.starttls() server.login(sender, password) server.sendmail(sender, receiver, message) print("Email sent successfully.")
Output:
Email sent successfully.

Conclusion

Python automation can handle files, web browsing, Excel, emails, and much more. Using libraries like os, shutil, selenium, openpyxl, and smtplib, you can automate almost any repetitive task efficiently.
