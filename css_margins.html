CSS Margin

Controlling Space Outside Elements

What is CSS Margin?

CSS margin is used to create space outside an HTML element. It controls the distance between elements on a web page.

Margin Properties

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Margin Example

Adding margin around an element.

div { margin: 20px; }

Individual Margin Sides

Apply margin to specific sides.

div { margin-top: 10px; margin-right: 20px; margin-bottom: 10px; margin-left: 20px; }

Margin Shorthand

Use shorthand to set all sides at once.

div { margin: 10px 20px 10px 20px; }

Auto Margin

Used to center elements horizontally.

div { width: 300px; margin: auto; }

Negative Margins

Moves elements closer by using negative values.

div { margin-top: -10px; }

Margin Collapse

Vertical margins of adjacent elements may combine into one. This behavior is called margin collapsing.

h1 { margin-bottom: 20px; } p { margin-top: 30px; }

Best Practices

  • Use margin for spacing between elements
  • Avoid excessive margins
  • Use padding for inner spacing

Conclusion

CSS margin helps create clean and readable layouts. Understanding margins is essential for proper page spacing.
