Ruby Classes

Object-Oriented Programming in Ruby

Introduction

A class in Ruby is a blueprint for creating objects. It defines properties (variables) and behaviors (methods) that objects created from the class can use.

Defining a Class

class Person end

Creating Objects

person1 = Person.new person2 = Person.new

Initialize Method

The initialize method runs automatically when a new object is created.

class Person def initialize(name, age) @name = name @age = age end end p1 = Person.new("Alice", 25)

Instance Variables

Instance variables start with @ and store object-specific data.

puts @name puts @age

Instance Methods

class Person def greet puts "Hello, my name is #{@name}" end end p1.greet

Accessors

Ruby provides shortcuts to create getter and setter methods.

class Person attr_accessor :name, :age end p1 = Person.new p1.name = "Bob" p1.age = 30 puts p1.name

Inheritance

class Student < Person def study puts "Studying..." end end s = Student.new s.study

Conclusion

Classes form the foundation of object-oriented programming in Ruby. They help organize code, promote reuse, and make applications easier to maintain.
