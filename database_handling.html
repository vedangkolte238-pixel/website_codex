Python Database Handling

CRUD Operations Using SQLite

Introduction

Python can interact with databases to store, retrieve, update, and delete data. SQLite is a lightweight database built into Python.

1. Creating a Database and Table

import sqlite3 conn = sqlite3.connect("students.db") cursor = conn.cursor() cursor.execute(""" CREATE TABLE IF NOT EXISTS students( id INTEGER PRIMARY KEY AUTOINCREMENT, name TEXT, age INTEGER ) """) conn.commit() conn.close()
Output:
Table created successfully.

2. Inserting Data

conn = sqlite3.connect("students.db") cursor = conn.cursor() cursor.execute("INSERT INTO students (name, age) VALUES (?, ?)", ("Alice", 25)) cursor.execute("INSERT INTO students (name, age) VALUES (?, ?)", ("Bob", 30)) conn.commit() conn.close()
Output:
Inserted Alice and Bob.

3. Reading Data

conn = sqlite3.connect("students.db") cursor = conn.cursor() cursor.execute("SELECT * FROM students") rows = cursor.fetchall() for row in rows: print(row) conn.close()
Output:
(1, 'Alice', 25)
(2, 'Bob', 30)

4. Updating Data

conn = sqlite3.connect("students.db") cursor = conn.cursor() cursor.execute("UPDATE students SET age = ? WHERE name = ?", (26, "Alice")) conn.commit() conn.close()
Output:
Alice's age updated to 26.

5. Deleting Data

conn = sqlite3.connect("students.db") cursor = conn.cursor() cursor.execute("DELETE FROM students WHERE name = ?", ("Bob",)) conn.commit() conn.close()
Output:
Bob deleted from the database.

Conclusion

Python's SQLite module makes it easy to handle databases for small to medium applications. You can perform **CRUD operations** efficiently and integrate with other Python programs.
