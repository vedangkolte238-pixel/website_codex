Python GUI Programming

Introduction to Tkinter with Examples

Overview

Python's Tkinter module allows you to create GUI applications. You can make windows, buttons, labels, input fields, and interact with users.

1. Simple Window

import tkinter as tk window = tk.Tk() window.title("Simple Window") window.geometry("300x150") label = tk.Label(window, text="Hello, Python GUI!") label.pack() window.mainloop()
Output:
A window opens with text "Hello, Python GUI!" displayed in it.

2. Entry and Button Example

import tkinter as tk def greet_user(): name = entry.get() label_result.config(text="Hello, " + name) window = tk.Tk() window.title("Greeting App") window.geometry("300x200") entry = tk.Entry(window) entry.pack(pady=10) button = tk.Button(window, text="Greet Me", command=greet_user) button.pack(pady=5) label_result = tk.Label(window, text="") label_result.pack(pady=10) window.mainloop()
Output:
Enter a name in the box → click "Greet Me" → the label shows "Hello, [Name]".

3. Multiple Widgets Example

import tkinter as tk window = tk.Tk() window.title("Widgets Example") window.geometry("400x250") tk.Label(window, text="Username:").pack() username = tk.Entry(window) username.pack() tk.Label(window, text="Password:").pack() password = tk.Entry(window, show="*") password.pack() def submit(): tk.Label(window, text=f"Welcome, {username.get()}!").pack() tk.Button(window, text="Login", command=submit).pack(pady=10) window.mainloop()
Output:
A login-like window with Username and Password fields and a Login button. On clicking Login → shows welcome message with entered username.

Conclusion

Tkinter is easy to use and comes built-in with Python. It allows rapid development of simple desktop applications. You can combine Tkinter with other Python modules (like SQLite) for more advanced applications.
