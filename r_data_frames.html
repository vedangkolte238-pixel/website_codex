R Data Frames

Tabular data structure used for data analysis in R

Introduction

A data frame is a table-like structure in R. It stores data in rows and columns, where each column can have a different data type.

Creating a Data Frame

Data frames are created using the data.frame() function.

students <- data.frame( id = c(1, 2, 3), name = c("Amit", "Neha", "Ravi"), marks = c(85, 90, 78), passed = c(TRUE, TRUE, FALSE) ) students

Accessing Data Frame Elements

You can access rows, columns, and individual values.

students$name students[1, ] students[, 3] students[2, 2]

Adding New Columns

New columns can be added easily.

students$grade <- c("A", "A", "B") students

Adding New Rows

Use rbind() to add rows.

new_student <- data.frame( id = 4, name = "Sita", marks = 88, passed = TRUE, grade = "A" ) students <- rbind(students, new_student) students

Removing Rows and Columns

Rows or columns can be removed using indexing.

students <- students[-1, ] students <- students[, -2] students

Useful Data Frame Functions

Commonly used functions for data frames.

head(students) tail(students) summary(students) str(students)

Conclusion

Data frames are the most commonly used data structure in R. They are ideal for data analysis, manipulation, and reporting.
