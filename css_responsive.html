CSS Responsive Design

Make Websites Work on All Screen Sizes

What is Responsive Design?

Responsive design means creating web pages that look good on all devices. The layout automatically adjusts for mobile, tablet, and desktop screens.

Why Responsive Design is Important

  • Better user experience
  • Mobile-friendly websites
  • Improves accessibility
  • Works on all screen sizes

Viewport Meta Tag

The viewport meta tag controls page scaling on mobile devices.

Flexible Width Using Percentage

Using percentage instead of fixed pixels helps layouts adjust.

.container { width: 100%; }

Responsive Images

Images should resize according to screen size.

img { max-width: 100%; height: auto; }

Media Queries

Media queries apply styles based on screen size.

@media (max-width: 768px) { body { background-color: lightgray; } }

Mobile First Approach

Design for mobile devices first, then add styles for larger screens.

body { font-size: 14px; } @media (min-width: 768px) { body { font-size: 16px; } }

Responsive Layout with Flexbox

Flexbox helps create flexible layouts.

.container { display: flex; flex-wrap: wrap; }

Responsive Layout with Grid

CSS Grid adjusts columns automatically.

.grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); }

Common Breakpoints

  • Mobile: 0 – 600px
  • Tablet: 601 – 992px
  • Desktop: 993px and above

Best Practices

  • Use relative units
  • Test on multiple devices
  • Avoid fixed widths
  • Keep layouts simple

Conclusion

CSS responsive design ensures websites work smoothly on all devices. It is an essential skill for modern web development.
