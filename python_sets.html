Python Set

Complete Explanation with Examples and Output

What is a Set?

A set is a collection of unique elements in Python. Sets are unordered, mutable, and do not allow duplicate values.

Creating a Set

numbers = {10, 20, 30, 40} print(numbers)
Output:
{10, 20, 30, 40}

Set Does Not Allow Duplicates

data = {1, 2, 2, 3, 3} print(data)
Output:
{1, 2, 3}

Accessing Set Elements

Since sets are unordered, you cannot access elements by index. Use loops to iterate through set elements.

for item in numbers: print(item)
Output:
10
20
30
40

Adding Elements to a Set

add()

numbers.add(50) print(numbers)
Output:
{10, 20, 30, 40, 50}

update()

numbers.update([60, 70]) print(numbers)
Output:
{10, 20, 30, 40, 50, 60, 70}

Removing Elements from a Set

remove()

numbers.remove(20) print(numbers)
Output:
{10, 30, 40, 50, 60, 70}

discard()

numbers.discard(100) print(numbers)
Output:
{10, 30, 40, 50, 60, 70}

pop()

numbers.pop() print(numbers)
Output:
One random element removed

Set Operations

Union

a = {1, 2, 3} b = {3, 4, 5} print(a | b)
Output:
{1, 2, 3, 4, 5}

Intersection

print(a & b)
Output:
{3}

Difference

print(a - b)
Output:
{1, 2}

Symmetric Difference

print(a ^ b)
Output:
{1, 2, 4, 5}

Set Methods

nums = {1, 2, 3} nums.clear() print(nums)
Output:
set()

Conclusion

Sets are useful for storing unique items and performing fast operations like union, intersection, and difference.
