Practical Examples Using Pandas

Project 1: Sales Analysis

Analyze sales data to get insights about total and average sales per region and visualize trends.

import pandas as pd import matplotlib.pyplot as plt # Load dataset df = pd.read_csv('sales_data.csv') # Data cleaning df.fillna(0, inplace=True) # Group by region region_sales = df.groupby('Region')['Sales'].sum() print(region_sales) # Visualization region_sales.plot(kind='bar', title='Total Sales by Region') plt.show()

Project 2: Employee Salary Analysis

Combine multiple datasets, clean data, and analyze average salary per department.

df_emp = pd.read_csv('employees.csv') df_salary = pd.read_csv('salaries.csv') # Merge datasets df_merged = pd.merge(df_emp, df_salary, on='EmployeeID', how='inner') # Handle missing data df_merged.fillna({'Salary':0}, inplace=True) # Group by Department avg_salary = df_merged.groupby('Department')['Salary'].mean() print(avg_salary) # Plot avg_salary.plot(kind='pie', autopct='%1.1f%%', title='Average Salary by Department') plt.show()

Project 3: Student Performance Analysis

Analyze student grades, handle missing data, calculate average scores, and rank students.

df = pd.read_excel('students.xlsx') # Fill missing marks with mean df.fillna(df.mean(), inplace=True) # Calculate total and average df['Total'] = df[['Math', 'Science', 'English']].sum(axis=1) df['Average'] = df['Total'] / 3 # Rank students df['Rank'] = df['Average'].rank(ascending=False) print(df) # Visualize top 5 students df.nlargest(5, 'Average').plot(x='Name', y='Average', kind='bar', title='Top 5 Students') plt.show()
