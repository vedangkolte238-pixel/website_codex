SQL Functions

Built-in functions used to process data

What are SQL Functions?

SQL functions are built-in methods that perform calculations on data. They return a single value after processing input values.

1. Aggregate Functions

Used to perform calculations on multiple rows.

  • COUNT() – Counts number of rows
  • SUM() – Returns total value
  • AVG() – Returns average value
  • MIN() – Returns minimum value
  • MAX() – Returns maximum value

Example

SELECT COUNT(*) FROM students;

2. String Functions

Used to manipulate string data.

  • UPPER()
  • LOWER()
  • LENGTH()
  • SUBSTRING()
  • TRIM()

Example

SELECT UPPER(name) FROM students;

3. Numeric Functions

Used to perform numeric operations.

  • ROUND()
  • ABS()
  • CEILING()
  • FLOOR()

Example

SELECT ROUND(AVG(marks), 2) FROM students;

4. Date Functions

Used to work with date and time.

  • CURRENT_DATE
  • CURRENT_TIME
  • NOW()
  • YEAR()
  • MONTH()

Example

SELECT CURRENT_DATE;

Conclusion

SQL functions help perform calculations, format data, and extract useful information. They are essential for data analysis and reporting.
