NumPy File Handling

Saving and Loading NumPy Arrays to Files

Saving Arrays to a File

NumPy allows you to save arrays to disk for later use.

Saving as a binary file

import numpy as np arr = np.array([1, 2, 3, 4, 5]) np.save('my_array.npy', arr) # saves array in .npy format

Saving multiple arrays

arr1 = np.array([1, 2, 3]) arr2 = np.array([4, 5, 6]) np.savez('arrays.npz', arr1=arr1, arr2=arr2) # compressed file

Saving as text file

np.savetxt('my_array.txt', arr, delimiter=',')

Loading Arrays from a File

Loading a binary file

loaded_arr = np.load('my_array.npy') print(loaded_arr) # [1 2 3 4 5]

Loading multiple arrays

data = np.load('arrays.npz') print(data['arr1']) # [1 2 3] print(data['arr2']) # [4 5 6]

Loading from a text file

loaded_txt = np.loadtxt('my_array.txt', delimiter=',') print(loaded_txt) # [1. 2. 3. 4. 5.]

Applications

  • Saving intermediate results in computations
  • Storing large datasets efficiently
  • Sharing data between programs or machines
  • Reloading arrays without recomputation

Conclusion

NumPy’s file handling capabilities make it easy to save and load arrays for data analysis, scientific computing, and machine learning projects. Using `.npy` or `.npz` formats is efficient and convenient.
