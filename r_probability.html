R Probability

Working with probability distributions in R

Introduction

Probability functions in R allow you to calculate probabilities, densities, quantiles, and generate random numbers for different statistical distributions.

Common Probability Functions

  • d – density function
  • p – cumulative distribution function
  • q – quantile function
  • r – random generation

Normal Distribution

# Density dnorm(1, mean = 0, sd = 1) # Cumulative probability pnorm(1, mean = 0, sd = 1) # Quantile qnorm(0.95, mean = 0, sd = 1) # Random numbers rnorm(5, mean = 0, sd = 1)

Binomial Distribution

# Density / Probability Mass dbinom(3, size = 10, prob = 0.5) # Cumulative probability pbinom(3, size = 10, prob = 0.5) # Quantile qbinom(0.8, size = 10, prob = 0.5) # Random numbers rbinom(5, size = 10, prob = 0.5)

Poisson Distribution

# Density dpois(2, lambda = 3) # Cumulative probability ppois(2, lambda = 3) # Quantile qpois(0.9, lambda = 3) # Random numbers rpois(5, lambda = 3)

Other Distributions

  • Exponential: dexp(), pexp(), qexp(), rexp()
  • Uniform: dunif(), punif(), qunif(), runif()
  • t-Distribution: dt(), pt(), qt(), rt()
  • Chi-Square: dchisq(), pchisq(), qchisq(), rchisq()

Conclusion

R provides a wide variety of probability functions for statistical modeling and simulations. Understanding these functions is essential for analyzing random events and distributions.
