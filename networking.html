Python Networking

Socket Programming and Client-Server Example

Introduction

Python provides the socket module to implement networking. Sockets allow communication between computers over a network using TCP or UDP protocols.

1. Server Program (TCP)

import socket # Create socket server_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM) # Bind to localhost and port server_socket.bind(("127.0.0.1", 12345)) server_socket.listen(1) print("Server is listening...") conn, addr = server_socket.accept() print("Connected by:", addr) data = conn.recv(1024).decode() print("Received from client:", data) conn.send("Hello Client!".encode()) conn.close()
Output:
Server is listening...
Connected by: ('127.0.0.1', 54321)
Received from client: Hello Server

2. Client Program (TCP)

import socket client_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM) client_socket.connect(("127.0.0.1", 12345)) client_socket.send("Hello Server".encode()) data = client_socket.recv(1024).decode() print("Received from server:", data) client_socket.close()
Output:
Received from server: Hello Client!

3. Explanation

  • Server: Creates a socket, binds to IP & port, listens, and accepts connections.
  • Client: Connects to server and communicates using send/recv.
  • TCP: Reliable connection-oriented protocol used in this example.
  • Both client and server must run on the same machine or network for testing.

4. Conclusion

Python's socket module allows you to implement networking applications, including chat apps, file transfer, or client-server communication. This is the foundation for more advanced network programming like HTTP servers or real-time applications.
