C# Installation

Steps to Install C# and Setup Development Environment

Step 1: Install Visual Studio

C# is primarily used with Microsoft Visual Studio, which includes the .NET framework and C# compiler.

  • Go to https://visualstudio.microsoft.com/
  • Download the free Visual Studio Community edition.
  • Run the installer and select .NET desktop development workload.

Step 2: Verify .NET SDK

Make sure the .NET SDK is installed, which is needed to compile and run C# programs.

dotnet --version

If it shows a version number, .NET SDK is installed successfully.

Step 3: Create Your First C# Program

Open Visual Studio and follow these steps:

  • Click Create a new project.
  • Select Console App (.NET Core) and click Next.
  • Enter the project name and location, then click Create.
  • Visual Studio will generate a default Program.cs file where you can write C# code.

Step 4: Run the Program

Write a simple program in Program.cs:

using System; class Program { static void Main() { Console.WriteLine("Hello, C#!"); } }

Click Run or press F5 to execute your program.

Step 5: Optional - Install Visual Studio Code

If you prefer a lightweight editor, you can install VS Code and C# extensions:

  • Download Visual Studio Code: https://code.visualstudio.com/
  • Install the C# extension from Microsoft.
  • Install .NET SDK as explained above.
  • You can now write and run C# programs in VS Code.

Conclusion

After installation, your development environment is ready. You can now start writing C# programs and explore the language features.
