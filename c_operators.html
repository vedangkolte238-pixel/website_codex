C Operators

Performing Operations on Data in C

What are Operators?

Operators are special symbols used to perform operations on variables and values. C provides a wide range of operators for mathematical, logical, and decision-making tasks.

Types of Operators in C

1. Arithmetic Operators

Used to perform basic mathematical operations.

+ Addition - Subtraction * Multiplication / Division % Modulus int a = 10, b = 3; int sum = a + b; int rem = a % b;

2. Relational Operators

Used to compare two values.

== Equal to != Not equal to > Greater than < Less than >= Greater than or equal to <= Less than or equal to if (a > b) { printf("a is greater"); }

3. Logical Operators

Used to combine multiple conditions.

&& Logical AND || Logical OR ! Logical NOT if (a > 5 && b < 5) { printf("Condition true"); }

4. Assignment Operators

Used to assign values to variables.

= Assign += Add and assign -= Subtract and assign *= Multiply and assign /= Divide and assign int x = 10; x += 5;

5. Increment and Decrement Operators

Used to increase or decrease a value by one.

++ Increment -- Decrement int i = 5; i++; --i;

6. Bitwise Operators

Used to perform operations at the bit level.

& Bitwise AND | Bitwise OR ^ Bitwise XOR ~ Bitwise NOT << Left shift >> Right shift

7. Conditional Operator

A shorthand for if-else conditions.

(condition) ? value1 : value2; int max = (a > b) ? a : b;

8. sizeof Operator

Used to find the size of a data type or variable.

printf("%d", sizeof(int));

Conclusion

Operators are a fundamental part of C programming. Understanding operators helps you write efficient and logical programs.
