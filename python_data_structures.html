Python Data Structures

Detailed Explanation with Examples and Output

What are Data Structures?

Data structures are used to store, organize, and manage data efficiently. Python provides built-in data structures that make programming easy and powerful.

1. List

A list is an ordered, mutable collection of elements. Lists allow duplicate values.

numbers = [1, 2, 3, 4] numbers.append(5) print(numbers)
Output:
[1, 2, 3, 4, 5]

Accessing List Elements

print(numbers[0])
Output:
1

2. Tuple

A tuple is an ordered, immutable collection. Once created, its values cannot be changed.

colors = ("red", "green", "blue") print(colors)
Output:
('red', 'green', 'blue')

3. Set

A set is an unordered collection of unique elements. Duplicate values are automatically removed.

numbers = {1, 2, 2, 3, 4} print(numbers)
Output:
{1, 2, 3, 4}

Adding Elements to Set

numbers.add(5) print(numbers)
Output:
{1, 2, 3, 4, 5}

4. Dictionary

A dictionary stores data in key-value pairs. Keys must be unique.

student = { "name": "Rahul", "age": 20, "course": "Python" } print(student)
Output:
{'name': 'Rahul', 'age': 20, 'course': 'Python'}

Accessing Dictionary Values

print(student["name"])
Output:
Rahul

5. String

Strings are sequences of characters used to store text.

text = "Python" print(text.upper())
Output:
PYTHON

6. List vs Tuple vs Set vs Dictionary

  • List – Ordered, changeable, allows duplicates
  • Tuple – Ordered, unchangeable, allows duplicates
  • Set – Unordered, unique values only
  • Dictionary – Key-value pairs, keys are unique

Conclusion

Python data structures make data handling simple and efficient. Understanding lists, tuples, sets, and dictionaries is essential for real-world Python programming.
