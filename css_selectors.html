CSS Selectors

Target HTML Elements Using CSS

What are CSS Selectors?

CSS selectors are used to select HTML elements that you want to style. They tell the browser which elements should receive the CSS rules.

Why CSS Selectors are Important

  • They control which elements get styled
  • They help apply styles efficiently
  • They make CSS flexible and powerful

Types of CSS Selectors

1. Element Selector

Selects all elements of a given type.

p { color: blue; }

2. Class Selector

Selects elements with a specific class.

.myClass { font-size: 18px; color: green; }

3. ID Selector

Selects one unique element with an id.

#mainTitle { color: red; }

4. Universal Selector

Selects all elements on the page.

* { margin: 0; padding: 0; }

5. Group Selector

Applies the same style to multiple elements.

h1, h2, p { font-family: Arial; }

6. Descendant Selector

Selects elements inside another element.

div p { color: purple; }

7. Child Selector

Selects direct child elements only.

div > p { color: orange; }

8. Attribute Selector

Selects elements based on attributes.

input[type="text"] { border: 2px solid blue; }

CSS Selector Priority

When multiple selectors target the same element, priority matters.

  • ID selector has highest priority
  • Class selector has medium priority
  • Element selector has lowest priority

Conclusion

CSS selectors are the foundation of styling in web development. Understanding selectors helps you write clean, efficient, and powerful CSS.
