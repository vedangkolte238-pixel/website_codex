Go Language Installation

Step-by-Step Guide to Install Go (Golang)

System Requirements

  • Windows, macOS, or Linux
  • 64-bit operating system recommended
  • Internet connection

Step 1: Download Go

Visit the official Go website and download the installer for your operating system.

https://go.dev/dl/

Step 2: Install Go

Windows

  • Run the downloaded .msi installer
  • Follow the setup wizard
  • Go will be installed in C:\Go by default

macOS

  • Open the downloaded .pkg file
  • Follow the installation steps
  • Go installs in /usr/local/go

Linux

  • Extract the archive
  • Move it to /usr/local
sudo tar -C /usr/local -xzf go1.xx.x.linux-amd64.tar.gz

Step 3: Set Environment Variables

Windows

Add Go to the PATH variable:

C:\Go\bin

macOS / Linux

export PATH=$PATH:/usr/local/go/bin

Step 4: Verify Installation

Check if Go is installed correctly:

go version

If installed correctly, it will display the Go version.

Step 5: Write Your First Go Program

package main import "fmt" func main() { fmt.Println("Go is installed successfully!") }

Conclusion

You have successfully installed Go on your system. You are now ready to start building fast and efficient applications using Go.
