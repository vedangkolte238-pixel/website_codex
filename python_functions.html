Python Functions

Complete Explanation with Examples and Output

What is a Function?

A function is a block of reusable code that performs a specific task. Functions help make programs modular, readable, and maintainable.

Defining a Function

def greet(): print("Hello, World!")

Calling a Function

greet()
Output:
Hello, World!

Function with Parameters

def greet(name): print("Hello,", name) greet("Alice") greet("Bob")
Output:
Hello, Alice
Hello, Bob

Function with Default Parameters

def greet(name="Guest"): print("Hello,", name) greet() greet("Charlie")
Output:
Hello, Guest
Hello, Charlie

Function with Return Value

def add(a, b): return a + b result = add(5, 3) print(result)
Output:
8

Function with Multiple Return Values

def arithmetic(a, b): return a+b, a-b, a*b sum_val, diff_val, prod_val = arithmetic(10, 5) print(sum_val) print(diff_val) print(prod_val)
Output:
15
5
50

Lambda Functions (Anonymous Functions)

square = lambda x: x**2 print(square(6))
Output:
36

Why Use Functions?

  • Reusability – write once, use many times
  • Modularity – organize code into separate blocks
  • Readability – easy to understand and maintain
  • Efficiency – reduces code duplication

Conclusion

Functions are a core part of Python programming. Mastering functions allows you to write clean, modular, and efficient code.
