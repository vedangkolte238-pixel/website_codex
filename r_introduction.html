Introduction to R Programming

Basics of R for Data Analysis and Statistics

What is R?

R is a programming language mainly used for data analysis, statistics, and data visualization. It is widely used by data scientists, researchers, and analysts.

Why Use R?

  • Excellent for statistical computing
  • Powerful data visualization libraries
  • Large collection of built-in packages
  • Open source and free to use

Basic Features of R

  • Vector-based operations
  • Built-in statistical functions
  • Easy data manipulation
  • Strong community support

Simple R Program

Example: Adding two numbers in R

a <- 10 b <- 20 sum <- a + b print(sum)

Data Types in R

  • Numeric
  • Integer
  • Character
  • Logical

Vectors in R

Vectors are the most basic data structure in R.

numbers <- c(1, 2, 3, 4, 5) print(numbers)

Applications of R

  • Data Analysis
  • Machine Learning
  • Statistical Research
  • Data Visualization

Conclusion

R is a powerful language for data-focused work. Learning R helps you analyze data efficiently and create meaningful visualizations.
