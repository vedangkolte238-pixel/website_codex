Python Web Scraping

Extracting Data from Websites Using Python

Introduction

Web scraping is the process of extracting data from websites. Python provides powerful libraries like requests and BeautifulSoup to scrape and parse web data efficiently.

1. Installing Required Libraries

# Install necessary libraries pip install requests pip install beautifulsoup4

2. Fetching a Web Page

import requests url = "https://www.python.org" response = requests.get(url) print("Status Code:", response.status_code) print(response.text[:500]) # Print first 500 characters
Output:
Status Code: 200
<!doctype html><html lang="en"><head><meta charset="utf-8">...

3. Parsing HTML with BeautifulSoup

from bs4 import BeautifulSoup import requests url = "https://www.python.org" response = requests.get(url) soup = BeautifulSoup(response.text, "html.parser") # Extract the page title title = soup.title.text print("Page Title:", title) # Extract all links links = soup.find_all("a") for link in links[:5]: # Show first 5 links print(link.get("href"))
Output:
Page Title: Welcome to Python.org
/psf-landing/
/about/
/downloads/
/doc/
/community/

4. Extracting Specific Data

# Extract upcoming events from Python.org events = soup.find("div", class_="shrubbery").find_all("li") for event in events[:5]: print(event.text)
Output:
Python 3.12.0 Release
PyCon 2025 Conference
Other upcoming events...

5. Tips for Web Scraping

  • Always check the website's robots.txt before scraping.
  • Use time.sleep() to avoid overloading the server.
  • Consider using Selenium for dynamic websites with JavaScript.

Conclusion

Python makes web scraping easy and efficient with libraries like requests and BeautifulSoup. You can extract, parse, and save web data for analysis, automation, or reporting.
