Pandas Read & Write

Loading and Saving Data with Pandas

Reading Data from Files

Pandas can read data from multiple file formats such as CSV, Excel, JSON, and SQL.

Read CSV File

import pandas as pd df = pd.read_csv('data.csv') print(df.head()) # Display first 5 rows

Read Excel File

df = pd.read_excel('data.xlsx', sheet_name='Sheet1') print(df.head())

Read JSON File

df = pd.read_json('data.json') print(df.head())

Read SQL Table

import sqlite3 conn = sqlite3.connect('database.db') df = pd.read_sql_query('SELECT * FROM my_table', conn) print(df.head())

Writing Data to Files

Pandas allows saving DataFrames to various file formats.

Write to CSV

df.to_csv('output.csv', index=False) # index=False to skip row numbers

Write to Excel

df.to_excel('output.xlsx', sheet_name='Sheet1', index=False)

Write to JSON

df.to_json('output.json')

Write to SQL

df.to_sql('my_table', conn, if_exists='replace', index=False)

Applications

  • Loading datasets for analysis and machine learning
  • Exporting processed data for reports or sharing
  • Integrating Pandas with databases and external data sources
  • Efficient handling of large structured datasets

Conclusion

Pandas provides simple and powerful tools to read and write data from multiple file formats. This makes it extremely useful for data analysis, preprocessing, and storage.
