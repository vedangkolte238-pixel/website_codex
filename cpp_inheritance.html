C++ Inheritance

Understanding Inheritance in Object-Oriented Programming

What is Inheritance?

Inheritance is a feature of object-oriented programming that allows one class (derived class) to acquire the properties and methods of another class (base class).

Why Use Inheritance?

  • Reusability of code
  • Improved organization
  • Reduces redundancy
  • Allows hierarchical relationships

Basic Example of Inheritance

#include <iostream> using namespace std; class Animal { // Base class public: void eat() { cout << "Eating...\n"; } }; class Dog : public Animal { // Derived class public: void bark() { cout << "Barking...\n"; } }; int main() { Dog d; d.eat(); // Inherited from Animal d.bark(); // Own method return 0; }

Types of Inheritance

  • Single Inheritance: One derived class inherits from one base class.
  • Multiple Inheritance: One derived class inherits from multiple base classes.
  • Multilevel Inheritance: Chain of inheritance where a class is derived from a derived class.
  • Hierarchical Inheritance: Multiple classes inherit from a single base class.
  • Hybrid Inheritance: Combination of two or more types of inheritance.

Multiple Inheritance Example

class A { public: void showA() { cout << "Class A\n"; } }; class B { public: void showB() { cout << "Class B\n"; } }; class C : public A, public B { }; int main() { C obj; obj.showA(); obj.showB(); return 0; }

Access Specifiers in Inheritance

  • Public: Public and protected members of base class become public and protected in derived class.
  • Protected: Public and protected members of base class become protected in derived class.
  • Private: Public and protected members of base class become private in derived class.

Advantages of Inheritance

  • Reduces code duplication
  • Promotes code reusability
  • Makes programs easier to maintain
  • Supports polymorphism

Conclusion

Inheritance is a core concept of OOP in C++. It allows classes to build upon each other and promotes organized, reusable, and maintainable code.
